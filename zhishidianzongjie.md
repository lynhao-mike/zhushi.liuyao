# 第一章 日月动变

爻得月建旺相有四

1.爻临月令 2.爻得月令合 3.爻得月令生 4.爻得月令扶

爻被月建衰败有四

1.爻遭月令冲 2.爻被月令克 3.爻克月令 4.爻生月令

爻在月建为平相有

爻得月气，在原始旺衰平关系的基础上，得气者趋旺，失气者趋衰。

--假设巳火为用神，在未月未日占卜，虽爻生月原始状态属于休囚格局，但同时火爻气在未，则巳火在未月不失为有气，只要得到他方照旺，巳火在未月依然可争取平相；巳火在未日为无生克属平相，爻于日月若都是平相，则爻总体为平相。若巳火失气，则在未月处休囚，总体将变为衰败。

在吉凶判断层面，碰上爻与月令相合又相克，如卯与戌；或爻与月令相冲又相扶，如丑与未；两种方向相反的卦理相撞，论旺还是论衰- ----对应现状---如世爻申金逢巳月，若客观现状是得到帮扶资助，则与相合的卦象相符，该论合；若客观现状是遭伤受损，则与相克的卦象相符，应论克----如果实在没办法对证到现状，那土爻逢月冲就一概视为月破；爻被日月相克之中又带合，一律都视为相克就是。虽然这有点儿纸上谈兵的味道，但毕竟固定套路也有它概率上的准确性”。

---爻于日建出现类似情况也应仿此理论---些易学者学了“日绊”理论之后，了解到日令相合卦中的动爻，只论合绊不论合旺，又想当然地以为动爻与月令相合，也可以理解为“月绊”。这其实是错误的，月令之合与日令之合的其中一个区别就在这里，在吉凶判断层面，卦中得月令相合的爻，不管它是静爻、动爻或者是变爻，都只是合旺的解读，而没有绊的定义；到了应期细节层面的飘逸思维下，才允许出现论绊的思路，这个要注意。

爻得日建旺相有五---

1.爻临日建 2.静爻得日令合 3.爻得日令生 4.爻得日令扶 5.爻临日令长生与帝旺--土爻逢申日则长生在日令，逢子日则旺在日令

爻在日建为平相有二----1.爻克日---2.爻生日

爻被日建衰败有二------1.爻被日令克----2.爻遭日令绝---如子水爻逢巳日，则叫爻绝在日

生旺墓绝在日令的应用 --长生与帝旺代表旺相，绝代表衰相，而墓则是中性的---爻旺者附旺，爻衰者附衰---这与论月气的应用原理相仿，如金爻绝在寅，酉金爻逢寅日，若酉金爻得动变力量生旺、或因月令显出兴旺，则酉金在寅日不论绝，为爻克日不显旺衰的平相；但酉金在月令衰败或被动爻相克显出总体衰相，则酉金在寅日就该论绝显出衰相了

----假设酉金为用神，在申月寅日占卜，酉得月扶故在日论平不论绝，总体旺相

趋旺的动变组合---1.爻被有用动爻相生 2.爻被动变组合形成三合局来生 3. 爻动变回头相生 4.爻动而化进神 5.爻动变出临日月之爻或合月之爻，且变爻不产生回头的生克墓等作用。

趋向衰的动变组---1.爻被有用动爻相克 2.爻被动变组合形成三合局来克 3. 爻动变回头相克 4.爻动而化退神 5.爻动而化绝，且不存在回头生 6. 爻动而化破（化月破或日破），且不产生回头生克墓-----注意一点：吉凶层面同样是动而化破，用神动而化破基本上属于凶兆定性，但自占自事中的世爻动而化破则另类，其吉凶定性，往往聚焦在世爻变出的六亲信息，而非聚焦在月破信息，不能光凭借世爻化破信息，就贸然断定世爻陷入衰败。因为自占自事中的世爻之动，它的运动方向只有一个，就是趋向自己的变爻，不是反映在动爻变无力，而是在反映变爻的最终发生。自占出现的世爻化破，其实在吉凶层面属于假破，这是根据“破”可以代表尚未发生的现状信息，延伸出来的特殊卦理。

# 旺衰间左右摇摆的动变组合-

---1.静爻被有用动爻所冲.旺则被冲起不衰，弱则被冲破而衰败。---（动而卦中的动爻冲静爻，若又是冲中带克的模式，这要看被冲的静爻是否为用神身份。在吉凶判断层面，用神安静被动爻冲克，只论用神受动克；而一般静爻被动爻冲克，则可论为冲起。自古皆是如此定论，其中关窍学者不可不知）

---2.静爻被有用动爻所趋合.旺则更显旺相，弱则会立足不稳反被吸引而显出衰相---（丑土发动克合子水、卯木发动克合戌土、巳火发动克合申金之类，于吉凶判断层面论动克而不能论相合，到了应期细节层面，生克冲合都可以自由发挥了）

--- 3.静爻被有用动爻所墓.旺则只是暂时被墓住，出墓依然不衰，弱则永远被墓而显出衰败之相

无用动爻---1.动变回头克之爻 2 动而化退之爻 3.动而化破--(化月破),化散--(化日破)之爻--不同时存在回头作用关系 4.动而化绝之爻-----以上四种无用动爻，都有一个共同的特征，就是动而变衰

# 矛盾趋势的分析辨别原则----

1.动能轨迹终点原则

2.重动轻静、重内轻外----受日令、月令因素造成爻之旺衰，属于静因与外因，如动而化散、化破、化日月建、化月合...----受卦内的动变因素造成爻之旺衰，则属于动因与内因，如动化进神、退神、化绝、化回头生克...

爻动而化月建的同时又是动而化绝，爻动而化月建是外因，爻动而化绝是内因，内重于外，则此化月建为假，化绝衰败无用才真，长事之占更是如此。

3. 自占自事世爻之变原则

---世爻之动，本身不存在动而有用无用的概念，只要见世爻发动，它一定就是吉凶层面必须重视的信息点。而世爻之动体现出来的旺衰定位特点，与一般的动爻也存在小差异，之上提到的两类定位原则，整体上也适用于世爻之动变，例如世爻动化回头生克的同时也化破，生克为真化破为假；世爻动变进退神又同时化破，进退为真化破为假。

而世爻动变体现的个性则在于---当世爻发动变出用神、世爻发动变出忌神、或世爻发动变出不回头生克的官鬼子孙之时，即使同时出现化日破月破、化日建月建，吉凶层面均不能按所化之日月定性，而应该按所化的六亲定性。例如自占自事世爻变出忌神或世爻变鬼的同时，也化日建月建，那么世爻变忌神变鬼不利为真，化日建化月建旺相为假；同理，自占自事世爻变出用神、或世爻变出子孙的同时，也化日破月破，那么世爻变用神变子孙吉利为真，化日破化月破衰败为假

# 复合之动的表现形式-

1.在数动爻中复合构成三合局，再整体以合局形式指向作用世爻或用神

2.数动爻彼此力量先复合构成连动相生，再以整合形式指向作用世爻或用神

3.数动爻彼此力量先构成连动相克，相互对抗而抵消，内部有机组合后再以整合形式指向作用世爻或用神

# 复合动爻的的运转规则要点---

1.三合局优先于单爻的连动

2.动爻因有明确的目标指向，故此复合连动，不管通过连动相生或连动相克的内部整合，其最终目标与线路都很清晰，都是谋求围绕着世爻或用神方向整体运动，没有了方向目标，动爻则失去复合连动的意义，彼此也不会再产生连动

3. 就算是单爻即可各自直接作用目标爻，只要连动后的方向依然围绕目标爻产生作用，各单爻就必须先予连动，按连动后的总体方向论吉凶

4.占问自身事情的卦象，在吉凶判断层面，世爻是不参与复合与连动的，只有别的爻围绕它作用，它却不会去作用任何别的爻

# 世用关系影响下的吉凶判断原则-

1.用神必须旺相

2.用旺世兴--用神旺相前提下，世爻只要得日、月其中一令扶助，即使在日月组合总体衰败也可以是吉利的兆头---用旺世衰,反而是由于自己原因而不成事，或者事成了自己却不能受益的凶兆

3.自占己事,世爻不能受动克或自变衰败---世动而化退、世动而化绝、世动而回头克、世动而化忌神、世动而化鬼（世动而化破除外）…世爻受制，岂宜自己之谋

# 动爻作用次序是先用神后世爻-

在细节分析上，是以灵活飘逸为原则，任何的生克冲合信息，在这个层面都有其现实的意义所在。如动爻生旺用神吉凶层面上代表工作可成，同时此动爻又存在克伤世爻的信息，在细节分析上则可意味着这份工作虽然可以得到，但将不利于自身发展或者存在压力。

动爻无目标指向则会回归其自然本性-----爻之冲合与生旺墓绝属于爻的自然属性，爻之生克现象，属于爻的卦理属性，当动爻完全不再作用（生、克、冲、合、墓）世爻或用神的时候，动爻本身的天性— “冲合墓”的自然属性功能就开始无限扩大，可冲起或冲破静爻，可趋合与动墓他爻，试图通过作用这些爻，从而达到间接性地对用神与世爻产生作用之目的。

# 第二章 占卦意念与代占

·野鹤曰:一念之诚，可格天地；二簋之用，可享鬼神。若有心怀两三事而占者，念既不专，应何能一？念多心乱，即是不诚，故曰遇事即占，不拘早晚，慎勿二念，二念不灵。

提防礼让导致的意念窜卦 ---如《增删》中来人欲占问父亲病况，正准备摇卦的时候，见有位官长突然大驾光临，只好让位给官长先占卦此例

失控性代占- --是代占人对所占他人之事，完全没有任何主导控制力，只是关心关注而问占。

主控性代占- -指的是代占人对所占他人之事，其实质是受自己主导主控的。

---有时候它会以围绕代占用神的动态出现，代占父母事，就将发动或被动爻针对的父母爻视为分析的用神。

当动态信息直接指向世爻的时候，我们对卦中六亲的定位也需及时调整，而是应该以自占的思维定位卦中六亲。

----例如大人代问孩子上哪家幼儿园更合适，如果卦中出现了针对子孙爻的动变，事件的利弊指向自然聚焦在孩子身上，分析代表孩子的子孙爻就可达到目的；如果此时卦中出现了妻财爻发动克世，我们千万别以为代占卦与世爻无关，毕竟这类主控性代占事情中，被代占者（孩子）实质是没有自主权的，决策主导权反而在占卦者（世爻）身上，此时妻财爻针对世爻动克，这妻财乃是指大人自己的财，提醒大人应该避免选择太贵价的学校，以免因为决策错误遭遇破财损失。

有些主控性代占类型，可能会以整体视觉的代占模式体现。 这类代占也不可草率地取被代占人为用神，有时则需要审视条件，直接取目标之六亲为用。

-如妻子代占丈夫在某单生意中得财与否，如果她丈夫的钱一向是独立自主的，等于丈夫的钱财得失与否，这钱都与她无直接关系，那么应该取代表丈夫的官鬼为用神，倘若她丈夫的钱一向属于家庭共享的，丈夫赚钱与否和自己赚钱与否，没有任何区别就是一回事，都归属于自家的财运，所以此财明为丈夫之财，在代占人眼中那都是自家的财，应该直接取卦中代表自家财运的妻财爻为用神

被动性代占----虽自己是摇卦者，应视同于提议者自占自摇方为正确。

# 卦象显示特征-

1.只答心念不答口述---直接看穿问事人的心中所盼，断卦之精确度自然可以大幅提高,这正是易学业余与专业要求的分水岭

5.意念扶正伐邪

6.所答不谋专事

# 第三章 动卦卦理性吉凶通论

# 动卦中的吉利卦局-

1.世用受生局---自占事，卦中用神与世爻重叠于一点，同时受到动爻生旺，不但事成而且自身也因此受益。这是众多吉利卦局中吉利指向最明显的一种

2.用神生世局---所问占的事情使我获得益处，当然是好事。前提只要是这个用神是有用动爻，世爻不自变衰败---可更细分为直接受益与间接受益、合局受益三类

3.用旺世兴局---世爻在没受动爻克伤或自变衰败的情形下，得日令月令其中之一生扶，即便总体衰败亦无妨---用旺世兴,寓意事情有成功机会的同时，自身也能把握住成功的机会，所以吉利---但这也是一种“人与事”瓜葛并不十分不明显的卦局，它很容易与卦意分析法、双核卦分析法等众多较深入的套路，造成分析思维的冲突。这需要学者们在不断丰富学识、积累实战经验的基础上，渐渐感悟“动必有因”真谛而培养出鉴别的能力。

# 动卦中的凶兆卦局-

1.用神衰败局

2.世爻受伤局---自占事若世爻受到动爻克伤，或动而自变衰败

3.世用受克局---自占事，卦中用神与世爻重叠于一点，同时受到动爻克伤- 与前两种凶兆卦局区别在于，不但谋事不成，同时自身也因事败而遭受伤害

4.用旺世衰局---事情成功并非没有机会，所以用神是兴旺的，而是由于自身的主客观条件不足，造成事情最终不利---世爻衰弱有时候也会体现出主观的一面，也就是自我对事情的主动参与欲望，其实并不强烈，导致该做的事情最终没去做，由此错过了成功机会

由于人对事情的主观介入意愿的缺乏，构成了诸如世爻变衰、世爻不兴等差卦趋势模式，这些预期并非来自事情的客观趋势，而是自我的怠慢心态所造成，自我对事情主观介入意愿不强的心理状态下，并不适合去摇卦问事

现实中存在着的另一类不适合摇卦问事的心理状态，当一个人正在处于高度慌张失措的一刹那，在短暂的非理性心理状态下，并不适合立即去摇卦，因为这样的卦一定会显出鬼爻旺的，不过这个鬼爻旺相，它并不是寓意趋势会忧虑成真，而是事主高度紧张的即时信息投影，就会造成心态卦的误判

5.用神克世局---用神克世爻，如同事情使自身受到伤害，不管用神旺相与否都不能算是吉利之兆

---若用神动而遭遇破败显出衰相，或者世爻衰败不堪用神之克，则只是事情不成功而已---若用神有用而动克世爻，而此时的世爻又显旺相的，短期却不排除有成事的机会，如占短期失物、短期工作、短期考试等等，用神发动克旺相世爻可视为成事之兆，只是过程压力较大或较辛苦而已

旦短事变长久，或本身为长事之问，即使能短暂成事，之后的趋势却是走上令自身因此受伤害的凶途

# 卦局通论中的特例-

1.自占求财---卦见财爻作为有用动爻动而克世爻，只要世爻处于在日月其中之一获生扶的总体平相，即为有财的预兆----也须要用神动而有用

---在细节分析层面，财生世是得财的同时，还令自身最终受益的信息，或为过程舒畅之象；而财克世则是自身在得财的同时，或压力大或过程辛苦，或招致另外方面的损失代价，或因得到而失去，或是长期不适宜此途的另兆

2.自占疾病医药---子孙爻作为用神动而克世爻，却是医药有效的征兆。既然世受子孙克，那持世的必定是官鬼爻，用神子孙爻克的不是代表自身的世爻，而是代表事主身上疾病的鬼爻，故为吉利-----对世爻旺衰没要求

3.占问行人---得用神动而冲克世爻，却是行人将很快回来的兆头---用神克世人即回---它与前两类区别之处在于，因这属于他人之问，只要是用神克世，便可据此确认行人即归，被克的世爻旺衰如何，都不会妨碍吉兆的成立

4.忧患之占的心态卦---得喜神子孙爻动而冲克世爻，却是扫去心中忧虑的吉利兆头--只有心态卦之占是百无禁忌，不怕世动回头克----对世爻旺衰没有要求

4.忧患之占的心态卦---得喜神子孙爻动而冲克世爻，却是扫去心中忧虑的吉利兆头--只有心态卦之占是百无禁忌，不怕世动回头克

# 第四章 系统性暗动理论

# 冲起暗动有用的六种情况----

1. 静爻得月令趋旺而受日冲---谓之旺相者，冲之不散

2.静爻得月令之气而受日冲----谓之有气者，冲之不散--但丑月占卦遇亥水静爻受日冲、辰月占卦遇寅木静爻受日冲、未月占卦遇巳火静爻受日冲，基本不应暗动反而应日破

3. 静爻正处于旬空状态下受日冲----即使在爻逢月破衰败之极的状态下，只要符合空爻被日冲的“冲空”条件，照样被冲起而短期成为有用动爻---谓之冲空

4. 静爻虽在月令休囚，但在受日冲的同时又得动爻来生旺----谓之动爻趋旺

5. 问事若用神得旺相，世爻若受日令冲动，则无论其旺衰，皆算冲起暗动---谓之用趋世兴

6. 明动之爻受日冲，不论旺衰，皆属冲起有用----谓之动不为散

# 暗动之爻的固有特征---

1.有动无变、瞬间峥嵘

2.过时则无效，不行将废，所谓近兆不管远卜

---若测短事，如问急病、单位应聘、短期性的投资投机、短期信息确认等等属于短事性质的卦卜，暗动之爻与明动之爻作用没有差别

---但若问事的性质稍见长期，则一概算日破无用

要注意的是，长事之问卦中若出现用神或元神暗动，它们并非被视为长期日破之爻，而是长事快应的征兆

暗动之爻的现实寓意---正在发生或即将发生的人事，万一它预示的是即将发生的事情，则暗动的短效特征将带来另一个信息昭示：不行将废。问事即使卦象出现暗动趋吉的吉兆，这可能不代表正在发生的事情，而是代表即将发生的机遇，如问占人不及时把握住这个机遇采取积极主动的态度，则时过将废，错过趋吉避凶的良机。从某个角度上分析，这类暗动已经带有指导卦的性质。

--从暗动的这个特征可深刻体会到，未来世界本身就是一个变数，从来没有绝对性不变的趋势，包括卦象的预兆。人与事两者却是互牵互动的，人事趋势会顺着卦象昭示变幻，卦象趋势也会因人事的轨迹改变而转变。卦吉利而人不为，吉兆如无；人见卦凶而应变，凶兆化虚。

变爻中冲起与冲散的鉴别---变爻是动变的终点，它若不作为能量中转站，就存在真破的可能，得月旺而受日冲的变爻，可定义为冲起，化暗动；遭月衰囚而受日冲的变爻，可定义为冲散，化日破或叫化散---只有变爻旬空而受日令相冲者例外，不用参考月令状态而直接定性为化暗动，皆因冲空即实。爻化暗动，短期仍不失其有用动爻的本色，而爻若化散，则如同爻化月破、爻化绝那样，因终局变衰而沦落为无用动爻

---短事之占化暗动与化散可直接根据卦理判定即可，但如问事性质属于长久之问，都将一概被视为日破---注意真假之分

暗动的应期细节层面分析---卦中出现暗动之爻，则寓意为刚刚发生、正在发生、或即将发生的事情，暗动成为判断所占问的人事、其发生时间缓急的一个重要信息依据。卦见暗动，应期必见急促，急者或应在时辰，即使缓者也比预期来得要快，乃是事情急应或比预期急促的征兆

1. 冲中逢值逢合而应的日冲应期原则

2.顺时而应的日冲应期原则---如辰年午月申日占卦，卦中出现暗动，应日的可能会应在次日酉日，应月的可能会应在次月未月，应年的可能应在次年巳年

3.爻得冲空，即时生效---多代表已经发生或正在发生的事态---而有些卦象中的“冲空”，因为不具备即时生效的条件，于是顺延到次日发生，这类情形在短线股票走势等短期性投资预测中出现得最多

# 静卦的判断原则

古传统的静卦判断原则- --静卦中旺相的静爻，功能就如同动卦中的动爻，可以生克作用其余静爻，若多爻同旺，彼此还可以连环作用，就如动卦中的多爻同动而需连动作用一般，这就是静卦的传统分析思路

简捷准确的自占己事静卦判断三步骤-

# 一：先看有无用忌持世

---自问自事若得静卦用神持世，不管世用旺衰与否，即属吉兆

---但当用神持世出现“无根”或“逢破”之时，则为凶

---静卦用神持世见无根者，只可保事态短期内暂时无虞，但却不利于长久，长远为凶。

---静卦用神持世见逢破者，凶兆却是即使生效

皆因无根之爻难持久，逢破之爻已遭殃的缘故

----自问自事摇得静卦忌神持世，不管世爻旺衰与否，即属吉兆

---忌神持世若逢破，反为吉兆的有四种情况

1.求财之占；

2.医药之占；

3.行人之占；

4.忧患之占；

若是求财，摇得静卦忌神兄弟持世，无论逢月破、日破，或被财爻冲起成暗动，这都是得财的吉兆

野鹤曰：故凡占得兄爻持世，而世值月破、旬空、化墓、及日月作财冲世，或世爻兄弟变出财爻，皆许得财者，俱屡占得验，始敢告之。若世持兄弟，非此类，不得以此例决之也

---除此四种例外的事占，其余之问得静卦忌神持世逢月破日破，一样是凶的兆头，如问官职摇得静卦子孙持世遭月破，或男占感情摇得兄弟持世遭日破，则一样是事败的凶兆

---此静卦用忌神持世判断之法，非以六亲作为用神的不适合应用。如占自身安危、或自占疾病、或自占出行等等，取世爻为用神不取六亲为用的；又如以星煞作用神的，如占问出国马星为用神；或以六神为用神，如占血液白虎为用神等等。这些不循六亲为用神的，即使摇得静卦也不适宜以用忌持世判断吉凶，另外代占性质的卦也不适合以用忌持世判断事之吉凶

# ： 再看世用生克关系

1．用神生世格局---只要用神在日月有生扶条件下保持总体平相，即为吉兆，一般并不需要再分析世爻旺衰与否，就象动卦里的用神动而生世一样，无须再分析世爻如何，有用神来生，世爻因此已被趋旺，也不用分析其余无关之静爻的旺衰与生克

2. 用神克世格局---不论用神或世爻旺衰如何，即不为吉

----用神衰败无力或用神有力但世爻却衰弱，则事情将无成功机会，虽表面上事败非喜，但用神失败却因此无力克世，事实上反是自身躲过一场可能受伤害的劫数，塞翁失马焉知非福，只是不自知罢了,这与动卦中用神克世的理论运用原则是一致的。

----用神得力的同时世爻又旺相，则所占之事却不排除短期有成功的可能，用旺世旺事虽成功，却属于用神克世的格局，此事长期延续下去的话将令自身最终受到伤害，犹如是引狼入室，祸由福起而不知不觉

# 静卦得用神克世反为吉利的情况-

# 1.求财财爻克世反得财

---用神起码是受日月之一生扶的平相，另外受克的世爻也不能衰弱，否则身弱不能担财，不是难求到财就是得财反致祸兆，或是受财拖累

-野鹤曰：古以占财者，喜财克世，殊不知占目下之财，财克世者必得；占终身之财福，财克世者，乃生平受财之累，因财之害也。财克世虽然是有财之兆，但也须看占事的期限，是求长事之财或是求短事之财，若是问占终身财福之类的超长期事占，则财克世反是为财受苦累的凶兆。

# 2.占行人用神克世人速回

除上述两类特殊之问，其余若静卦出现用神克世，不论用神旺衰与否， 中长期看皆属凶兆；短事占之利弊，则要看世用的综合旺衰状况而定

# 三：非用忌持世，又非用神生克世爻，则参照动卦吉凶判断格局中的“用旺世兴”原则，各自单独分析用神与世爻于日月组合的旺衰程度，或它们受暗动之爻的旺衰作用，只要符合用神旺相同时世爻得日月之一生扶，则可断为吉兆

# 自占自事静卦中暗动的应用

-也遵循着暗动短期起效之特征。问事若属于长事性质，其用神或元神、忌神的暗动，乃代表长占短应的预兆，如酉月辛巳日占夫妇将来和好否的静卦“地天泰”，问夫妻之长久关系，用神妻财亥水暗动兴旺，这只能速应与其妻短期内安然无恙，长期则以静卦忌神兄弟持世论凶了。又如戌月戊辰日占终身功名有无得静卦“山风蛊”，官鬼持世元神妻财戌土暗动，本属终身的长久之问，由于元神暗动导致短短五年内就实现了仕途梦想，这也是目的长占快应的表现

----除用神、元神、忌神暗动之外，问长事静卦中出现的其他爻暗动，于吉凶层面可以一概忽略，其暗动信息一般只在应期细节层面中分析。而短事性质的事占则相反，所有暗动都可以介入静卦的吉凶定性之中，成为决定事件趋势的关键

# 静卦应期层面的特点知识-

1.静而逢值逢冲应---旺逢冲，衰逢值

2.元神在静卦中的信息地位也相当重要，有时候静卦的应期，不聚在用神那里，反而集中在元神身上，事吉的应在元神出空或临旺发力之时，事凶的应在元神衰败被冲之日

3．识别静卦远兆近兆，并不一定“静卦无急兆”，有不少数量的静卦，显示的其实是急兆的信息，这种急兆的信息，有些是纯粹的慢占快应，有些却是占此应彼，多与静卦中出现暗动或用神带日月相关。

# 卦意分析法(直读法)

---有些卦象，它是几种卦意分析的综合体现

--任何物质（包括所有的有形与无形）它的具体表现都可能两极化，不会只有一种单向的表现方式，这是周易之道“太极生两仪”即“一种本质下两种相对表达”的思维延伸

# 1.世化用忌法---自占自事-

以官鬼为用神的事占，出现世化官鬼的格局，则要多一份思考，化出的官鬼于日月衰弱则是祸患之鬼，不为世化用神，为世动化鬼

# 2.用忌(仇)互化法---代占他事-

只有一种情形属于例外，那就是自占自事，卦象出现用神持世同时又自变出仇神

# 3．鬼用互化法-

---问事而鬼用互化，祸患与事情纠缠不散，求财而鬼用互化，是求财反被骗或落入祸非的兆头；问病而鬼用互化，是病重忧患趋势的凶兆；求事而鬼用互化，多是事非将纠缠不断，当官鬼为用神之时，不存在鬼用互化

# 4.世动化鬼法-

问卦人将在所占问的事情中被祸患纠缠。野鹤有言：“世动化鬼，非临场病阻，即灾祸临身。”占事之成败而世动化鬼，是非纠缠难如愿；求财而世动化鬼，恐财被诈或因财招祸；自占时运而世动化鬼，祸非临近；代占亲朋而动化鬼，所愿难求忧患成真···

---官鬼为用神的事占如果变出的官鬼于日月不衰，则理论上此官鬼为官星

---心态卦中，父母持世动而化官鬼回头生为卦理吉兆，不为化鬼

# 5．世用（应）背向法-

1.世兴前提下，用神动，变出的爻与世爻相合

2.用旺前提下，世爻动，变出的爻与用神相合

3.用神旺相不动，应爻动，变出的爻与世爻相合

# 三凶：

1.用神动，变出的爻与世爻相冲

2.世爻动，变出的爻与用神相冲

3.用神不动，应爻动，变出的爻与世爻相冲

-趋变的爻，卦理上又能回头作用动爻，或两用事爻间彼此可以产生卦理作用关系，彼此分析结果却恰好相反，则卦理为先，世用背向法的正确应用，多在用事爻在动变过程中，没有显示出明显的卦理吉凶趋势条件下使用的，一旦卦象在动变过程中显示出明显的卦理吉凶趋势指引，则必须舍卦意而循卦理

# 6.搭桥趋变法- 1

世用动变出寅木，卦中又另有寅木动变出午火，于是视同于世用通过寅木为桥梁，动变出午火，直接以世用动变午火看待

# 注意四点：

1.吉凶层面两动爻之间，不能存在针对世用目标爻的卦理生克关系

2. 两动各自变爻与自身动爻，也不能存在针对世用的卦理回头生克关系

3.两动变爻之间不能合成三合局

4.两动搭桥最终变出的变爻，属于卦意之法，它并不以生克体现其吉凶趋势，而是以一种物质通过搭桥过渡的方式，变化为另外一种物质的趋向表达，所以搭桥法的吉凶定性在于其变出的六亲，而非其变出之爻的回头作用

# 7.牵连聚合法------

通过爻动把相关各目标爻牵连在一块寓意祸福趋势的卦意分析方法。多用于占行人往返、求聚首会面、缘分瓜葛、或问婚姻感情`

1.动爻与变爻相连---如女占婚姻，得世爻动变官鬼，自己与用神官鬼相结合，是婚姻如愿的卦意吉兆

2.各动爻聚合成三合局---如求财而世爻与用神财爻同在一个三合局内，寓意与财同局得利

3.牵连聚合法的思维方式有时候更加可以扩展到卦象层面---问工作调动一例:井变升

# 8.代入确认法(替身法)------

注重分析与世用同属性或相冲、相合的爻---因相冲者代表对手，相合者代表朋友（问婚姻代表配偶），相同属性者代表同类

---如问求财，世为未土，用神财爻动却与戌土相连互变，则寓意此财被你的同类得去

---如问彼此感情，世爻代表我，应爻代表她，与世相冲的爻代表我的情敌等等，再通过分析各爻的彼此关系信息，预兆趋势吉凶的方法

-变爻通过它的载体间接与世爻产生对比关系，这是卦意预测法中“替身法”的另一种表现模式

# 9.藏爻确认法----

---多见于有指定目标或明确定位的嫁接用神之测-----如问与他能成婚否，找他能得官否，与某方合作能得财否，出国能有事业否```

----用神藏伏不露是此法的一个最明显特征，此时应爻代表目标者，若用神藏伏在与世应爻相冲或相同属性的爻下面，则寓意所求之愿不在此处而在另一处，若用神恰恰藏在应爻之内，则是找对门路的吉利兆头

# 10.间爻阻隔法----

若阻隔成立，即使用神旺相也一样不改凶兆

# 11.冲合预兆法-----

根据与用神相冲者或与用神相合者的变化趋势判断吉凶的卦意分析方法

-----除了与卦理不能冲突外，用神定要藏伏且与用神相冲、相合者在卦中独发动变

# 12.动动相连法-----

在吉凶层面上将动爻与动爻优先连接的分析思路，而不是一味把动兆的作用方向僵化式地指向世爻与用神，此法的应用条件也非泛泛，于存在竞争性质、明争暗斗之类的事占最为适用，也较多用于短事之占

# 卦意分析法之取用神

# 1.用神替身法----

-----.将用神所寄生的飞爻或动爻比作为用神，再与世爻、动爻论彼此的生克关系

-----卦象可透过用神替身法的手段，绕过事卦与心态卦交融混淆的大弯子，清晰地发出吉凶趋势提示

----让事卦用神藏伏在心态卦用神之下，以心态卦的喜神与忧神作为事卦用神的替身，按心态卦的模式显示吉凶趋势

# 3.世应之风采-----

·亲友、同事、同窗、上下级等关系的，还是应套取相应的六亲为用神，只有与对方非亲非故非上司下属、完全无关联利益的，这时候方可脱离六亲的定位取用模式，直接取应爻为用神

·事实一卦中的应爻，无论吉凶或应期细节层面，它都是一个非常重要的信息点，应爻是所有问念的固定点数据库，又是所占人事的无声代言者

·在分析卦象时完全忽视应位信息，后果可能会因此导致严重误判，因一卦中的世爻与应爻，它们是事内整体因果反映的固定点。只要是应爻一动弹，意味着事情的趋势被显出来了，虽然此动未必就是用神，可它在事占的吉凶层面，起码可作为半个用神的节奏来看待。世或交重，两目顾瞻于马首；应如发动，一心似托于猿攀。世爻或者应爻发动，都可能对卦象定性构成重大影响，都是应该十分重视的核心信息

·双核卦象是由于应爻信息因某种指定被强化，而派生出来的卦象，这种卦象赋予了应爻更特殊的身份，将其作用上升到与用神不相上下的核心地位，当卦象动兆与之形成某种对比格局的时候，必须把应爻与用神信息组合应用，才能正确把握趋势定性

特指事占：某意念目标的成败，寄托在某指定的事物方位身上

在上海的事业发展将如何、指定某单位的应聘能否成功、指定某项目的运作是否盈利、指定某器材是否适合我专业的发挥、指定的某医院能否对我帮助···

嫁接事占：某意念目标的成败，寄托在某指定的人物身上

与指定某人合作是否有财运、通过指定某人帮忙能否成事、问指定的某人能否完成我的需求、与指定某人的姻缘····

-这些卦象由于代表意念目标的用神，本身已显示出直接的方向，因此对只注重用神旺衰以及用神与世爻互动的易者，产生分析定性的误导

问入职官生世预示能入职，但是不是所指定的单位能入职呢

问婚姻妻财旺预示能结婚，但是不是与指定的对象结婚呢

问跟随某人学习某项技能，父母克世预示不利，但不利是否就一定是此特指之人带来的呢都不一定的，这时候就必须将用神与应爻综合分析，才能得出正确的预判

--很多易者对于同一事项的不同决策，习惯一项接一项地去起卦预测，当掌握了双核卦象的分析技能，知道特指与目标的关系，懂得特指与同类的对比知识之后，选择其实不用逐一起卦那么麻烦，尽管心里有多套方案，只挑其中最上心的一个选择发问就可以了，心有所念卦有所显，心中既然有多套选择的想法，卦象自然也跟着显示各种选择的利弊，前提是你了解并掌握了特指及同类对比信息的技术

卦意用神法要应用精通，对六爻各类学识的全面兼容要求很高，因为这往往是吉凶与细节、据理与取象、用神与世应之间关系，做到层次分明高度圆融的综合体现

# 月令

月令对爻可旺可衰时要联系实际客观分析------

·月破信息倒影到现实，就是受到冲击已衰败或尚未发生的现状---（所谋已受重创、问病时已病重药穷、问工作时已遭解雇---事情尚未开展、如问生意时尚未开张、出行行人尚未到位、问行人时尚未到埠、占出行时尚未成行…）---用事爻现状有此对应，可断定为月破，否则只能按月扶看

·月合信息倒影到现实，就是曾经受到提携、与人合作、或时不时发生的寓意---（问病时已见名医良药、问工作已有人辅助得益、问生意过去时而得到合作支持、或者相关举动过去时不时有发生…）---用事爻现状有所对应，可断定为月合，若实际不是迎合反是受困遭伤，只能按月克看了

·如果实在没办法对证到现状，那土爻逢月冲就一概视为月破；爻被日月相克之中又带合，一律都视为相克就是。虽然这有点儿纸上谈兵的味道，但毕竟固定套路也有它概率上的准确性

# 动破有真破假破之分-----

1.动而逢破，不为真破

2.动而化破，但变爻可回头作用动爻者，不为真破

3.动而化破，但动化进退者，不为真破--（特别是动而化进神又化破者）

4.自占自事条件下，世爻动而化破，不为真破---（若不回头作用，则多通过世爻化出的用忌仇、六亲，定位吉凶---有时也透过变出相关的六亲对比信息，投射出应期细节方面的自身趋向变化）

# 月令在应期层面的应用知识----

# 1.临月建之爻---

·用事爻临月建而得旺，则可能当月应吉；长久之测，用事爻临月建被动兆克伤，虽最终为凶，但在当月内可保无碍---（临月建化回头克时，有时不是当月不受克的预示，反而是当月内被克伤的提醒）

·爻临月建、化月建，若是动而生旺世用，则可能当月应吉；爻临月建、化月建，若是动克世用的忌神，则可能当月应凶

# 2.得月合之爻---

·卦象属凶，而世用得月合，可保数下三个月内暂时无碍；卦象属吉，而世用得月合，数下三个月内应吉

·动爻得月合或带月合，若是动而生旺世用，数下三个月内应吉；若是作忌神冲克世用使之衰败，则数下三个月内应凶

· 一旦当令之爻与与月合之爻是敌人，这个应期法则就不再有用，例如辰月占卜，酉金动变午火回头克，则不再是酉金在接下巳、午、未月内免遭殃，在午月酉金就会倒霉被克伤，月合的应期应用有其明显的局限性，时段范围太长或太短的事占，都不大适用，这点要注意

# 3.逢月破之爻----（应期：实破--临值、补破--逢合、时期出破。。动变之爻月破，则多会应在实破或补破的时日；静爻月破，则不排除会应在时期出破的时日。也有可能是众多的应期信息都一一轮过了才会应事，只是易者往往忽视了这方面知识的掌握）

·用事爻月破同时又受克显凶，则破爻逢伤更重，祸事连连或始于当月

·月令的信息范围不仅只是反映过去已经存在过的信息，它还涵盖了过去已经存在而一直延

伸到当下的信息 。牵涉到过去、现状、未来、定位、强化、另兆、指导及祸福程度的各类信息都有。无论是月建、月破、月合、月刑、月绝、月墓等等，它们在细节层面都具有其个性特征的解读

·月破信息在细节上除了寓意当下发生的伤害之外，还有类似于旬空所表示的目前“尚未发生”的寓意

# 日令

·吉凶层面生旺墓绝可用于日令，但“爻旺附旺，爻衰附衰”，不是一见日令存在生旺墓绝，就一定按照生旺墓绝分析

·静爻得日合才论日合显旺，动爻若被日合，则成为“日绊，分真绊假绊

·爻得旺而动受日冲起正得力，爻休囚而动受日冲（破动），也不为破散，只是事情在开始阶段陷入困境

·变爻得月令旺相而受日冲，称之为化暗动，短期可以成为有用有力的动爻；在月衰而受日冲，则为化散衰败无用-----（长事之测一概以化破论）

·假化散：1.化回头作用之爻（动能中转）2.动而趋亢进之爻（重内轻外）3.自占自事而世爻化散（世爻趋势）

日令的作用时段规律-----而问常事或长事，日令的作用时段多在月外，问短事应在日内，如把时间段比作一条竹竿，月建则管中间那一段，日令却只管头尾两截---短事日转月不转；常事月转日不转；长事日月皆不转、只有太岁动

日令的应期层面知识------（问短事“日转月不转”，当卦中的卯木爻到了卯日，此爻正好值日，在卯日内它就如同司令，受克而不凶，作为动爻则大旺，作用他爻正当其时，但当过了卯日到辰日，卯木爻的这些主宰功能就将失去，又轮到辰土爻放肆了）

1.短事之测，见用事动变爻临日令，快的话当日就会应验（但也有顺延到次日应的，出现这类情况，很多是由于当日不具备应验条件）

----在吉凶上断定趋势即将衰败，当日也可暂时不见凶兆，可抓住一瞬时机趋吉避凶

----短事之测，用事爻或动爻受日冲而成暗动的，在应期层面，都是短期内很快即将有结果的预兆，快则应于当日或次日，慢者也将应于逢值或逢合之日

----短事之测，但凡出现元神用神日破或破动的，即使定性为吉兆，也不大可能急应在当天或次日，因为应期层面是逢破须补，用事爻遭日令破散，起码意味着当天或次日，此爻是动之乏力了。非短占之事，即使言吉也不在近期

2.问常事，若用事爻临日令，需要防范的只是冲破日建或忌神临值逢合的月份，其余月份则可无往不利-----（日月在吉凶层面同功同权，问常事长事应期细节层面不同功同权，值月的功能总大于日，但月令常换日令却始终不变，月令虽是主宰却不能持久，日令力量总在次位却能恒长）

# 日令的细节层面知识-----

1.爻临日建动而生克世用，作用程度明显大于一般的动爻，所预兆的祸福程度更为深刻（有用动爻）

---用事爻得临日建之爻生旺，将见吉运亨通深远

---用事爻被临日建之爻克衰，如雪上添霜，凶急犹久

2.卦中临日建之爻，代表着事态目前的状况，通常可以用之对轨

----动爻或静爻临日破，细节上则寓意目前是动之徒劳、尚未出现或尚未触及的状态

----卦中见动爻与变爻，分别临日建与月建，寓意“日常往来”的常态现状信息，代表过往至今长期维持的一种常规状态

3. 卦中之爻无论是动静，若与日月组合构成“日合月冲”或“日冲月合”，一律被视为是目前存在的现状信息

# 旬空

·旬空只是事物一种状态的标志，并非纯粹是一种旺衰的标志，旬空本身没有固定旺与衰的标签，它随着自身条件的不断变换而定性

·空，反映到现实中就是不存在、消失掉、踏空、没发生、未出现等等象征

假空：旺相之静爻，动变之爻----旺不为空，动不为空，化空不为真空（吉凶层面）

真空：并不是整体衰弱的静空之爻就一定属于真空、不衰弱的空爻就不是真空，在不同的条件真空的定义也有所区别

---用神真空：静而平相得用神亦为真空----不符合用旺世兴

---世爻真空：自占自事用旺条件下世爻在得日月其一生扶状态下出现静空，即使总体衰败也不算真空，只有于日令月令均无生扶，或受别的动爻相冲克条件下的静空，才属于真空

旬空在吉凶判断上的特殊应用 -----利用空暂不应，逢空必填的规律

1.占病用神旬空，病况能短期痊愈或暂见好转-----但无论近病、久病，这种痊愈只是短期性的，并不代表最终趋势---近病者多因病根较浅而痊愈的概率大些，对于久病者而言，暂见好转的短语更为恰当----旬空只是短期吉凶的关键依据，动变是长期或最终答案的一种预兆

2.求财得兄弟持世空亡，短期得财，不利长久之财，先盈后亏-----有小利及时收手，否则得不偿失

3.问行人往归，世爻空亡或用神旬空，都是行人即将归来的寓意-----出空必填，等行人来填实相见

4.占仕途子孙持世旬空，占问短期数日内有结果的，或有功名及身之象，但这不能说是好事，因功名最终会有不继之忧，有得官而终难到任的；有得位却半途被削的；有曾得仕途但中故因病辞职的类似等等，总之结果就是功名难持久维系------自占事摇得静卦，即使用神持世，但遭月破依然是凶兆，问官更麻烦，因官鬼相连，世破则官持世变鬼缠身，所以尤防他变

5.忧患心态之占，喜神子孙逢空，反是忧郁短期内无法了结或事主将长期忧心忡忡的征兆，但这其实不算凶卦，因为忧患只属于暂时的短期现象，子孙最终会填空，事态趋势也终将无恙；若占得官鬼旬空，结果则恰恰相反，短期无虞，只是东窗事未发而已------但有些心态纯粹属于个人心理阴影，子孙空亡有时候反应的是一个人长期挥之不去的心理障碍---杞人忧天

旬空之爻出空有三-----

1．待到值空的时日到来出空，称为填空

2．待到冲空的时日出空，称为冲空

3．待到出了冲空或值空的时段就算出空---动变爻旬空，多会应在填空或冲空等时点出空的时日，静爻旬空，则多应在时期出空的时日。当然这只是概率的问题，不是绝对这样

# 旬空理论在细节层面的应用-----

·从旬空的时空特性、心理特性、物质特性去推敲取象。若见卦中有旬空之爻存在，不管其是动爻或静爻，吉凶判断层面不纳入卦面内容的藏爻、对位爻（变卦中的静爻）等等，都可以取其信息，作为事态过去、现状、未来分析的解读点之一。比如在问病的卦中，出现了代表疾病的巳火旬空，这可能就是病人之前在巳、亥年月生过病的过去事实读点。而空亡作为现状信息解读，它的现实寓意就是目前不存在、或者当下尚未发生---动空寓意即将发生—静空寓意总会发生

·静爻之旬空，也可以作为细节层面分析中，事后的未来信息解读点。根据断卦的七步流程，其中包括了事外与事后分析这个细节内容，因为卦之显象往往很丰富，能显示意念本身之事情的吉凶利弊之余，有时候还会同步释放出与事件相关或无关的另外信息，称为事外另兆；有时候也会同步释放出事件预期之后的潜伏际遇，这叫事后之兆。而卦中某些静爻凸显出来的空亡信号，就是为了提前预告事后某种可能而准备的，所以我于六爻细节分析中，在学界第一个提出了静爻“逢空必填”的概念

·卦中世爻旬空的信息，可取之用作摇卦人心态反映的读象，寓意摇卦人内心不安定、不踏实、心悬着、没把握、底气不足、犹豫不决；心虚、不上心、没诚意、粗心大意、无心装载….

·爻旬空的信息在细节层面，也可以用之比喻各种内空性质的实物之象。例如水爻空可作为水瓶水井之象；火爻空可看作是燃气瓶、塑胶管之象；土爻空可看成是道路下陷或陷阱之象；自古就有“金空则鸣”的说法，所以金爻空也可以是钟鼓长笛之象；木爻旬空可喻作衣柜、书架之象。宅中的大门与窗户，对于四面土质围墙而言，门与窗犹似在密封墙体中凿空的一个窟窿，所以也惯用“土空”来比喻建筑物之内的对外门窗

# 六冲

1.爻之六冲存在于日令---暗动、日散

2.爻之六冲存在于月令---月破---除了寓意破败之外，也可以代表尚未发生的细节

3.存在于两动爻之间，或动变爻之间---反吟

4.爻之六冲存在于动静爻之间

-有用动爻冲克安静之用事爻，被冲克的用事爻不管旺相与否、无论是否旬空，都将肯定遭受伤害，冲克的损害力度得到速度上的助推变得更加锐利，其祸凶快而急速，静爻自身旺相与否在吉凶层面已经没有任何意义，只能在细节层面作为推敲受损程度之依据了，毕竟爻若得旺，即便受损也将有个限度

-动静爻之冲的理论中，动爻只有在不能参与直接作用世用的前提条件下，才会选择去冲动其他静爻-----1.受冲的静爻日月总体旺相与否 2.受冲的静爻是否旬空 3.冲静爻的动爻是否真正有力 4.不再有纯冲与冲克之理论区别

六冲卦在吉凶层面的认识-----六冲卦只代表事情缓急程度与维持时效长短的一种细节性信息。主卦现六冲只代表目前事态的多变与突发性，而卦变六冲，则说明事态未来发展轨迹的短期多变或不能恒久，它们本身都不是吉凶的代名词-----但某些细化条件下，六冲可以作为吉凶的代名词，例如长久之占…..

主卦六冲的寓意------主卦中的六冲信息，应用在细节分析方面居多----也可以在作为吉凶判断层面信息应用，如占病况、疑虑之占等方面….动卦主卦见六冲，只是反映事件客观现实中，存在着近段突发之现状、或因事多变的行为----问事业突起波澜、占寻人或出行正在张罗寻觅、求仕途而忽见机缘、求财运刚遇起伏、占健康而忽然发病等等事在临近突发的征兆

静卦六冲与卦变六冲的功能-----而卦变六冲（包括静卦六冲）反映的是一种短变急兆的未来趋势，同时它也是持久性缺乏的代表信息，短事碰上正合其本性，长事见冲则显出负面来

-------近事之占遇到卦变六冲或静卦六冲，对事件直接的正面或负面影响都不大，但近占出卦变六冲，也是短期趋势变数幅度增大的一个提醒，凡事应尽快解决为妥，否则事件即使有既定方向，其演化之急速多变犹似一只变色龙，不是对方中途生变，就是自己中间变故，令人无所适从

----静卦六冲与卦变六冲，在短期的事占中除了代表趋势多变之外，也带有快而急促的寓意，若卦中的世用配合得力，则可作为成事快捷通畅的信息读点---（特别是金融的短线操作中，静卦六冲与卦变六冲往往成为有利短线操作之象，皆因冲就可以不受干扰地自由进出，而最怕出现静卦六合或者卦变六合，短线者一旦碰上了，进则难进、出也难出徒增烦恼）

--------只要碰见时间跨度较长的事情，静卦六冲与卦变六冲对祸福的直接影响便显露出来，即使占得好预期，也多因有始无终或后劲不济而影响吉兆的成色，当然它的时效性也可以冲淡凶兆的负面影响幅度，所谓事若吉兆好运不长，事若凶兆坏事不久，谋事之测多会有始无终，或半途中止

---作为吉凶判断与细节分析的辅助信息，六冲卦在某些领域作用相当明显。特别在占病方面，古人很早就发现卦出六冲，近病痊愈与久病恶化的几率都相当大

--------另外六冲卦的特征，对于心态卦的正确定性判断，往往能给我们提供重要的辅助信息。心态卦的形成，核心在于摇卦人患得患失的忧患情结，郁结难除谓之合，无忧无虑谓之冲，冲去顾虑忧患之意。对于这类暗心态卦，除了观察子孙、官鬼、世应、用神的位置与动变方法来分辨，有时候难免遇到心态卦与事卦特征模棱两可的时候，这时卦中六冲六合的信息，对正确的卦种判定将起到关键作用

# 六合

1.爻之六合存在于日令----静爻合旺，动爻或变爻被日令所合，则称为日绊

2.爻之六合存在于月令----动变静爻皆为月合，细节上才可论绊

3.爻之六合存在于两动爻（动变）之间----动绊、化绊

# 4.爻之六合存在于动静爻之间----趋合

-- ---纯为趋合不带生克才算真正意义上的动而趋合，爻旺被趋合，得力更旺，爻衰被趋合，立足不稳，反被牵引过去。合中带生、合中带克在吉凶判断层面并不能作合断

六合卦在吉凶层面的应用----问事出现六合卦或卦变六合，并非意味着所占趋势一定就吉利，而是一种说明事态缠绵不息的现状与趋势的信息显示

主卦六合的含意-----所问之事现状已经是拖延良久，或已维持有一段时日，如问病者病已长久不愈、占求人办事却迟迟不得落实、问工作调动纠缠未结、问彼此感情交往多时等等。主卦六合信息多应用在细节层面，与客观事实对证，用于验证卦象显示信息的真伪方面

静卦六合与卦变六合-----事情的发展趋势将变得拖延、缠绵不绝，纠缠难定。多数不能主导吉凶，但或多或少会加深与影响祸福结论的成色

------占长事，吉则吉兆久远，凶则长恨缠绵

------对短占性质的事态影响更突出一些，这种影响有时候甚至可以提升到吉凶层面，干扰原本利好的定性

··若吉六合主要体现在应期的迟缓方面··若凶多是事态短期内将郁结不解的旁证，又或是事情短期内不了了之、被迫中止放缓的预兆

----对于某些特殊的事占，卦变六合甚至可以大幅度延缓既定的动变趋势，使事情变质为吉兆···如在占寿元方面，占得静卦六合或卦变六合，不管世爻旺衰，起码不用担忧有短期的变故，这从某些程度上看并不算是一件坏事

-郁结缠绵的特征在反映心态的卦象中，往往就是忧患难结、疑虑不除的代名词，六合虽也主忧患疑虑，但它并不是心态卦中祸福趋势判断的关键点，一般只看作是旁证性质，以及细节展开时的功能信息

卦变六合六冲的指导意义-------部分变卦的卦辞，其意并不在于显示未来的客观趋势，更象是在指导事主当前应该如何去做-----如卦变师，是指导你应该继续深入追究；卦变需，是指导你应该继续静观其变；卦变观，指导你应该继续留意观察；若问投资卦变家人，是指导你应该管好钱袋不要随意投钱…..可以说变卦的卦辞，它不仅仅是客观趋势的一种解读信息，同时也可能是卦象发出的主观指导信号，我们常常把变卦卦辞，称作是“半个指导卦”

----正因于此，变卦中的卦变六合与卦变六冲，有时候也同样带有这类指导功能，尤其当卦中动变等核心信息定性不利的时候，卦变六合多为事情短期应该暂缓、搁置的指导信号，而卦变六冲则是应该半途中止、中断的善意提醒------不是所有的卦变六冲、卦变六合都带有指导寓意，尤其是卦内显出吉兆定性的卦，变出的六冲六合更多为客观特征的解读

# 六合六冲互化

# 六合变六合----

---拖延不决的现状加上缠绵不绝的趋势

---问的事情属于短期“急件”时，碰上这样的卦就显得有点不和谐，吉事变拖沓、凶事也没那么快应验，变得非常缠人，尤其于很短期的事占方面，六合变六合可以一定程度上介入到吉凶定性。往往代表事情近期内将不了了之、暂时搁置、没有下文

# 六冲变六冲----

---突发冲起之事紧接着又起波澜，刚冲起之事又再被冲散不能维持长久

---谋事刚起波澜又告平息、求财刚到手又被夺去、占孕刚怀上又遭流产、占病患刚发作便消匿无形等等

---短期动荡多变，持久力缺乏，这就使得占事出现六冲变六冲，客观上败事的几率要比成事概率更大

---只是事件状态与时效性的一种表现模式，体现出一种快而急速的环境与变化，并不具备事态吉凶定性的关键功能

---对短事之测或反应瞬间心情的心态卦不会产生多大影响，对于长事之占，则往往导致祸福趋势带有更多的负面掺和性质，即使是世用兼旺，好事也将不长，那么这到底算是好事坏事，就难讲得很了，当然反过来也是这样

六冲变六合---六冲卦的含义，就已经不再是原来的突发冲动的现状，而是代表相关事情已经遭挫破败的过去或现实，如占婚姻已经离异，问财运已遭破败、问事业已被辞退等等，随之变出的六合卦的信息，却是专门针对六冲而言，所以变出的六合也非是原来缠绵不息的趋向意思，而是有目的性地预兆原先的破败将得到愈合的趋势，如占婚姻离异必再有婚，问财运遭破败必又再兴、问事业被辞退必另有续处等等。

六合变六冲---六合不再是延续不止的客观现状信息，而是意味事情目前正处于相合状态下的事实缩影，如问婚姻正在蜜月，问伙伴正处合作，占学业正在研读，求功名正在要位，问生意眼前热闹等等，随之变出的六冲卦却是专门针对六合而言的信息，变出六冲不再是事不持久的征兆，而是有目的性地昭示原先的相合状态将会遭受破坏的趋势，如问婚姻蜜月后见纠纷，问伙伴合作将分道扬镳，占学业正研读却后放弃，求功名在要位却遭罢贬，生意眼前热闹终久萧条等等

---问官非缠身之事、盗贼连绵之患、或者忧患不结之问，遇到卦出六合变六冲，反而是好兆头，预示纠缠不休的烦恼事，终于是时候结束了

---有一种情况，无需分析卦内五行动变的生克作用，直接根据六冲变六合或六合变六冲的卦象，即可定性事态吉凶。前提条件是“就事论事对应现状”。你问这件事情的趋势，现状如果此事正好遭受过破败，那么六冲变六合，则专指这件事情趋势将转为吉利。打个比方，如问甲单位应聘，卦出六冲变六合，这个甲单位正好之前应聘过但又失败了的，这就算是—一对应上六冲现状了，如此这次再应聘甲单位，就有很大机会成功。但若问的是甲单位应聘，之前应聘不成功的却是乙单位，这事不同那事，即使出现六冲变六合，也不符合“就事论事”的条件，不能再单凭六冲化六合信息定性其好坏，要细看卦内五行生克动变了，那这个六冲化六合的卦象，只能应用到细节层面分析，意味整个应聘过程，目前虽遭遇困难挫败，但最后终能如意。如果不符合“就事论事对应现状”的条件，那六合与六冲互化卦象，将不能成为定性吉凶的关键信息，

# 三合局

# 吉凶层面的三合局----

1．组成合局的三爻皆动（包括暗动），都是在主卦位置上发动的，不包括变爻

----要注意暗动之爻力不持久，必须短期把握住机会，这个合局才能发挥作用

2．组成合局的三爻皆动（不包括暗动），必须在卦的初爻、三爻位置上动变的，这动变则包括动爻与变爻

3．构成合局的三爻皆动（不包括暗动），但必须是在卦的四爻、六爻位置上动变，动变也包括动爻与变爻

· 旦三合局形成，具有单爻趋势特征的诸如动变回头克、动化破绝等等无用动爻的定义模式，一概不再适用（世爻的趋势除外）

# 吉三合局----

1、自占事，用神与世爻同在三合局内，形成的不是忌神局，而且世爻没有动变回头克

2、自占事，含用神在内的三合局动而生旺世爻

3、自占事，有用神动生世爻，而世爻构成三合局且没有动变回头克

-----2，3其实是“用神生世”动卦吉利通局的一种演变模式，如果三合出用神局生旺世爻，则等于是直接用神生世局；若形成生旺世爻的非用神局，而用神又处于局内，则相当于是间接用神生世局

4、自占事，动爻构成三合局生旺用神，而世爻得力

5、自占事，动爻构成三合局生旺世爻，而用神旺相

-----4，5即形成“用旺世兴”框架模式的动卦吉利通局

# 凶三合局---

1．自占事，世爻在三合局内但却动变回头克

----三合局世爻（或占他人事中的用神）回头克，不但预示了祸兆的强度，很多时候也预示了凶事的时限跨度，从这方面讲三合局有个特征，就是祸事更多“应在长期不应短期”

2．自占事，用神构成三合局动克世爻（或占他人事中的用神）

----问疾病医药、忧患心态、求财获利、行人往归等，三合成用神局克世兆吉

3．自占事，动爻构成三合局动克用神

----无非是将克伤用神的动爻放大成三合局，说明阻碍成事的力量之强大与蒂固，并且越长

# 期伤害越大

4．自占事，世爻与合局不存在生克关系，用神构成三合局，但世爻却在局外

----这种实际应用率相当高的模式，虽然不能从纯粹卦理上得到解释，但符合卦意分析思路的要求，是卦意分析法中“代入法”与“相连法”的综合体现

5．自占事，在用神对世爻没有产生作用的前提下，世爻没与用神构成三合局，反而与用神性质相冲的爻合成三合局，或三合成与用神相冲之局

----这个凶局模式，其实是基于上述第四点凶局模式的思维拓展，只不过上一点是好事被别人沾上了（合局），这一点却是自己沾上了好事的对立面（合局），等于是坏事了，这又带上点卦意分析法中“背向法”的味道

·思维上要谨记一点，自占自事在吉凶层面上，发动的世爻不会针对任何别的动爻或静爻运转，唯一的运动方向只是自己的变爻，世爻所形成的合局也是一样，这个合局不会去生克冲合任何别的爻包括用神

·假三合局， 指的是卦面中出现三合局的同时，也有冲局的动力存在----冲局者必须是动爻，受冲者也必须是动爻

# 细节层面的三合局----

1.凡在日建、月建、动爻、变爻、暗动位置上有的，只要构成合局的三元素齐全，均可以形成应用于应期细节层面的三合局---指应期迟缓，或是细节层面上祸福程度、吉凶结力、聚散离合的寓意

2.卦里出现可构成三合局的条件，其中两爻发动（只限制在动爻，不包括变爻），但却有一爻为静不动；或者卦中缺失另一爻，则待日月时令转换到冲起静爻、或填补所空缺之爻的时候，就可形成三合局

三合局的应期分析：找破绽，如合局中出现月破、日破、旬空、被绊、入墓等等破绽，应期的具体推理如下：

◎ 局内逢日破月破者,待到值令补破之时可成局--例如巳酉丑局中的丑土逢破，到了值令之丑、或补破之子的时令，这个时期就是合局发力的理论应期。

◎ 局内逢被日月所墓，或陷于动墓者，待冲墓出墓之时三合局就能发挥作用--例如申子辰局中申金被墓，到了冲墓之未的时令，这个三合局就可以成立。

◎ 局内逢旬空者,待填空冲空之时可以成局--例如寅午戌合局中寅木旬空，到了填空之寅、或冲空之申的时令，合局就可以生效。

◎ 合局中的组成元素被日令、月令或动爻绊住的，冲绊之时便是合局的应期

◎ 卦中合局三元素缺一的应期：所缺者若是静爻，冲起静爻或静爻逢值之时，就是成局之期，如卦中亥卯为动，未土是静爻，待冲起未土的丑日，或是未日即可为应期；所缺者于卦中完全不见的，转到补全缺失那个元素的时令，为合局应期。所缺者现于日令的，合局短可应在当天，长可应在逢值的时令，例如申子辰合局，申辰为动爻，子水在日令，短事可应在当日，也可以应验在子年或子月。所缺者现于月令的，则合局一般会应在当月

◎ 如果三合局完全没有破绽，更要把握住“外局喜成内局要破”这个原则。如三合局以外来者的身份生克作用世爻或用神，这叫外局，外局喜成，成则能作用目标爻，没有破绽的外局，待合局整体临值临合之时即算成局。例如巳酉丑三合出酉金局，如果局中没有破绽，到了逢值之酉，或逢合之辰的时令，可以成局作用目标爻。而世爻在三合局内，或世爻与用神均在三合局内的情形，叫内局。完美内局的应期分析重点，在于寻找三合局的破解之策，合局由三个爻构成，理论上将三个爻其中一个给冲掉，那合局就不能成立了，如巳酉丑三合局，逢亥、卯、未的时令，冲去形成合局的其中一爻，合局就不复存在。

# 三合局在细节层面的应用：----

1．占长久之事，长远持久，但若非如此，细节上三合局则可能是事态迟缓应验、或短事被拖长的信息反映，特别对于某些近急的占事

2．事态程度超乎常态，卦有三合，其祸福程度的昭示往往都会变得特别强烈

3．世爻或用神处于三合局内之时，则有受困或者难有作为之意。特别是在行人往归、计划行程、或往来调动等等方面事占中，应用面十分广泛

# 化进化退理论

# 化进化退在吉凶层面的应用：-

化进：寅→卯 申→酉 丑→辰 未→戌

化退：卯→寅 酉→申 辰→丑 戌→未

----作为作用他爻的动力，化进如有源之水，力量源源不绝而有用。化退则后劲不继，终局变衰而无用，有动如无

# 化进化退的特殊应用：-----

化进化退的实质乃所占之事祸福连贯而至、废而渐废或兴而渐兴的趋势，进或退都需要一个缓冲周期，换一个相反的角度思考，如果不具备时间上的缓冲条件，无论进或退都无法兴风作浪了。所以性质上化进化退并不大适用于短事之占，过程越是短暂化进化退的作用力就越少-----恰恰与暗动相反

1、心态卦中无进退

2、短事之占无进退---一般周期在数天内断事之问，卦中出现化进化退，基本上可以确定是真发动假进退。凡出现真动假进退的异象，卦中往往会出现一些确认的辅助信息，如出现进退的动爻临日月化日月、临暗动化暗动等短兆或现状特征

-----短期事占其进退的本性会变得模糊，必须依靠其他信息辅助才能明确其是否在进退

3、往来占中无进退---以摇卦人为轴心，化进为远离化退为靠近---在一些没有明确往来目的卦占中，化进者往往寓意出差与出游， 化退者则寓意“回归”性质的事情，如回国、回乡、回老家等等

# 化进化退的应期分析：-----

化进者逢值（动爻）、逢合（动变爻）、逢进（变爻）应

化退者逢冲（动变爻）、逢退（变爻）应

-----化进化退的各种定式只是一个大框架，实断中还需综合考虑旬空、日破、月破等

化进化退的细节寓意：------祸福连贯而至、所占之事渐而渐颓废、或越来越兴旺的趋势缩影----化进神，求财得财过后又得财，财利不绝；问事业此地如意那方又受重用；卜生意则将各处顺景渐渐发大等等；

---化退神，占病则越来越重人渐被消磨；问工作则或见诸多之不顺；求仕途恐江河日下官运渐离

进神：亥化子，寅化卯，巳化午，申化酉，丑化辰，辰化未，未化戌。

退神：子化亥，卯化寅，午化巳，酉化申，辰化丑，未化辰，戌化未。

# 三绊

·吉凶层面三绊：日绊、动绊、化绊----无月绊·既带绊又带生克，无论是生绊或者克绊，其根本性质都依然为生或克，只是生克中带上了相绊的信息，这有点象纯冲与冲克的理论。一般的事占遇到生克又相绊，因为绊一般都属于假绊，所以在吉凶层面可以忽略这个绊的信息，直接按生克分析就可以

# 真绊：-----

1、卦象构成决定的----外卦或内卦三爻全部动而化绊，或六爻全部动而化绊

----世爻或用神被真绊，不是事情悬而不决拖延无期限，就是事与愿违的尴尬预示

2、问事性质决定的----有着明确的目标期限或行程计划清晰的短期性事占

----实践中出行与行人之占中较为多见，如打算在某日出行办事、或拟定某周出外旅游、或计划某时辰请客会友等等类似问题，卦象出现三绊信息的，则定性为真绊，预示事情在计划期限内难以如愿的征兆

# 两种反映现状的特殊组合：-----

1、爻在日绊的同时出现月破

2、爻动化绊的同时出现日冲月冲

-----此两种不再是被绊，反而属于现状的信息点。大大提高了卦象对客观事态的造影作用

三绊在应期细节层面的应用：----可以论月绊。

1、冲绊----吉断，一般避开冲用神、元神的时令，凶断，冲用、元神之时令应，孕产则宜冲用神

2、解绊---合住动绊两元素为解绊之期，实际上就是只要到了相绊两元素逢值的时令，就算解绊

-----“并不并因多字眼”，意思是说性质相合的两元素皆发动，谓之相并，如果相并的这两个元素其中一方又多出一个，反而就不能算并了，因为彼此的拉力已不再均衡。例如子水丑土动而相绊，卦中若又再多出一个丑土发动，则原先这个绊反而不再成立。这个理论于三绊的吉凶层面也同样有效，只不过因为在吉凶层面的绊，绝大部分属于假绊，所以实际应用范围很窄小。而在应期层面作为解绊的其中一个常见模式，却可以广泛地体现在实例之中

2、磕磕碰碰的寓意，体现出进程不顺畅的特征

# 反吟

# 卦之反吟----

1、卦象反吟

----即主卦与变卦出现对冲，如震变兑卦、兑变震卦之类，凡出现是卦象反吟，必然是六冲变六冲，本身就是短期内反复多变的寓意，只不过如果列为反吟，其反复折腾的变数将会更为剧烈，

2、卦宫反吟

---即是指卦的主卦宫与变卦宫产生对冲。卦宫反吟又细分为内卦宫反吟，以及外卦宫反吟，内宫反吟会因为内因而显得反复；外宫反吟将是外部环境令事情变得折腾

---静卦之中是没有反吟信息的，静卦的折腾程度信息，多通过是否游魂卦、归魂卦来判别

# 爻之反吟-----1、动吟 2、化吟

----无论动吟或化吟，对目标而言，都属于相冲而不相克的类型，即是两动反吟对世用没有构成冲克，动爻化吟也不存在回头克伤

吉反吟---动变相冲却不相克，世用动爻遇之，不为凶相只见反复，动爻也能如常作用他爻凶反吟---动变相冲又相克，世用遇之，多为反复事败的凶兆，动爻遇之，反复之下则最终变得有动如无，沦为无用动爻

# 反吟于应期细节层面的应用：------

卦中不管内外，都起码要有两爻同时发动，才可能形成反吟卦象，两动加两变，总共有四个地支元素，一般到了这四个元素的逢值之期，就是反吟的理论应期--出现反复周折

---外卦反吟，外部原因造成事态反复的细节寓意，可谓身不由己

---内卦反吟，内部原因使事情出现周折，这包括自身的因素、决策、心态变幻造成事态的反反复复

---处于不同动爻的变爻彼此直接相冲，也是反映事态反复的一种特定模式，细节层面，可以通过变爻之间的相冲信息，拓展想象成为“反吟”态势

---我们也可以举一反三，把这种思维模式借鉴到别的理论当中，比如搭桥法、代入法、入墓法等等的细节分析方面。六爻的细节层面允许丰富的想象，需要思维活跃，五行、六亲、神煞、动变、卦形、象辞无一不包。但必须以“易理”为依托，--虽万象缤纷仍须一理贯通

# 应期理论公式归纳

# 常用应期公式归纳：----

（1）爻静者，逢值逢冲之期应事，多用于静卦。主事爻静旺者，多会应在逢冲之期；主事爻为衰弱者，较多会应在逢值之期；有时也会在元神或忌神临值之时应吉凶

（2）爻动者，逢合逢值之期应事。如动爻临酉金发动，后遇辰期或酉期应事

（3）月破者，逢填实、补破或出月破之期应事

（4）日冲者，逢填实、或补破之时应事，当然最快者，也会在当日（期）或次日（期）应事

（5）旬空者，逢填实、逢冲实、或出空期之时应事。例如主事爻临午火旬空，后逢子、午之期应事，或过了子午之期应事

（6）遇合者，逢两冲之期应事；爻遇冲，逢两合之时应事。包括三绊理论的应期思路，也是遵循这个原则，遇相合逢两冲之期冲绊，例如午动变未为绊，后逢两冲之子、丑期应事；也有逢午、未两值之期应事，注意在应期分析上，日月动的冲合都可以算是遇合遇冲

（7）动逢月合者，一般数后三个月内应事

（8）应期单一者，同属性之期应事。如主事爻临子水，本来应该应在子期，但最后却在亥期应事，特别是在属性相同又相连的时期

（9）变爻者，若非回头生克，逢值逢冲之期应事之吉凶；若存在回头生克，逢值逢合之期应事

（10）化进神者，于逢值、逢合或逢化进之期应事。如申动化酉为化进神，后逢值逢合之申、巳，或逢化进之酉期应事

（11）化退神者，逢两冲或逢化退之时应事。如酉动化申为化退神，后逢两冲之卯、寅，或逢化退之申期应事

（12）三合局者，局内凶者破局之期应事，冲成局三爻之时可破局；局外忌者成局之期应事，补局内众破绽之时可成局；吉断者不管局内局外，喜局宜成忌局宜破；三合缺一者，可在补缺成局之时应事，如巳酉缺丑，逢丑期成三合局之时应事。应期层面的三合局条件较为宽松，日令月令也可以与动变，组成应期细节层面的三合局

（13）三墓者，逢冲墓、冲爻或出墓期之时应事。如主事爻临午火墓于戌，后逢辰期、子期，或过了辰子之期限而应事。注意在应期层面，日月动变皆可论墓

（14）三刑者，本身带有事凶意味，特别是疾病与官非之问，多应用于应期细节层面。一般三字之刑的构成，乃巳申寅、酉午亥、丑未戌、丑未辰、辰戌丑、辰戌未等，前两种最为常见。卦遇三刑，三刑缺一者，逢补全三刑之期应事，例如卦中日月动变中，已经有亥与酉，三刑唯独缺午，至午期补全三刑之时应凶。三刑齐全者，逢构成三刑之元素临值之期应凶

（15）反吟者，逢反吟元素逢值之期应事。例如反吟的组成元素是寅与申，到了寅申之期，就会出现反复周折

（16）伏吟者，若为最终目标爻，逢值逢冲之期应事；伏吟若非最终目标爻，则与一般动爻无异，于逢值或逢合之时应事

（17）藏伏者，待其逢值或逢合之期应事，或飞爻逢冲之期应事，古人谓之“冲飞露伏”，其实应期层面，冲飞与露伏可以是各自单独的分析途径。也可逢飞爻空亡的期间内应事，这叫“飞空伏现”

（18）定性吉而爻受克者，待忌爻受冲克之时应事

（19）定性凶而爻受生者，待生旺元素被冲克之时应凶

（20）事吉爻过旺者，逢墓或逢克之期应吉，----太过者损之斯成。例如主事爻临午火，遇旺火的日月占卦，卦中又见旺火动爻，后逢相克的亥期应事，或逢火墓之戌期应事。事凶爻过衰者，逢长生或逢生旺之期应凶。例如某事定性为凶，主事爻酉金于日月动变过于衰弱，则逢生旺的土期，或逢其十二宫长生位置的巳期应凶

21、用神多现，或元神忌神多发者，则宜墓库以收藏，逢重叠多发者之墓期应事。例如主事爻为辰土，卦中巳火、午火齐齐发动生旺，此为元神多发，一般逢火爻之墓的戌期应事

# 应期推断的常用原则：------

1、吉断吉应凶断凶应

2、盯面不盯点----

-----利用应期理论，尽量将应期锁定在某一范围内，理论应期过多的时候，我往往会按多现、就近、相邻等原则来筛选，理论应期过少的，我会通过相同属性原则来补充，将应期备选控制在多点的某个时段上，因此实用效果理想

3、应众不应寡---

-----从各个角度分析出来的理论应期，若有重复出现的，那这些重叠信息，成为真正应期的概率往往会比较大

4、应早不应迟---

------不要忽视最早到达的理论应期，由于它最早面临，往往就是它成为事实应期的最大优势所在

5、应邻不应单----如理论应期是卯、巳、午、酉，因巳午相邻，所以巳午成为实际应期的机率更大

# 应期之测的某些注意事宜：-----

-----首先要结合事情的实际情况，定好主应期基数然后再去断应期，应期若再深究，则分有主应期与细节应期-----细节应期按主应期元素逢值、相冲考虑

---如问行人，算到主应期应该在戌、亥日归来，那么再细推更具体的时间，就可以推在戌亥日的辰巳时、或戌亥时归来

# 对应期产生变速作用的特殊信息-

1、用事爻的明动冲起与暗动、带日令而动、卦变六冲等等，这些元素都可以成为应期的加速器-----占长事而用神暗动，乃是慢占快应之兆，暗动把事态的应期提速到更短的时间内应验

2、三合局、三绊、卦变六合、甚至变出渐卦恒卦等等，都可能成为应期迟缓的信号

# 根据卦象分析过去现状

卦象信息与现状对证的方法------

（1）日令与月令----日月为外部环境，为过去、现状的信息（月令：过去曾经发生或过去一直延续到现在）

---（1）日令月令生扶世用，是过去或目前获益受辅助的现状信息

---（2）日令月令克世用，是过去受损或目前正在受损的现状信息（月令休囚，虽然信息不

算突出，但也可以作为现状分析点）---日月同克，外部环境很不利

---克世为吉利的事占，出现在日令月令相克，照样不吉，如求财，出现财爻在日令月令克世，代表曾经或目前因财困损受累，只有卦中的财克世，才是有财的征兆

---（3）日令月令相合世用，相关、纠缠、合作、时而发生，要依据生合或克合的性质、主卦信息、卦中动兆定性方向等，再作出向好向坏的现状推测

------如主卦出现六合困，再加上日月合世爻，因为主卦也是现状信息，所以日月之合，也跟着趋向于过去现在被纠缠的解释

---（4）月令与世用相冲，一是已经受破损、或全无效用的现状信息，二是代表未发生的现状寓意

---（5）日月为长生，之前或目前长久存在；日月为帝旺，目前处于鼎盛阶段；日月为墓，之前目前正受困被墓；日月为绝，之前或目前绝迹无望

（2）带日令月令的静爻、动爻与变爻------过去、现状的信息

--(1)动变同时带着日令与月令，叫日月往来--如日常经营或上班，或每天爬山、每天两地奔波等等，几乎是每天都在重复做这样的

--(2)主卦静爻的藏伏、对位爻带日令月令，也可以看作是现状信息的一个标志

（3）暗动之爻----

--(1)暗动也代表正在发生，特别是冲空模式的暗动，更是明显的现状信息，因为冲空即时生效---主卦静爻的藏伏、对位爻出现暗动，那么这个爻，是主卦之爻为现状信息的标志点

--(2)动爻日冲在细节上，若动爻受月令生扶而旺，这时受日冲为暗动，代表正在发生，或者即将发生；若动爻受月令克损，这时细节上受日冲为日破，代表暂时不见效、未发生

--(3)静爻日破也属于现状信息

（4）爻于日月出现“冲中逢合”，为事态过去、现状的信息反映。卦中动爻、变爻、甚至静爻，出现日合逢月冲，或者日冲逢月合，这爻都算为目前存在的现状信息---可扩展到伏藏、对位爻

（5）主卦的卦象---主卦寓意现在，变卦代表将来。如果是静卦，这个卦象信息，既代表现状又说明未来趋势。如求财得动卦，主卦出现损或大过，不是说这事最终亏损，而是说明目前存在损失

（6）动卦中的持世之爻---不是反映事后的最终结局，就是反映现状信息

---如果问求财财旺世旺，但兄弟持世，那么就要问事主目前是否陷入亏损或花费，如果是，这兄弟持世就代表现状信息，如果不是，得财之后，往后总有破财之虑

（7）旬空之爻----

--卦中动变之爻、静爻、世应用神旬空，都可以反映为目前尚未发生、或目前不存在的现状信息--可扩展到对位爻、藏伏之爻

# 透过卦象懂得“读心术”

心念爻---只隐藏于卦象中世爻所在爻位、以及君爻所在爻位之间

1、若世爻不动，其对位的爻又与世爻不一样，世之对位爻为心念爻

2、若世动或世之对位与世一样，又世之藏爻与世爻不一样，世下藏爻为心念爻

3、若世之对位、世下藏爻都不为心念爻，又君为静爻且与世爻不同，君爻为心念爻

4、世位不成立，君位又动，又君下藏爻与世不同，君之藏爻为心念爻

般情况下：心念爻的作用，是为细节层面分析的开展，它与吉凶定性程序是完全割裂开来的，心念爻只是事主的一种想法，而人的心念与问题，往往不会绝对同步的，所以根据心念爻定位用神，属于本末倒置的错误思维。心念爻对于断卦的作用，只在于某一方面的完善----问病，心念爻出现官鬼，担心的反映，出现子孙，求医药的想法，出现用神之墓，想知道病危应期

# 如何区分事卦与心态卦：----

（1）事卦题目，却不具备事卦的用神特征，反而符合心态卦用神特征，才去考虑心态，如问事卦，事卦用神不上卦，或事卦用神属于假用神，反而心态信息明显

（2）心态卦题目，心态用神特征不明显，反而事态用神特征明显，才去考虑事态

（3）事态题目，具备明显的事态用神特征；心态题目，具备明显心态的用神特征；按题解就是了

（4）遇到心态与事态信息都很明显的时候，才用到心念爻去鉴别，(特别是世化鬼、世化子孙、世应子官发动，心态与事态方向矛盾的卦象)---子孙与官鬼如果出现在心念爻中，多为心态卦

# 真假用神的分析辨别

“真用神”共有五种特征供鉴定之用----世应、两动、日月、空墓

（1）有动爻作用的---用神被元神发动相生，或被忌神发动相克

（2）用神为动爻（包括暗动）---或为动之变爻，其载体是直接针对世爻显示吉凶的

（3）用神临世应---静卦忌神持世，也属于真用神信息---临君爻的的同时又为心念爻，也算真用神

（4）用神与日月相关的，如值日、值月，合日、合月，日冲、月冲（月破）

（5）用神旬空、入墓-----爻不会无故而空无故而墓，爻于日、月、动、变出现空墓，必与事态相关，为真用神

------五项特征没有一项符合的，基本可以确定这个所谓的“用神”其实是假用神

# 客观取用法主要有三点要掌握：-----

(1)懂得真假用神的辨别，思维进一步的话，可以根据卦象重点显示形式定位用神

------真假用神，可有效发掘出一部分真实心念，但不是全部，更多的真实心念是通过重点显示的形式得以体现

------这个重点主要体现在：1.信息显示的重要程度 2.简单直接程度-----以动爻直接作用（直接对比）、用神发动、用神持世、用神临应爻、静卦的用忌神持世等方式，直截了当地表达出来

(2)根据事态的客观现状、客观性质、个性元素，灵活取用

-----取与客观事实相对应的备选爻为用神，特别在用神多发的时候，懂得对证现状取用神，因为卦象是事情的信息倒影

(3)懂得心态卦的用神特征

----对事情关心过度，可能变成对事情的担心，于是卦随念转，事态卦变成了心态卦，卦象的信息也会出现明显的心态特征，而事态特征反而显得单薄

注：伏神、飞神都不破不空，藏伏的其实是假用神

# 有别于事卦的心态卦

明心态卦----提出的主要问题，已经直接地反映出忧虑、疑虑、忧患、怀疑

----如我忧虑这举措可行否、或担心会否发生什么事情、或质疑某事物真假等等

# 主要存在如下几种模式----

# 1、针对世爻之动

---（1）子孙发动生世、子孙持世受动爻生旺、子孙发动克世鬼，皆为放心喜兆

---（2）官鬼发动生世、官鬼持世受动爻相生，皆为有患无实忧之好兆

---（3）子孙持世受动爻相克代表忧患，官鬼发动克世，实患大凶之兆

# 2、世爻之变

---（1）世变子孙为放心，世变鬼为忧患，世变官鬼回头克，实患大凶之兆

---（2）世变子孙回头生，放心喜兆；世变官鬼回头生，有患无实忧之好兆

---（3）子孙持世变回头生，放心喜兆，变回头克有忧

---（4）官鬼持世变回头生，有患无实忧之好兆，变子孙回头克，短事之忧为放心吉兆，长事之忧则放心又反复，最终伤身

# 3、与世爻无关的独发之变

---（1）官鬼独发有用为忧虑；官鬼独发变衰为放心喜兆（回头克、化绝化破等等），动爻独发变鬼终为忧患

---（2）子孙独发有用为放心，子孙独发变衰为忧患凶兆；动爻独发变子孙放心喜兆

# 4、与世爻无关的多发变动

---（1）子孙发动克伤发动之官鬼，忧虑可去

---（2）多动连环生旺子孙无忧，连环克子孙不利

---（3）多动连环生旺官鬼有忧，连环克官鬼有利。

# 5、针对静卦

---（1）子孙持世寓意放心，官鬼持世代表忧虑---逢破反断

---（2）卦面无官鬼爻，又应爻临子孙，代表无忧；卦面无子孙爻，又应爻临官鬼，代表有患

---（3）卦非官鬼持世而子孙暗动，代表短期放心；卦非子孙持世而官鬼暗动，代表短期忧患

---（4）子孙或官鬼持世无根，代表趋势不持久，区分长忧短忧，分别判断

---（5）静卦世应为重点，子孙或官鬼持世，为定性的解读重点；若子孙官鬼非持世，则子孙官鬼临应爻为定性解读的重点

# 6、关于旬空---不论子鬼发动与否

---（1）子孙旬空代表空忧患，或短期忧虑未去---短事之占，子孙旬空不是好预兆，往往代表忧患依然在短期内存在---长事之占，实际上并没有实患，但往往是放心不下的心理反应

---（2）官鬼旬空代表短期忧虑未至，但始终会来---短事之占，官鬼动空旬空，短期反而会无碍---长事之占官鬼动空旬空，虽短期无患，但祸患终会来

# 判别暗心态卦---表面是事卦，实际上属于心态卦

（1）事卦信息淡薄，如事卦用神藏伏不露，或事卦用神属于假用神，或事卦用神位置蹊跷无法清晰解读

（2）与此同时，心态特征却很明显，如世应子官、世变子孙、世爻变鬼、子孙或官鬼发动注意：事卦信息特征显著的卦，即使具备某些心态特征，直接按事卦思路分析就可以，只有事卦信息相对单薄的同时心态信息彰显才按心态卦分析

# 事态心态难以区分时：---

---事卦信息很直接显著，心态信息也很直接显著时，特别是世化官鬼、世化子孙之类，这类卦需要通过心念爻的信息来鉴别---心念爻带子孙官鬼的，属于心态心念，心态爻属于其他的，属于事态心念

遇到官鬼为用神的事占时：----如功名之占，丈夫之占，事业之占

---（1）若为静卦，出现心态卦的几率非常少，如得静卦官鬼持世，往往代表有功名，而静卦子孙持世，则代表没有功名，按事卦思路分析就可以

---（2）当世爻变出官鬼的时候，这类卦最考人，按事卦，应该是世变用神，按心态卦，这应该是变出忧患。官鬼者，官与鬼并存，旺则为官，衰则是鬼。所以官为用神之问，出现世化官鬼，要看此官鬼在日月的总体旺衰，旺则是官，衰则变鬼

# 心态卦的类型----

（1）担心忧虑之占

（2）犹豫不决之占

（3）怀疑与疑惑之占

（4）主观喜厌、主观倾向之占

---子孙代表主观的喜好，官鬼代表主观的厌恶。多用于测竞技球赛或者竞选，主观喜好的球队、竞选者结局以子孙代表，主观反感的球队、竞选者结局以官鬼代表，或大众热门为子孙，大众冷门作官鬼````如此类推

---子孙旺动往往代表了自己如愿的趋势，而官鬼旺动则代表自己厌恶的趋势即将发生

注：有的时候，暗心态卦中甚至会同步显像出事态与心态两种信息，预兆方向一致当然也有个别暗心态卦象，其事卦与心态卦信息显像同步，彼此预兆方向却完全相反

# 日月与变爻为用的分析

# 变爻为用神----

# 1、世爻变出用神

---（1）世变用神回头生，事吉且受益

---（2）世变用神回头克，事凶受损---即使问病子孙回头克世、问财妻财回头克世亦凶

---（3）世变用神无回头生克，为卦意分析法中的“世变用神”，吉兆---即使变爻用神遭遇月破或日破，都不按化破看，无碍吉兆成立，破只是应期的信息

# 2、动爻变出用神----带生则吉，带克则凶，带废则败。

承载爻能直接作用世爻的几种情况：

---（1）带生--承载爻不被回头克，动而生世，吉。若该动爻趋合世爻，还要看世爻旺衰

---（2）带克--承载爻不被回头克，动而克世，凶。即使用神直接克世为吉兆的特殊事占，变成间接克世后，一律要按凶兆看---受动变之伤，向后终难称意

---（3）载体属有用动爻，变爻用神也跟着有用；载体属无用动爻，变爻用神也一块被作废（化破化绝化回头克）---不管动爻生世或克世，只要动爻变用神回头克--自断归路

# 承载爻不直接作用世爻的几种情况：-

---（1）承载爻不生克世爻，成为间爻阻隔，卦意凶兆

---（2）承载爻不生世爻，反生应爻、与世爻相冲的爻、或与世爻同属性相的动爻连接相生，事不成---等于是用神与他人相接，却与自己无关，在竞争性质的事占中，这类情况最为常见

---（3）占彼此关系的，承载爻不生克世爻，可以直接把承载爻看作是用神的替身，直接与世爻对比得出结论

---（4）承载爻独发有用，不作用世爻，不在间爻，这可以是最终成功的吉兆

注：

-----若独发的承载爻不指向世爻，那么动能方向就是指向变爻，就不再存在动爻的有用无用之别，这叫“动始变终”，尤其在独发的动兆中，这个动兆预示事态的整个始终趋势。如求财，世爻官鬼兴旺，动爻兄弟酉金独发，变出用神财爻寅木，动始变终，这是先破财而最终得财的预兆，因为变出的是用神，所以不能看作是化绝

-----若变爻用神无法通过载体或发动形式（例如独发，回头生克等），清晰地回答事情之问，那么这些变爻，很可能是假用神

# 日令与月令为用神----

（1）月为用，生世、合世，为吉兆---用神直接旺世型

（2）月为用，克世、冲世、世生月、世克月，为凶兆---用神直接伤世型

（3）日为用，生世、合世，为吉兆，克世为凶兆

（4）日为用，世生日、世克日----套用“用旺世兴”模式

（5）世爻于动兆中受伤，凶---用旺世衰

（6）日为用冲世爻，一般是当日或短期内成事的信息，不论此世爻在月令旺衰与否

----用神在日冲世，只适用于近事急兆，长事之问则不适合取冲世的日令为用，宁愿去取伏神为用

# 了解用神在日月的时态：（1）对证现状 （2）掌握时效

（1）日令可表现状，月令可表过去，当卦中出现日月为用神时，若对应客观事态，正好也是过去或目前正在发生的事情，就可以直接套用日月上的用神，与世爻的直接作用关系，定性吉凶---对证现状明确用神的方法

（2）日令月令本身，还含有时效范围的信息。一般卦中不见用神，才会选日月为用，所以当日月为用神的时候，证明这个用神是有时效性的，月令的时效范围就在当月。而日令的实效范围，则短在当日，长在月外----故断卦时要注意是属短事还是长事

----长事之问月转日不转，用神在日令的，可以长期生效，用神在月令的，值月的时候才会生效，如占长事，寅月卦得用神在月令，长占急应之兆，应在当月赶快去办，过了这个寅月，那就要等到下一个寅月了

----短事之问日转月不转，用神在日令的，当日或次日就要赶快争取；用神在月令的，就要争取当月内成事，过时则吉兆全无

注：

---有用神在日月，就必然有用神藏伏，当通过伏神的信息，卦理上或卦意上可以明显地得出结论，但通过日月的用神，信息却颇费周折，这时候应该套取伏神为用，更为准确。因为卦象往往会以最简单直接的方式显示事态趋势。例如问某人病，用神在日令，同时藏伏用神的位置，飞神出现旬空月破，通过“飞空伏现”伏神理论，就是用神短期旺相的信息，据此直接就可以得出结论，那么这时在日令的用神，反而是假用神了

# 特殊日月组合

# 旺相式特殊日月组合----

(1)月建日建、月合日建、日合月建、日合月合（高高共指向）

(2)月建日生、日建月生、月合日生、日合月生（高低共指向）

(3)月建日扶、日建月扶、月合日扶、日合月扶（高低共指向）

注意:动爻被“旺相式特组”笼罩，形式是月旺日合的，这个动爻，在吉凶判断层面就不为“日绊”，而为高层次的合旺

# 衰败式特殊日月组合----

（1）月破日克 （2）月破日墓 （3）月破日绝

注意：一切日冲，都不能定义为衰败元素，冲的本身并没有旺衰的固定标签，冲起或冲散，即使是月破日破，也不能算是特殊日月组合

# 特组的特殊之处----

（1）旺相式特组，面对动变逢克不衰；衰败式特组，面对动变逢生不起

（2）世爻、用神静而不动时，受到特组的笼罩，就会被固定在衰败或旺相的格子上衰败凶，旺相吉

# 特组不能发挥功用的情况----

（1）被动爻生克的例外情况

-----生中带合的动爻，构成三合局相生的动爻，都可以生旺“衰败式特组”的爻；克中带冲的爻，构成三合局相克的动爻，则可以克伤“旺相式特组”的爻

（2）动而化进、化退、化绝、化破、搭桥等等

-----特组不再发挥特殊功用，这时的动力方向是变爻---如：旺相式特组动而化衰，一样是衰，衰败式特组化进，一样是趋旺

（3）避免特组与“太过”卦象的混淆

-----“太过”卦的其中一个组合公式是日月“高低共指向”加上动兆，如果旺相式特组的爻，再加上动变爻相生、化进等；或衰败式特组合的爻，再加上动变爻相克、化退等；就应该按“太过”卦定性

（4）注意特组合的时效性

-----特组的构成，是由日令月令组合而成，而日月也代表着事态的现状环境，这种环境是不会一成不变的，时间越长，外部环境变迁的几率就越大，这种大环境只能在短期内保持稳定。所以卦理上的特组，拖的时间越长，它的特殊功效就越低。而这个“时效”，指的不单是现实中的时间段，而且包含事主心目中对这件事情的时间预期。于是针对特殊日月组合的生克，就有些事后信息的味道

·大家在实践应用中，不要过多去钻牛角尖，例如丑月对未爻，直接按月破看就是，不用太考究现状是月破或是月扶。真正实用的理论，不应该是概全的理论，更应该是简单直接的理论，这样才有利于思维清晰地去分析问题，如果太复杂，反而会起到干扰思路的副作用。论点要简单清晰，尽量不要往复杂的方向去想

# 灾卦的信息提示

（1）日月皆克世爻，又日月入动爻相克（或相关六亲）

（2）众鬼发动克世（或相关六亲）

（3）世爻发动，变出鬼爻回头克（或相关六亲）

（4）日月皆克世爻，又世化回头克---这点是目前佐证实例相对最少的（相关六亲）

----在程度上，它所预兆的事项，属于可能创伤较深的突发祸患，它可以令一件事情的不利预期、或负面影响充分放大，造成事主损伤深重，有些甚至危及性命安危

----在范围上，它可以跳出事主的思维趋势，直接通过灾卦信息，警报近身近期可能会发生的祸事，而完全不理会事主想知道的问题，因为卦象显示有避轻就重的特征，如果事主或其至亲，将有不自知的及身灾祸降临，卦象自然会把更“值得关注”的信息透露出来

灾卦并不可怕，它其实一种善意的提醒，提示某种趋势未来可能带来的潜在危机。只要清晰了解它的现状特征，准确定位它的真实所指，以及预见它发生的可能时效范围，不管灾卦所指的事件跨度有多大，都可以使这灾祸被消灭在萌芽状态，永远不会变成现实

# 灾卦的三个明显特征-----

# （1）灾卦的预示范围

往往离不开问事的范围，就算离开问事范围，都会有相关的信息提示。所以出现灾卦，更多应从事情本身的线索，展开定位分析

# （2）灾卦的时效范围，最长一般不会超过半年，更多的是数日或数月内发生的事

结合问事的时效来分析更合理；也可以通过动爻的性质，例如是否暗动、动爻五行属性、主变卦名等来综合判断事情发生的时间段

# （3）灾卦往往都留有现状、细节提示的蛛丝马迹

通过现状分析点、所带五行六亲、星煞、六神等等信号，一般可以正确地定位灾卦的实际所指

# 应期卦的分析应用

应期卦：吉凶无虞只有应期之虑---注意：有些卦题表面好象是应期卦，实质上却是吉凶卦-----所有动兆不论生克，都只是事情应期的信号，有动兆就会有应期

----诸如占问何时结婚生子、何时迎来事业、何时办事合适、占天时何时雨何时晴、占寿元等等，都属于应期卦的范畴

# 对于应期事例的问占，需要随时注意两点：-

# （1）提防假应期卦

---（1）玄武持世----最具有实效的辨别信号，多寓意事主心中藏有隐晦私念，或别样期盼

---（2）心念爻与卦象(世应动)重点信息的特征敏感把握（这需要卦感熟练）

---（3）不是所有假应期卦都会有特殊信息提醒，也有为数不少的假应期卦，没有丝毫异象

# （2）要洞悉应期意外

---（1）有动兆就用应期，即使是日月动变全克，也有应期

-遭遇应期意外的，绝大多数是静卦，应期卦静而没有得到日令月令之一生扶，称为“缺根”，乃应期意外的特殊信号

好事期盼的应期卦：如问事业婚姻于何时等，卦象往往会通过世用没有任何生扶的静卦，来告知应期意外

坏事预见的应期卦：如问寿命等，卦象会透过世用旺相的静卦，来告知应期意外----这种应期意外对事主来说是好事，所以卦象以世用旺相的静卦形式，告知事主还有很长一段人生时光，别急着知道将什么时候逝去；除此之外的不管是动爻生克世用都不是好兆，都是寿终的

信息反映

（注：应期意外指的是真应期卦，却遭遇没有应期的意外，或应期远还没到该发生的时刻）

# 指导卦

指导卦的具体定义：卦象信息不是显示事态的趋势，而是指导事主应该如何运作方为正确的一种卦象

指导卦产生的前提，是卦象是否愿意发出指导信号，不是诚心学易或尊重卦占之人，一般没有接受指导的缘分

# 指导卦的四种类型-----

# （1）针对世用的化墓之动

----化墓，指的是趋势将朦胧难辨，即是指趋势具有可变性，你从卦中读懂了指导之意，按指导而行，自然可以达到希望的目的，但如果定位错误读不懂指导信息，那么错误的举措也会带来另一种未来趋势

----其实卦变蒙或者静卦之蒙，卦中都应该存在着指导信息，因为这也属于趋势朦胧可变的一种表现模式

# （2）带暗动的动变

----暗动之爻的特征是短期性、现状性，而万事的轨迹演变，皆源自于当下的举措，所以暗动之爻可以成为指导卦的其中一个定性

# （3）带日令的动变

# （4）带月令的动变

-----很多变卦（或静卦）的卦辞，本身也带有指导的含义

# 现状卦

只反映事情或物像目前处于什么状态的卦种，称之为“现状卦”。现状卦的产生，主要来自于问卦意念，一般带有现状特征的问题，都可以归类进现状卦的范畴，常见的以下几类问题，都可以按现状卦思路去分析：

▲ 问某人、某事、某物的目前状态（卦象反映的是这人、事、物定格在目前的信息）

▲ 问地形、地貌的利弊（卦象反映的是所问山势、地形的环境利弊现状）

▲ 问阳宅、阴宅目前的风水状况（卦象反映的是所测之宅目前的风水状态）

▲ 寻找探知某人、某物所处方位（卦象反映的是所询问的人、物目前在哪里）

▲ 自问关于自身健康、感情、事业等等各方面的目前景况、现状处境

# 现状卦主要有如下几种改变：-----

（1）所有信息，包括世应、动兆，都不再是趋势的预示，而只是现状的反映

----水爻就是现状的水，土爻就是现状的土

（2）旬空、月破、日破、动破、变空变破、相合、相冲、三墓等，它们一概都是现状信息----空就是失陷缺陷挖空的现状，破就是损失破损破坏的现状，冲就是对冲冲击对立的现状，合就是粘连合并相交的现状，墓就是埋藏受困蒙蔽遮掩的现状

（3）现状卦中的现状利弊反映，依据的是爻在日月的旺衰，以及爻受动变的作用。现状卦是定格在目前显像反映。它会通过动变以及日月对爻的影响力，来反映该人事物目前的利弊或好坏状态。按动必有因的原理，动兆依然占有信息的主导地位

（4）现状卦的显像重心，在于现状的分布而非得失趋势

-----所以物件得失的现状之问，分析只看重用神，无需再考究世爻旺衰，如：问物件丢在什么地方了这样的问题，它属于“现状卦”的一种，卦象的信息重心在“用神”，世爻的旺衰可以被忽略，因为卦象只告诉你物件可能在哪里，却不管你是否能够找到它

# 特殊时效卦

特殊时效必须同时符合两点条件-----

（1）所占事情要有清晰具体的时间范围

---所占事情的全过程，肯定会在当天或当月令内完结，或明确为终身之占

（2）特殊元素必须在卦中清晰显现

---用神与动变，必须带着日令、月令、日破、月破，或者持世---（被日月来合不为特殊元素）

特殊时效卦一般包括三种：

（1）日令时效卦：事情必须在当天内完结；带日令、被日破的用神与动爻为特殊元素

（2）月令时效卦：事情必须在当月内完结；带月令、被月破的用神与动爻为特殊元素

（3）终身时效卦：事情必须带有终身性质；世爻作为特殊元素

·在日令时效卦中，爻在月令旺受日冲，才叫暗动，在月令衰受日冲，不管其动静，受它爻相生与否，一律视为日破

·旬空之爻，冲空即实，依然算暗动

·月上有气受日冲的，亦算日破---因为论是否有月气要依赖爻的衰旺，这里只能直接论为衰

# 日令月令时效卦要注意的一些事项：-----

（1）动亦为真破，带着日令月令特殊时效的动爻，即使化破、化退、化绝，都不会变废

（2）特殊性只限于动爻与用神，不包括世爻

---如自问本月官运，用神或动变月破，都可视为特殊元素，但世爻破不算，世爻月破，往往带有目前尚未展开行动的现状信息，且自问自事，世爻不需旺相，只要兴就行---注：世用合一受破的时候，等于是用神受破

（3）日令时效卦所问的事，并不固定指摇卦当日之事，还包含“某天”的概念。月令时效卦的概念，并不是指一个月内，而是指月令的涵盖范围

（4）日不管月，月不管日

---事情在当天内完成的，日为司令，日令入爻，此爻不怕动克

事情在当月内完成的，月为司令，月令入爻，此爻不怕动克

---日令时效卦，理论范围上日月都可以覆盖，但日令时效卦，日为司令，月令不是，这叫月不管日事。

----一般的卦世化用神，即使月破定性也是吉兆，月令时效卦，化出的用神月破，就是本月不利之兆。细节应该有出月之说，可以作为事后信息

终身时效卦----------特殊性，不在日月，而在世爻

---现状细节分析角度，持世之爻的六亲属性，可以代表现状或者最终结局

---但终身之问，这个功能可以提升到吉凶判断层面

如：问终身财运，以后功名，一生的婚姻趋势，都属于终身性质的问题

这类卦，持世的六亲属性，为最终的际遇，动变只代表短期的过程际遇

-----用神持世终吉利

-----忌神持世与被用神相克者持世、鬼持世，终不吉

----如问官运，兄弟持世，或者子孙持世，都是最终不利的信息，兄弟持世则多被压制被剥削，子孙持世则多辞职辞退

# 终身时效卦要注意两点：------

（1）明确终身意念的卦才叫终身卦，例如问终身功名、终身财福、终身婚姻等等。还有一种问念也属于终身性质，就是运程已过大半问后运，例如年近退休者问今后事业功名，中老年者问今后财运等

（2）有些依赖持世吉凶定性的卦理，要迁就终身卦来思维重组。如问功名得静卦官鬼持世，但官鬼日破或者月破，按照一般事卦的套路，这是无官的预兆，但在终身卦毕竟是用神持世吉兆，所以这类卦只能迁就终身卦法则，寓意最终有官运，只是目前还未到时机

# 连占卦的特殊原则

连占注意点：

（1）保持客观性（2）注意时间点---（同一天内）（3）注意意念

# 连占卦的特征与卦理应用原则：------

(1)针对前一卦产生的问题、疑惑作专门的回答，包括对卦理的疑惑、对其他要求的补充想法，只要不是因为主观不接受卦象趋势显示、或是为了有备无患主观想多占一卦的心态，其余的客观想法都可以被卦象接受

---连占卦象的针对性，是针对意念想法，而非针对某个人，就算换了人来代占，只要代占之人不是自己独立产生的想法，而是被动依附他人的想法代占同一件事，那么卦象还是疑惑者想法的反映，卦象的信息特征依然带有连占性质

（2）连占卦卦象的信息，会比一般的卦象更为粗放，解卦者无需再以一般卦理的思维去分析，而应以比一般卦象放大一层的角度去定性，如下：

---（1）静卦：凭用忌神持世即可定性，无需再细致分析破与否或者有根与否

凭用神生世克世即可定性，而无需再细致分析用神旺衰与否

心态卦、问病卦，用忌持世、用忌临应爻即可定性，无需再细分世应旺衰

---（2）对于动爻：凭动爻作用方向即可定性，无需再细致分析该动爻是否特殊旺衰、是否变无用、或者变爻对它作用如何

---（ ）对于世用自身变化：凭变爻的属性、变爻回头作用方向即可定性，而无需再细致分析变爻是否受特殊旺衰影响

# 太过卦象的尺度

# 太旺卦-------亢龙有悔

（1）旺相特组+动变：爻在旺相式特组条件下，再受动爻或者变爻相生、化进

（2）日月同时令目标爻旺相+多动变：爻在日旺月旺条件下，再至少同时受两处动爻或者变爻相生

# 太衰卦--------

（1）衰败特组+动变：爻在衰败式特组条件下，再受动爻或者变爻相克、化退

（2）日月同时令目标爻衰败+多动变：爻在日衰月衰条件下，再至少同时受两处动爻或者变爻相克

# 太过卦注意四点：-----

（1）只针对卦中的世爻或用神产生作用

（2）太过卦象的现状信息往往是不对轨的

-----这也成了识别太过卦的一个有用手段，事实上有些太过卦会显得相对隐蔽，甚至可能会出现在静卦当中，特别是对于一些病危病重或高龄病者，这种非典型太过卦更为常见，这就提醒我们，判断太过于否必须结合事情的实际分析，不要太死板于公式，死板于公式套路，只是初学者的思维-----隐性太过卦

（3）不同条件的太过卦，长期与短期寓意均有所区别

-----太过卦同时涵盖了几种寓意：短期趋势进展缓慢、短期趋势停顿、趋势即将反逆转、趋势目的需要较长时间等

要参考所问事情的性质、长短来取舍------

（1）年轻的一般重病患者问康复，与高龄重病患者或绝症患者问康复，太旺的寓意可能就不一样，前者可能是康复需要较长时间的寓意，但后者可能是器满将倾含义，必须参考患者的年龄、患病性质来灵活分析

（2）长期的问占，如问长期事业发展趋势、长期的财运走势、婚姻趋势等，太过可能是趋势短期停顿、长期缓慢的预示----如问长期性的事占，出现世用“太旺”，意思是短期无法向好应吉，必须等较长时间才会慢慢好转，或提醒有利的趋势将出现拐点了，因为太旺就令向好的态势出现加速，旺相就不会持久，拐点就来了

（3）对于某些非成则败的短期事占，更多属于趋势反逆转的预示，如问投标、竞聘等非成则败的短期事占----如问生意短期会不会向好，出现世用“过旺”，就是说生意短期没有改善，维持现状

（4）对太过倾向性作用力的理解

----太过的构成机理，是日月动变出现一边倒的旺衰倾向，只有日月动变的作用元素，都绝对性地显示出或旺或衰的定位前提下，才能套用太过理论

----实际应用中，好些作用元素的作用力并不是绝对性的-----如丑月对子水，可以论合也可以论克，申日冲卦中寅木，这对寅木属于冲起或者冲散，也没有绝对倾向性的，爻旺爻空则冲起，爻衰则冲散，巳火发动冲亥水，巳火对亥水的作用力也无法绝对定性，出现诸如此类这些不绝对性的作用力，我们就不能把它归类到“太过”理论之中

---不能把发动趋合，归类到对世用的趋旺元素，因为动爻趋合理论的本身，并没有固定的旺衰倾向，要视受力者的情况定性，旺者被趋合则旺，衰者被趋合则衰，并不属于绝对性的趋旺元素，卦象也就不存在太过的可能

# 独发、独静与全动分析

·独发：爻之独发，可直接根据动兆的趋势变化，正推或反推出定性结论

# （1）多发的卦：

----动爻链作为生旺世用的元神，此元神动兆被克制或变衰无用，谓之根源衰竭，一般定性为凶兆-----（包括短事之占的暗动）

----动爻链作为克伤世用的忌神，此忌神动兆被克制或变衰无用，却不能据此定性为吉兆，

要依据世用本身的旺衰来定性

# （2）独发的卦：

---不管在何位置，独发动爻作为生旺世用的元神，若有用自然为利，若此元神动兆变衰无用，则是根源衰竭，一般反断为凶兆----（短事之占的暗动也算）

---不管在何位置，独发动爻作为克伤世用的忌神，忌神有用则应凶，若此独发忌神变衰无用，则可以反断为吉

---独发之爻成为间爻阻隔元素时，有用则成为阻隔凶兆，化衰无用则阻隔不成，成为利好的定性信息

# 注意三点：

（1）太过形式的动变定性，无论多发或独发，必须迎合太过理论的定性来变通

（2）心态卦或短事占，影响独发的思维定位

-----爻动而化退，或动而化日破，都属于动爻变衰的一种模式，但心态卦中是没有进退概念的，短事之占很多时候也没有进退概念，这样性质的问占如果出现爻动化退，就要注意变通

有时候短事之占的元神日破，也可以作为反断不利的依据，因为短事之占日冲之爻可以提升到吉凶层面，暗动可以看作是有用之动，日破自然可以对称地作为无用之动

（3）某些带有放弃意念的问占，改变了独发的正统思维

------卦象是意念的成像，某些主观放弃的意念，卦象往往通过相关爻动变废弃的模式来显示，这时不能再按其无用来定位，而应变通地看成是反映“放弃观念”的整体有用之动，吉凶定性也随之颠倒----这样分析的前提是事主必须有放弃的主观想法

·独静：其个性主要集中在细节上，-----内外竞发事必翻腾：事件过程反复多变，且应期往往应在此静爻临值或相冲时

·全动：只需要关注世爻或用神本身的动变趋势，吉凶定性便大体了然

# 近占远占的思路区别

近占、远占的范围：以日为主导应期的事，可以明确为近占；以年为主导应期的事，可以明确为远占。而有些事情的趋势，却是从短期到长期都必须跟踪与了解的，此时长短期的分析区别就显得更为重要了

-----不管明冲或是暗动，冲都只会反映短期趋势，对长期趋势没有用武之地

近占、远占的区别主要从以下八个方面去阐述-

（1）明冲之爻的长短占概念区别

----（1）明冲之爻多适宜反映近占，远占力不恒久反复波动---包括明动之爻直接冲世用，或动爻冲动静爻由静转动去作用世用

----（2）带有事情紧急或快速的意味，这类动兆很少反映远占之事，当出现理论套路选择的时候，可以根据事情的远近性质，来取舍更合适的理论套路

当然这是在有套路选择的情况下，若问长事，卦象只出现了冲动的信息，这要从三方面的可能性去推敲：

1、事主心中存有长事急成的意念，心有所念卦有所显，卦象通过近期变化的“冲兆”，向事主表达近期利弊趋势的信息

2、卦象在告知事主，事件长期趋势会跌荡起伏不稳定，因为冲的状态不可能长期保持，所以它长期看来意味着反复波动

3、虽为长事之占，但要注意将会在近期很快发生变数，因为冲也代表短期内发生

# （2）暗动与日破之爻的长短占概念区别

---近占遇到用神日破或元神日破，都可以作为吉凶层面短期不利的信息解读，其作用等同于用神或元神明动变废，这在独发卦中对吉凶定向作用尤为明显

---远占之事，元神暗动或日破信息，只能用到细节层面

# （3）复合动变的长短占概念区别

---短占：在有用动兆与无用动兆混杂时，剔除无用动爻留下有用动爻定性

---但若卦中全是无用动兆，动兆是吉凶定性的主宰，表面无用，其实质却是有用的，此时若元神动而变废，可参与吉凶层面

---长占：要注意，元神若为无用动爻，反而成为根源决绝的凶兆依据，实际应用中，还要注意元神无用与被制的思维差别

有用、无用动兆混杂远近之占思路的的例子：

-----假设世爻卯木为目标爻，卦中未土丑土两动，目的是克制亥水的。且亥水与丑土属于无用之动，而未土却属于有用之动。若这卦属于近占，可以先将无用动兆亥水与丑土剔除掉，只留下有用动兆未土来定性。若这卦属于远占，则按元神亥水变废或被制的思路定性

-----假设世爻寅木为目标爻，子申两动，子水化无用，近占思路是先剔除子水，然后凭借申金有用之动来定性。若此卦为长事占，子水作为世用寅官的元神，它的发动变废将被视为根源不继的吉凶定性信号。我们假设这卦不是子水化无用，而是子水的根源申金变废无用，那么短事思路是留下发动的子水定性，长事则视为子水的根源申金断绝，子水也随之失去生世用的动力，长期为凶兆

# （4）六冲与六合卦象的长短占概念区别

-----六冲为急发急退、多变、不继之象，六合为拖延、纠缠、久远之象

占近：六冲短期变数增多，六合短期纠缠难解

远占：六冲远事难恒久，六合远事长久远

近占见六合，其弊端大于近占见六冲；远占见六冲，其弊端大于远占见六合

# （5）三合与三绊长短占的概念区别

------三合与三绊，都带有延迟的特征

近占：遇三合或三绊，其延迟的负面信息将被放大，延误的特性会表现得相当明显

远占：遇三合或三绊，其延缓信息与远占性质相符，反而释放出更为久远的利好提示

# （6）动而化绝长短占概念区别

---时间越长久化绝的定位就越清晰，时间越短化绝的定位就越模糊

远占：动而化绝肯定作为化绝来定位

短事占与心态卦中：不能轻易将动而化绝归类为无用动爻

-----有些个别近占案例，它明明是化绝了，实际却没有体现出化绝的趋势，这可能是短事占中，爻动而化绝的同时，又出现了化日月、化暗动、化冲（反吟）等短期信息点，十二宫本身有从于卦理的特征，当两种理论同时出现，这时候的化绝就不能再按化绝定位了，起码短期内不能按化绝定位。如亥水变出巳火，此动变出相冲叫反吟，虽然名义上也算是动而化绝，但由于同时它也是化冲，所以一般按动之反吟有用看，而不按动而化绝无用看

# （7）日破日建或月破月建之爻的长短占概念区别

临建：带现状意境的“建象”，更适宜在短事中发挥出利好，长事则带有变数的意味

临破：带未来意境的“破象”，更适合在长事中发挥出利好，在近事占中却表现出利差居多

-----如远占终身功名，卦中出现了妻财临月令发动生世官，说明未来的仕途前景，是建立在目前环境条件上的，但未来一生将经历那么多年，仕途的环境与条件总不会一成不变的，一旦改变了这些已知条件，未来官运就会带来变数可能

----如问短事财运，子孙临日破生世财，是元神短期变无用的无财信息；子孙临月破发动生世财，如果事件定要在月令内完结，这可能陷入了月令时效卦而变成凶兆，即使事件时间无限制，月破之动为假破定性为得财利好，至少应期上在月令内较难得到财运

# （8）用神克世的长短占概念区别

---除了四种用克世反吉的特例外，其余一律要区分近占远占的区别

近占：用神发动克世，世爻处于有日月之一生扶的总体旺相前提下，则可以短期论吉利远占：用神发动克世，不管世爻状态如何一律视为凶兆

---如问失物见用神父母发动克旺世，问求官见用神官鬼发动克旺世，这都可以看作是短期事成的吉兆

-----这里一月左右算近占，事件超过这个时限，就不再属于近占范围了

# 远近结合：--

（1）有些卦问的是远占，却是远占近兆的提示

---如问长期财运，出现财暗动生世，卦象正在告知事主应该更加留意短期的财运机遇

（2）有些卦问的是近占，却反映出长事特征

---如问短期官运，出现三合官局发动生世，其实是通过长期特征的信息，提示事主不要急于求成，耐心等待长期仕途机会的出现

（3）很多事情其实是长短性质皆有的

---有些事问的主要是长期如何，但短期如何的细节也需要知道，有些事虽然问短期将如何，但事件的性质造成这事的轨迹必须长期发展下去

# 双核卦象中的应爻作用

·双核卦象并不是说应与用一样重要，而是说特指应爻在吉凶层面能干预事件定性，客体最重要的分析元素依然是用神，主导事情趋势最重要的信息依然是动兆,，但用神旺只代表事情的成与败，还要通过动兆了解这个事情成败与特指元素的关联性

双核卦象：（1）特指事占（2）嫁接事占

特指事占：某意念目标的成败，寄托在某指定的事物方位身上

指定某单位的应聘能否成功-- --指定某项目的运作是否盈利

指定某器材是否适合我专业的发挥--------指定某医院能否对我帮助

嫁接事占：某意念目标的成败，寄托在某指定的人物身上

指定某人合作是否有财运---- ----指定某人帮忙能否成事

指定某人能否完成我的需求-----------指定和某人的姻缘如何

有些双核卦象吉凶层面不可忽略应爻：-----

问指定的某事物是否有碍财运，卦象显示另一事物有碍财运

问与指定的某人可否结婚，卦象显示与另外一人更有婚缘

问求助某人能否达至某目的，卦象显示求助别人才能达到目的

问到指定某单位考公务员能成否，卦象显示另一单位才有考公务员成功的希望

问到某地工作是否会不顺，卦象显示留在本地工作更不顺

问与某人合作是否有利事业，卦象显示与别人合作事业会更加理想

此时只注重世用易产生分析定性的误导---

问入职官生世预示能入职，但是不是所指定的单位能入职呢

问婚姻妻财旺预示能结婚，但是不是与指定的对象结婚呢

问跟随某人学习某项技能，父母克世预示不利，但不利是否就一定是此特指之人带来的呢

-----都不一定的，这时必须将用神与应爻综合分析，才能得出正确的预判

# 双核三个常见类型：------对比信息：相冲、同类、相同

（1）应爻与动兆用神无关型----不干预

---应爻与动变、用神没有任何对比的情形，也不通过动兆与世用关联，也没有发动或被动兆单独作用

（2）应爻与动兆用神对比型---干预

---应爻与动变、用神产生对比关系

（3）应爻与动兆用神关联型---干预

---应爻自身发动、动兆只针对应爻（动爻一般只针对世用作用）、应爻与用神构成整体动兆

若卦的问题是问与丈夫的姻缘趋势，世爻代表自己，应爻可代表特指者（自己丈夫），然而特指者目前已经是自己的丈夫了，所以理论上官鬼寅木也可代表这个特指的丈夫，这时候问题来了，到底谁才真正是指定丈夫的代表元素？遇到这类卦，答案就是动兆为重，哪个爻发动了哪个爻就成为定位的重点，哪个爻就代表真正的指定者

# 注意如下三点：-------

（1）单核与双核的区别

---只有属于特指、嫁接事占的卦，才可套入特指分析思维

（2）双核卦中，应爻只是通用性的特指元素定位，非必然的特指元素定位

----卦象往往是现实的倒影，有些双核卦，特指不以应爻而是以动爻方式出现，因此必须根据卦中元素的现状及特征，与事实对轨的方式来明确特指者

（3）学会把双核卦的特指对比思维，旁通到如何正确定位选择类卦象方面，可以有效地解决多占的烦恼

# 无用动爻的有用解读

·细节层面没有无用有用之说，只要发动了，就具备某方面的提示寓意

# 吉凶层面无用动爻演变成有用动爻的五种情况-------

（1）特殊日月环境、特殊时效卦条件

（2）动化回头克、化破、化绝，现实中有时带有放弃思维、放弃计划、或者客观上的衰竭、绝迹、堵塞之象的一个整体信息，在这种实情条件下的“无用动变”，在吉凶层面需以整组合的有用信息看，来预兆主观放弃、客观衰竭带来的利弊

-----只有主观上心中产生了“放弃”的心思，或者客观实情出现了“放弃或衰竭”之类的对应，卦象才会产生这样的整体解读

-----如问病因，亥鬼变辰回头克冲世爻，“亥变辰”成为一个整体病因之象，来反映血液方面的器官衰竭或者堵塞的病因，吉凶层面成为有用动爻，直接冲克世爻巳火引发短期伤害

（3）短事之占中：日破之爻、动而化退之爻、动而化绝之爻

日破之爻：泛指月衰状态下受日冲的静爻，专门针对世用的“日破”之爻，短占中也是吉凶层面的有用信息，它释放出短期内对世用无力针对的信号，日破无法生世用短期可视为凶兆，日破无法克世用短期可反视为吉兆，特别是独发状态下的日破，定性指向会更加明显

动而化退之爻：短事论动不论退，心态卦中无进退

动而化绝之爻：特别是一两天内有结果或心态类的问占，吉凶层面论动不论绝

-----变绝需要时空演变过程，时间越长化绝的定位就越清晰，时间越短化绝的定位就越模糊

----注意：该类卦大部分按化绝无用看是对的，小部分按发动有用看是对的，说明“动而化绝”在特短事占中特有的飘忽个性，只能通过不同角度的旁侧分析，来综合明确方向

（4）长事之占中：

----作为生旺世用的元神发动变废，短事之占，可以作为无用动爻，吉凶层面忽略

----但在长事占，元神发动变废的信息，可以提升到吉凶层面，世用失去泉源定性为凶

（5）动化绝、化破的同时又化用：

---决定利弊的关键，在于事主的主观性，若放弃了这个方向，等于是动爻变废无用，若延续了这个方向，等于是动化用有用

多是主观意念上出现选择意向，卦象中出现与之对应的一种信息反映。出现这类卦象，是因事主主观方向的不明造成的。变爻可以随作废的动爻一起沉沦，也可以因为变爻是用神，而令原本变废的动爻成为有用之动，理论上没有吉凶的固定方向

# 化解凶卦的办法（重要）

（1）世用回头克，要放弃或停顿才可以避凶

（2）爻受克，但有元神为静的，可以考虑冲起元神，化克为生

（3）忌神动，如果卦中有克制忌神的静爻，也可以冲它制约忌神

（4）忌神动，如果卦中有与世爻相同的爻，冲起它，动动相连的特征，会吸引这个忌神克动爻不克世爻

（5）爻受克，但没有元神，模拟世爻发动是否能改变卦象，如果不能，就要放弃了

（6）忌土在外卦动，或世爻在外卦，不要外出。相反，忌土在内卦动，或世爻在内卦，适宜外出避。

（7）忌神带谋星动，要停止计划之行

（8）忌神或世爻带马星，不要出行

（9）趋势不好，不参与是比较常用的化解方法

# 应爻的特殊分析

（1）应爻是卦中一个重要的信息点，既可以代表事情，也可以代表他人

（2）应爻不动的话，往往是事态应期细节层面的信息库，应爻一旦发动，则是吉凶层面的重要信息点，应爻出现独发，即使不是用忌，也是事情重要趋势的预兆，应从卦理卦意重点发掘

（3）应爻为用神（带着变爻用神），生克冲合世用：

---（1）应爻将跳出复合连动（三合局除外），直接作用世用

---（2）应爻的趋势变化，存在反断原则

--应爻为忌神克世用，变出回头克，反断为吉

--应爻为元神，变出回头克，反断为凶

--应爻带用神趋合世爻，直接作用，无须复合连动

------（留意世爻旺衰）

·终身时效卦持世元素为主导祸福的最关键信息

·动卦中持世元素多只用于细节：（1）事态现状（2）事态的最终结局---不是前者则为后者

# 关于用神克世的思维

非特殊的事占，用神克世多为凶兆：----

（1）用神化衰，为事败之凶

（2）用神旺，克不旺之世，为事败之凶

（3）用神旺，克旺相的世爻

----（1）长久事占，短期或成事，但长久终不利

----（2）短事之占，为事成的信息----用克世为辛苦、不适合、过程不如等的细节体现

用神克世为吉的四种情况：-----

（1）求财：用神旺世兴，用克世反得财---问终身财运例外，财克世为受财所累之兆

（2）问医药：子孙爻克世反吉，用神有用即可，世爻旺衰无需考虑,，但化子孙回头克不吉

（3）问行人：用神克世人即回，用神有用即可，不考虑世爻的旺衰

（4）心态卦：子孙爻动而冲克世爻，是扫去心中忧虑的吉利兆头

--只有心态卦之占是百无禁忌，不怕世动回头克----对世爻旺衰没有要求
