# 失物占报告标准

## 用神取法

失物占（`question_type = "shiwu"`）的**默认用神为妻财爻**（贵重财物之价值属性），而非传统教材中默认取父母爻（物件本相）。

理由：
- 多数客户遗失的是贵重物品（首饰、手机、钱包等），本质上关注的是财物价值能否回归
- 妻财爻动化鬼、合化等信息可直接揭示"物是否已易主"
- 父母爻作为辅助视角，在双视角报告中保留

## 双视角报告顺序

启用双视角时，排列顺序为：
1. **视角一（主）**：妻财 — 贵重财物视角（物之价值）
2. **视角二（辅）**：父母 — 物件本相视角（物之载体）

## 报告输出格式

所有失物占报告应以 `format_readable_report()` 为标准输出格式，七段式结构：
1. 基本信息（占问/卦主/时间/干支/旬空）
2. 卦象（可视化爻图）
3. 各爻旺衰总览（白话）
4. 动变解析（有效/无效动爻 + 世爻受克/受生）
5. 用神分析（双视角各自判断）
6. 综合结论（双视角互证 + 应期）
7. 给卦主的建议（吉凶分流）

技术报告（`format_dual_report`）仅用于研习存档，不面向客户。

## 代码对照

- `YONG_SHEN_TABLE["shiwu"] = "妻财"`
- `DUAL_PERSPECTIVE_TABLE["shiwu"]` 第一项为 `("妻财", "贵重财物视角...")`
