# 断卦强制检查清单

> 主干理论：朱辰彬《古筮真诠》
> 辅助参考：《增删卜易》《黄金策》《卜筮正宗》《易冒》及项目内案例库
>
> Geju原则：吉凶判断只允许一套主理论（古筮真诠），辅助书籍只提供验证和细节补充，不可反过来动摇主干定性。
>
> Ponytail原则：能用一句话说清的规则不用两句话；能查表的不推导；能确认一层再进下一层的不跳步。

---

## Step 3a：静卦三步曲

1. **用忌持世？** → 用神持世=吉，忌神持世=凶
2. **用神生克世爻？** → 生世=吉；克世需分占类（财克世短占可得财，长占为累）
3. **用旺世兴？** → 用神旺 + 世爻得日月一生扶 = 吉

> 辅助验证：静卦三步曲出自《古筮真诠》；《增删卜易》中大量静卦例证可对照；《黄金策》"静遇日冲，必为他人而去"等经验句可作细节补充。

---

## Step 3b：动卦分析

### 3b-1：有用/无用动爻鉴别

逐个检查每个动爻，按以下四类剔除无用动爻：

- [ ] 动而化回头克 → 无用
- [ ] 动而化退神 → 无用
- [ ] 动而化绝（且无回头生） → 无用
- [ ] 动而化月破（非假破） → 无用

**假破四种情况（不废）：** 动爻本身月破 / 动化回头作用之爻逢破（动能中转） / 化进退同时化破（重内轻外） / 自占自事世爻化破

> 辅助验证：化破/化散真假鉴别详见《古筮真诠》月令章；《增删卜易》有多例"动不为真破"旁证。

### 3b-2：动动相连（有用动爻之间）

- [ ] 有用动爻之间是否有五行生克关系？
- [ ] 若有：A 生/克 B → A 能量流向 B，A 不再直接作用世/用
- [ ] B 被 A 作用后，是否还有余力作用世/用？

**三合局优先：** 若三爻动变构成三合局，必须舍弃单爻连动，按合局整体方向论吉凶。

> 辅助验证：动动相连出自《古筮真诠》卦意分析章；《增删卜易》"两动一静缺一待补"用于应期，不在吉凶层。

### 3b-3：动爻对世/用的最终作用

- [ ] 经 3b-1、3b-2 过滤后，还有哪些动爻能到达世爻或用神？
- [ ] 生世/生用 → 吉方向
- [ ] 克世/克用 → 凶方向（需分占类：财克世、子克鬼世、行人克世等）
- [ ] 无直接作用 → 看间爻阻隔或日月旺衰

### 3b-4：世用旺衰独立分析

- [ ] 用神于日月组合：____
- [ ] 世爻于日月组合：____
- [ ] 符合"用旺世兴"？（用神旺 + 世爻得日月一生扶）

> 辅助验证：《增删卜易》中"用旺世兴"实例可作对照；《黄金策》"日辰为六爻之主宰"提醒日令权重。

## Step 4：吉凶定性

- [ ] 动卦：动兆为重，日月为辅
- [ ] 静卦：先持世，次世用生克，再用旺世兴
- [ ] 两个层面严格区分：
  - **吉凶判断层面**：严谨细致，泾渭分明（动兆主导）
  - **应期细节层面**：灵活飘逸，择象套用

> 辅助参考：
> - 《增删卜易》：提供历史验卦实例，用于验证定性方向是否合理
> - 《黄金策》：经典断语可用于辅助定性，但不得推翻已有的动兆定性
> - 《卜筮正宗》：理论纠偏（如纠正古论"动爻帮扶"谬误）
> - 项目案例库（`docs/reference/new/`）：同类型问事的历史案例对照

---

## Step 5：应期推断（吉凶定性之后）

- [ ] 吉断有吉应，凶断有凶应
- [ ] 短事日转月不转，长事月转日不转
- [ ] 动爻逢值逢合应事
- [ ] 暗动者冲中逢值逢合而应
- [ ] 旬空者填实或冲空而应

> 辅助验证：应期公式详见《古筮真诠》应期章；《增删卜易》原注中应期推算可直接参考；《黄金策》"动值合而绊住"用于应期层面。

## Step 6：细节取象（最后展开）

- [ ] 月令兆过去，日令寓现状
- [ ] 六神、星煞、爻位、纳音、十二宫用于细节定位
- [ ] 六冲/六合：急缓信息，非吉凶标签
- [ ] 世应关系：细节层面辅助，不主导吉凶

> 辅助参考：
> - 《黄金策》：大量细节断语（如"龙为青龙主喜、蛇主惊恐"）
> - 《易冒》：古法取象参考（如"卜在何方，察所属之方"）
> - 项目案例库：同类型细节解读对照

## 辅助资料使用规则（Geju原则）

| 辅助资料 | 主要用途 | 使用时机 | 限制 |
|---|---|---|---|
| 《增删卜易》 | 历史验卦、理论验证 | 吉凶定性后对照、应期推算参考 | 不得推翻主卦的动兆定性 |
| 《黄金策》 | 经典断语、细节象意 | 应期细节层面、六神星煞解读 | 吉凶层面只取与动兆一致的断语 |
| 《卜筮正宗》 | 理论纠偏、基础校验 | 基础知识点确认、查漏补缺 | 其中与《古筮真诠》冲突的，以《古筮真诠》为准 |
| 《易冒》 | 古法取象、爻位含义 | 细节层面的方位/人物/状态定位 | 不用于吉凶定性 |
| 项目案例库 | 同类型案例对照 | 定性后验证方向是否合理 | 案例是经验参考，不是理论依据 |

## 常见陷阱（本次案例教训）

1. **世爻是最终目标爻，自占自事不参与复合连动。** 世爻即使发动，方向只趋向自己的变爻，不会去作用其他动爻或用神。
2. **变爻只能回头作用本位动爻，不跨位。** 变爻不能直接生克其他动爻或静爻。
3. **动爻作用方向只有一个。** 吉凶层面，动爻优先作用用神（若有），其次作用世爻；作用了用神就不再作用世爻。
4. **先分清层面再用理论。** 吉凶层面用不到的理论（三刑、相害、月墓、六神等），不要提前套用到定性中。
5. **三合局优先级高于单爻连动。** 三合局成立时，必须舍弃单爻组合分析，按合局整体方向论吉凶。
6. **辅助资料是补充，不是主干。** 《增删卜易》《黄金策》等用于验证和细节，不用于颠覆动兆定性。
