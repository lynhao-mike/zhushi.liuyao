## Step 2：动静分流

## Step 3a：若为静卦，三步曲

1. **用忌持世？** → 用神持世 = 吉，忌神持世 = 凶
2. **用神生克世爻？** → 用神生世 = 吉，用神克世（需分占类）
3. **用旺世兴？** → 用神旺 + 世爻得日月一生扶 = 吉

---

## Step 3b：若为动卦，分层分析

### 3b-1：动爻简化（剔除无用动爻）

逐个检查每个动爻是否为无用动爻：

- [ ] 动而化回头克？ → 无用（特殊情况除外）
- [ ] 动而化退神？ → 无用
- [ ] 动而化绝（且无回头生）？ → 无用
- [ ] 动而化月破（且非假破条件）？ → 无用
- [ ] 非以上四种，保留为有用动爻

**假破不废的四种情况：**
1. 动爻本身月破（动不为真破）
2. 动化回头作用之爻逢破（动能中转）
3. 动化进退神同时化破（重内轻外）
4. 自占自事世爻动化破

### 3b-2：动动相连（有用动爻之间）

- [ ] 有用动爻之间是否存在五行生克关系？
- [ ] 若存在，能量从 A 流向 B（A 生 B 或 A 克 B）
- [ ] A 的能量被 B 吸收，A 不再直接作用世/用
- [ ] B 是否还有余力作用世/用？

### 3b-3：动爻对世/用的最终作用

- [ ] 经 3b-1、3b-2 过滤后，哪些动爻能到达世爻或用神？
- [ ] 生世/生用 → 吉方向
- [ ] 克世/克用 → 凶方向
- [ ] 无直接作用 → 看间爻阻隔或日月旺衰

- [ ] 世爻于日月组合的旺衰：____

## Step 4：吉凶定性

- [ ] 动卦：动兆为重，日月为辅
- [ ] 静卦：先看持世，次看世用生克，再看用旺世兴
- [ ] 两个分析层面的区分：
  - 吉凶判断层面：严谨细致，泾渭分明（动兆主导）
  - 应期细节层面：灵活飘逸，择象套用

---

## Step 5：应期推断（吉凶定性后）

- [ ] 吉断有吉应，凶断有凶应
- [ ] 短事日转月不转，长事月转日不转
- [ ] 动爻逢值逢合应事
- [ ] 暗动者冲中逢值逢合而应
- [ ] 旬空者填实或冲空而应

---

## Step 6：细节取象（最后展开）

- [ ] 月令兆过去，日令寓现状
- [ ] 六神、星煞、爻位、纳音等用于细节定位
- [ ] 六冲/六合：急缓信息，非吉凶标签
- [ ] 世应关系：细节层面的辅助信息，不主导吉凶

## 常见陷阱（本次案例教训）

1. **世爻是最终目标爻，不参与复合连动。** 自占自事，世爻即使发动，方向只趋向自己的变爻，不会去作用其他动爻或用神。
2. **变爻只能回头作用本位动爻，不跨位。** 变爻不能直接生克其他动爻或静爻。
3. **动爻作用方向只有一个。** 吉凶层面，动爻优先作用用神（若有），其次作用世爻；若作用了用神就不再作用世爻。
4. **先分清层面再用理论。** 吉凶层面用不到的理论（如三刑、相害、月墓），不要提前套用。
5. **三合局优先级高于单爻连动。** 若三合局成立，必须舍弃单爻组合分析，按合局整体方向论吉凶。
