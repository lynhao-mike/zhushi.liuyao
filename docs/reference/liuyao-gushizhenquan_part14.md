四是月令有休囚之说，爻生月令或爻克月令，这两种模式都同属于休囚，是爻衰败的象征，但在日令这点就行不通，无论是爻生日令或爻克日令，对于爻而言一概属于不旺不衰的平相。这些日月令基础理论的相异之处，易学者都应该清晰才好。

__动冲与化冲的区别__：动爻受日冲，古谓之“动散”，动而遭日冲散，远古卜书最看重散，有云：“爻旺而动，冲之愈动，爻衰而动，冲之则散，以论祸福，不拘旺相休囚，概以散论”，即爻动而受日冲，不论旺相与否，一律看作动散而衰败无用。可这个理论到了清初古筮实践家野鹤手里，却几乎被完全推翻，野鹤批驳：“独余屡试绝不应乎散，因知神兆机于动，动则必验，何尝见其散也。旺相者，冲之愈强，休囚无气者，间或有散，亦百中仅一二也”。本人又曾将应验卦例专题分析，种种实例证明，后者野鹤之言无疑才是正确的。究其因由，正如野鹤所言，卦机在于动，动则必验，爻若得旺而动，受日冲起正得力，即使爻休囚而动受日冲，也不为破散，因动不为真破，动而日破只是事情在开始阶段陷入困境，最终的趋势仍没定局，所以爻动而受日冲，无论休囚而动、还是旺相而动都不论破散真破，仍为有用动爻，在吉凶判断层面，就当它不散按正常动爻看待就可以。如子月丁亥日占自陈，得“归妹之解”卦

__日令的应期层面知识：__日令在应期分析层面的理论应用，是古筮预测学中的一个较不好掌握的知识点。其因在于应期分析层面，日建的作用时段范围并不同于月建，月建有分春夏秋冬四时与十二月令，各月令在其管辖月内如同司令一般，可主宰操纵各爻旺衰，得月令者，在月内克之不衰；遭月破者，在月内生之不起。但这种作用时效也只能维持在当月之内了，待时日运行到别的月份则时过失效。例如在分析应期上，当轮转到卯月的时候，卦中的卯木爻值月，在卯月内它是受克不凶，若其作为动爻则大旺，正是作用他爻之时，因此爻动能生克他爻的理论应期是逢值逢合应事，因为逢值逢合的时令，正是动爻值旺发力之时，吉凶就将在此时应验。而酉金爻在卯月则遭遇与卯木爻相反的命运，因为被月破，所以逢生不起，弱不禁风逢克即见凶兆，若其作为动爻，在月破之月内也无能力作用他爻。但当出了卯月到了辰月，卯木爻的这些功能就失去了，这就叫时过则衰，酉金爻出了卯月却被解放了不再枯萎，又轮到值月的辰土爻作威作福、戌土爻破败了。而日建的作用时段范围，则完全不是这个概念，时效有长短之分，日令的作用时段规律是：“问常事应在月外，问短事应在日内”，假设我们把时间段比作是一条竹竿，月建管月内的中间那一段，日令却只能管日内与月外的头尾两截。按日月组合在应期层面的转换公式（具体原理推算，详见本著之后的第四十章应期章）：“短事日转月不转；常事月转日不转；长事日月皆不转、只有太岁动”，问短事“日转月不转”，当卦中的卯木爻到了卯日，此爻正好值日，在卯日内它就如同司令，受克而不凶，作为动爻则大旺，作用他爻正当其时，但当过了卯日到辰日，卯木爻的这些主宰功能就将失去，又轮到辰土爻放肆了，所以凡短测之卦，见爻临日令发动，动必有因，快的话或许当日就会应验。如午月丙午日占何日雨，得卦“大壮之升”

__第二十八章: 旬空理论的应用  __

__旬空的定义与寓意：__旬空，是古筮预测学中的一门重要理论，无论在吉凶、应期、细节各个分析层面，都具有相当实用的现实指导意义。旬空也叫空亡，是指以日柱为计算的主体，由十天干的甲字起头，日干之十数为一旬，配合上对应的十个地支，据此套算出属于空亡性质的两个地支。例如，甲子日至癸酉日为一旬，称为甲子旬，此天干地支所组成的十日之内，并不包含戌亥两个地支在内，所以戌亥称为空亡，又名旬空。月有30天，月有三旬也因此而来，因为天干为10数，地支为12数，日柱是由天干搭配地支组成，天干与地支对比，由于地支总是多出两个数，天干地支相配，始终会有两个地支要剩余出来，在旬内消失不见，所以此两地支称为旬空。空，如同字意，反映到现实中就是不存在、消失掉、踏空、没发生、未出现等等象征。

__旬空之爻的简便套算法：__自古对于一卦中旬空之爻的套算方法，有一套快速运算的捷径，当然你若选择用电子工具帮助排卦，则不用你劳神，排卦工具会自动帮你排出哪两个地支是旬空，但人不能总是依赖电子产品，学会套算旬空也是知识积累的一个要求。具体的快速运算方法是：将日柱中的干支同步向前逆推到甲，推算出与甲相配的地支，那该地支的前两位就是旬空；或将日柱中的干支同步向后顺推到癸，推算出与癸相配的地支，那该地支的后两位也属于旬空。具体你可根据日干位置偏前偏后，选择向前推或向后推更为快捷。如丙子日，日干偏前，就选择向前逆推，将干支同步逆推到甲，就是甲戌，戌前两位是申酉，申酉就是旬空。又如辛丑日，日干偏后，就选择向后顺推，将干支同步顺推到癸，就是癸卯，卯后两位是辰巳，辰巳也就属于旬空，如此类推。

__旬空的吉凶层面应用：__古筮理论对于旬空的论述，自古就多如牛毛，应用之法诸古书名目繁多，令人眼花缭乱，有真空、假空、动空、冲空、填空、援空、无故自空、有故而空、墓绝空、害空、安空、破空等等。其中古论里最明显的一种错误观点，就是把旬空与旺衰定性直接划等号，如《黄金策》里写道：“长生帝旺，争如金谷之园；死墓绝空，乃是泥犁之地”，意思是说，当爻临十二宫中的长生帝旺之地，是属于旺相的，可一旦堕入十二宫中的死墓绝位置，或爻临空亡，就不是好事了。把旬空完全定义为凶相，这实在是误导后人的说法，所谓虚实互变有无相生，宇宙间有色必有空，有现状必然存在未来，所以空其实只是事物发展轨迹中的一种时空状态，既然是没有或未发生，本身就存在诸多不确定性，目前没有不代表将来一定会失去，现在未发生不代表以后不会发生。但古人这种“空即失陷”的错误易理观念，却延续了数个朝代之久，一直到明末清初野鹤的《增删卜易》，终于通过不断的实践验证，使旬空的卦理应用开始步入正确的轨道，野鹤说：“旧文必成凶咎，余得验者，旺不为空，动不为空，动而化空，亦不为空，吉凶皆应于冲空、实空之日，余故更之”，归根结底一句话：“静旺不为真空，动变也非真空”。事实证明，这个客观说法，才是旬空理论吉凶层面应用的至真之理。旬空只是事物一种状态的标志，并非纯粹是一种旺衰的标志，旬空本身没有固定旺与衰的标签，它随着自身条件的不断变换而定性。因此，吉凶判断层面上的旬空理论，存在“真空”与“假空”分别。所谓假空，指的是旬空之爻只属于暂时性的空亡，并非真正的空衰，在吉凶分析时你就完全忽略它，当它没空一样看待就行了。确认假空的条件是：卦中旺相的静爻出现旬空，在吉凶层面上属于假空，卦中的动爻或变爻出现旬空，也属于假空，在吉凶判断层面，须如同不空一样看待。如子月辛亥日占远行求财，得“山天大畜”卦
