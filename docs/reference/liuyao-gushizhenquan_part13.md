某易友自占考学，见子孙寅木临月建动而生旺世爻，呵，这不就是“动必有因”吗？自以为必定是过关的玄机。但这玄机偏来跟他作对，结果竟然是没通过，考试不过关预测也不过关，这回真个是输得一塌糊涂，烦恼透顶，急来找我圆卦。我指着卦中的应爻父母酉金问他：“你不会连取用神都不懂吧？”

他答：“我是取父母酉金为用神没错，但你不是说动重静轻吗？我见动爻寅木很明显是去生旺世爻午火，自己受益难道不是吉利的预兆吗？”

我笑：“世爻代表你，用神代表事情，你问事，代表事情的用神是分析的核心，它不吉利，你受益又有什么用？用神必须旺相是占事趋吉的基本前提，这卦的用神酉金没有动变力量作用它，只好依赖日月组合，酉金于子日无生无克不见旺衰，但要命的是酉金于寅月却为休囚，月令导致用神最终衰败，这才是考试不过的关键信息，至于动爻生世，只说明此过程中有令你获益之事罢了。”

他奇怪：“不明白！考试都不过，我还有什么获益？”

我指着卦象：“寅木带贵人，所以这个动爻代表的是人，你是午火，生旺你的寅木化出的是巳火，两者同属性，应该是你的同路人帮忙使你获益。”

他一拍脑袋：“对！对！是有这么回事。我的好同学考试前专门抽时间辅导了我一周时间，使我通晓了不少以前弄不懂的知识点，真的是令我获益不少。”

这两卦古例，我是反复看了很多遍才算弄通。吉凶层面爻得月合者为旺相，这是经得起古今众多实例验证的基础理论，按此卦理分析，两卦中的用神巳火，在得申月合旺的条件下被亥日冲动，应该都论暗动才对，尤其第一卦中用神兄弟巳火因暗动而兴旺，明显就是患者短期病势趋好的定性。原注却解释说用神衰败，原因是巳火被申月刑亥日冲散，而经验告诉我们，三刑理论只适用于应期细节层面，吉凶层面理论中是没有三刑之说的，所以月令申金对用神巳火不能论以刑衰，只能论月合旺相。第二卦事卦用神藏伏而得静卦子孙持世，暗心态卦格局明显，也同样是放心的利好定性信息，但这痔疮病情的趋势为什么最终应凶了呢？既完全想不通其中卦理，更想不通为什么区区痔疮之病会那么快就夺人性命。后来才明白原来误读的根源，是因为自己缺乏历史常识，受这表面字义误导了，古代所谓的痔病不是指痔疮，而是当今对恶性肿瘤癌症的一种泛称，在古代痔病被认定属于不治绝症之一，因而问痔病者之心态，主要是担忧绝症恶化之期，而非想知道能否治愈（明知道痔病是不治的，问也多余），卦象反映的是一个人心念之疑，因此这两卦字面上虽是吉凶卦的问题，其实它们却属于应期卦的概念，区别就在于前者意念集中在病势好坏变化，而后者意念集中在病什么时候恶化。此节破题，一切思路便可迎刃而解，第一卦用神兄弟巳火虽然属暗动，但以应期卦的定位解题，这只代表病人短期暂保无恙，待时令转到彻底能冲克用神的亥月，就是其友痔病凶变的理论应期。第二卦按原注思路，藏伏用神父母巳火不利的原因，是寅木月破导致“破木”不能生巳火，若此不利的应期应该在冲克寅木的申月才对啊，为什么反而在寅木得旺的亥月应凶了呢？可见这思维方向有问题。事实上卦显像重心并不在藏伏的事卦用神巳火，而在卦面上世应子官的心态卦格局。静卦得喜神子孙持世兼且有根无破，按吉凶卦的套路本应是放心之定性，但如果这不是吉凶趋势的问事心态，而是出于担忧不利时点什么时候发生的心态问卦，那么就不能再按吉凶卦的套路走了，应该转而按应期卦的思路，等到代表忧患的鬼爻寅木逢旺的亥月，就是其父痔病凶变不利的应期。

__月令对爻可旺可衰的要联系实际客观分析：__这里要注意的是，若出现爻与月令破中又见相扶，如未爻于丑月，丑未相冲，与月令相冲本为月破论衰，但丑未同属土，又可以论扶旺；或爻与月令合中又见相克，如子爻于丑月，子丑相合本为旺，但丑土又克子水又可论克衰；遇到这类情况应如何决断呢？__古法对此另有一套令人啼笑皆非的理论，谓之：“子与丑合，丑中有土，土克子水，谓之合中带克，克三合七。巳与申合，巳火克申金，谓之克七合三”。似乎古人有种神奇的仪器，可以精确测量到合与克各自的粘力与克力的成分比重，这其中的奥妙倒在其次，那如何据此理论实际应用呢？倒令人无所适从，难道卦卜的推算要如配药一般，须要称一称这个作用力有几两几钱的份量？可这不切合实际又无操作意义的怪论，在古代居然好象还很吃香，__《易冒》也有类似“以日克月生，得生之八；月克日生，得生之七”的所谓“力度精确论”。而这些繁杂无用的理论，到了野鹤老人手里是一概被剔除掉，什么乱七八糟统统的不要，他在《增删卜易》中质问道：“__克三合七、克七合三之说，有问凶吉者以此答之，何以决人之疑__”，而《增》书后学李我平也清晰指出：“七分八分，何以决疑？即如所存占验。占产不增克制，危而即安；占痘叠见逢伤，命之不保。一死一生，七耶八耶”？两人对这种古法理论提出公开的质疑。当然，炮轰归炮轰，问题还是要解决的，野鹤根据自己的实战经验，对类似情况作了理论上的适当改进：“__余得验者，子丑作合，若于辰戌丑未月日占之，而子水并无他处生扶者，即言克不言合，若有生扶，仍以合论。所以子水逢丑月、戌土遇卯月、申金逢巳月，再遇动爻克者，克而又克，何能言合耶？”———意思很明白了，就是说见有爻与月令合中带克，爻得他处生旺的（包括日令与动爻），则论合，否则论克。他还特意举了一个实例，丑月庚辰日占子病，得“__山地__剥”__

__动破有真破假破之分：__非常值得重视的是，对于动爻而言，月破在吉凶判断层面有真破与假破之分。有四类情况属于假破之列：爻动而月破者不为真破，其一也；例如寅木临月破发动。虽动而化破，但变爻可回头作用动爻者不为真破，其二也；例如寅木变出的子水逢月破、或寅木变出的酉金逢月破。虽动而化破，但动变呈现化进化退趋势者不为真破，其三也；例如寅木变出的卯木逢月破、或卯木变出的寅木逢月破。虽动而化破，但属于自占自事条件下的世爻动化破者不为真破，其四也；例如自占自事世爻申金发动变出的亥水逢月破。只有动而化破兼不回头作用、不趋向进退、不属于自占自事世爻动变的才算是真破。真破与假破彼此的本质区别在于，在吉凶定性层面假破始终有发力之时，世用临之依然有力，动爻临之依然有用；而真破则终会导致世用走向衰败、动爻动而变废无功。

对于预见未来的手段，人们可能会更倾向于相信现代科技，或依赖经验数据的累积。前者是科技探测对时空的理性估计，后者则是有现实经验支撑的合理预期。例如天气预报，科学家探测出气旋在目前某个方位的力度、方向、速度，通过对气旋惯性趋势的预计，发出未来某个时点气旋将到达哪里、可能造成什么影响的预警信号。即使无法掌握这样的科技手段（比如在古代），人们也可以根据自然界天气变化前，种种异象特征的过去经验数据总结，预先洞悉未来可能的天气变化。这两种人们情感上信赖的预见方法，都具备一个共同特征，就是基于过去现状“眼见为实”实证条件下的趋势分析。对比之下的古筮占卜术，从来都是那么神秘又那么无从解释，自古尽是些幻影甚至鬼神之说，到当今可信度自然被大打折扣，人们对它的实效性心里面不踏实，或者心存质疑，也属于人之常情。所以，古筮学长期受到社会舆论的偏见影响负面，自有其合理性。不知道易学者们有没有仔细想过，受质疑的根源可能出在我们本身，是我们古筮学本身的理论体系，存在着无法提供实证依据的诟病，它缺乏能够说服大众的实证观点。假设古筮学问之中，有着可以提供事件过去与现状分析的即时对证理论，情况将大不一样，它的理论将更贴近于科学分析，它的应用也变得更加理性，也将获得人们更多的信任与支持。试想一下，当你满腹思虑地摇出一个卦象之后，解卦人在毫不知情的情况下，居然能够点出你所关心事情过去目前的某些事实，甚至心中的真实想法，你会不会感到吃惊和愕然？你还会不重视他对未来预期的种种推论？所以对古筮理论发展来讲，如何做到“眼见为实”太有必要了！此外，再让我们回过头来看看古筮学的原理，虽然理论上，由意念引力产生的古筮卦象，它可以正确反映事件的全息轨迹（包括未来的趋势），但毕竟，古筮学所要分析的目标，不是事件的本体，而是本体的信息投影（卦象），这就引出另一项疑虑，如果卦象显示的不是某件事的惯性轨迹，那岂不是机关算尽也会牛头不对马嘴？准确预见更是无从谈起。所以，验证某个卦象是否真是这件事情的“镜子”，对证现状这一分析步骤将非常重要。利用卦象中的过去现状读点，与事情的已知现实逐一对轨，有效检验显像真实性的同时，又保证了卦中各个元素的正确定位，它对未来趋势的准确预期，起到了航标式的关键作用。“能够透过卦象，正确模拟事件的过去目前轨迹，与现实对应不爽，才能更准确地把握事件未来的轨迹惯性”，这就是我对古筮学分析的思维定位。以同样源于周易五行理论的传统中医为例，据我了解，真正有功力的中医都有一手绝活，他们摸脉问诊后都可以准确地复述，患者的过去与目前种种症状，与患者对质，这就是实证 ——— 对病症过去与现存轨迹的一种模拟。他们往往会主动这样做，这不仅仅是说给病人听的，同时也是说给自己听的，他们明白，要药到病除就必须先准确断症，而准确断症的前提，就是必须通过实证步骤，来鉴证自己的分析方向是否正确。古筮学也应该这样，必须先通过卦象提示的现状特征信息，模拟出事情的过去现状轨迹，主动与求测者对质，取得一致后才进一步推论未来的趋势。这样一来，整个古筮分析预见流程将变得更加严谨与科学，求卦者会更觉得可信，解卦者也会更加自信，提出的解决方案也将更为靠谱。预见的实效性将因此大幅提升。

__第二十七章：日令的功能作用__

__日建的定义与其信息特点：__日令，即摇卦本日的日辰，又称为日建，一年有365天，古代以十天干十二地支相配，组成日柱六十甲子，周而复始循环不息，是为日令。日令也属日月动变四大影响因素之一，《黄金策》曰：“日辰为六爻之主宰”，可见其于古筮术数中的地位昭然，其功能如同月令，能对卦中所有爻同时产生普遍性的作用与影响，也能同时生克冲合卦中所有的爻。当然虽然在吉凶判断层面，日令与月令的功用相当权力也一致，古人谓之“同功同权”，但在作用范围与信息显示上，日令与月令还是有明显区别的，例如我们知道爻生月令、爻克月令这两种信息显示模式，一概称之为“休囚”，是爻在月令衰弱的一种征兆，但换作的对象是日令，就完全不是那么一回事，爻生日令或爻克日令这两种信息，吉凶层面上对爻不构成任何的伤害，当然也不形成任何的生旺，属于无生无克的平相。而日令还有一个月令不具备的特点功能，就是它可以冲动卦中的静爻，卦中的静爻只要受日令相冲，在吉凶层面即由静转动，区别只在于这种动若是无用衰败之动，称为日破或动散，动了等于没动一样，或是用事爻衰败的象征；若成有用兴旺之动，那就很厉害，形成一种凡易学者都相当耳熟能详的卦理——— 暗动。这个理论的正确掌握对卦象的吉凶判断影响相当之大，所以本著的总论篇中，都有专门阐述暗动理论的章节。当今的不少易学者，研易如同囫囵吞枣，片面以为掌握了月令的旺衰模式，就可以把这些概念完全照般到日令上，于是瞎搞出不少“子水爻休囚于寅月也休囚于寅日、子水爻受日令冲动也可以受月令冲动”等等令人啼笑皆非的名堂来，然而古人对于筮法理论的态度，却远比现代人想象的要严谨，“休囚”之说只存在于月令不存在于日令，冲动之理则恰恰相反过来，只存在于日令不存在于月令。千万不要忽视这些很显浅的六爻基础理论，是否正确掌握对预测功力影响深远。

__日令对卦象吉凶趋势的影响力__：在吉凶判断层面，日令作为四大影响因素之一，作用如同于月令，对一卦的吉凶成败甚具影响力。爻临日建、爻得日生、爻得日扶、静爻得日合、爻在日长生与帝旺等等，都会使爻变得兴旺；相反若爻被日克、爻在日绝，爻会因此而变得衰败；爻与日的关系是爻克日、爻生日的，则显示爻于日令中不显旺衰，为平相。虽然日令在以动变组合为趋势主导的卦象中，往往不能成为吉凶定性的关键主角，但由于并非所有的用事爻都会受动兆影响，没有动兆作用的用事爻，日令对一事之吉凶趋势判断，作用就变得相当重要。如酉月乙巳日占升迁，得“萃之否”卦
