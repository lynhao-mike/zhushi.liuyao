__第三十三章: 长生十二宫理论__

古筮六爻中的长生十二宫理论，应该算是相对较为复杂、容易使易学者陷入思维迷宫的一项研究课题，难就难在它的复杂多变上。自古对这方面的论述已经繁多，如明代筮法经典书籍《黄金策》中“养主狐疑；墓多暗昧；化病兮伤损；化胎兮勾连；凶化长生，炽而未散；吉连沐浴，败而不成．．．．．．．”等语，都是与长生十二宫理论相关的总结与描述。古人对长生十二宫论述的全面性，那是无可挑剔的，但问题也随之而来，如果一个技能方法的理论方向性太多太繁杂，不能做到清晰指向，这其实不能称为一个成功的实用理论，勉强去求全等于在做连续自转运动 ———自己把自己转晕头。这就要求易者必须了解十二宫重点的功能特征，懂得舍弃皮毛取其所要，使之能真正适用于实践。我这章节的内容，就是以此作为系统研究的出发点。 

__长生12宫的实用定义：__长生十二宫，也称为长生十二神，是以长生、沐浴、冠带、临官、帝旺、衰、病、死、墓、绝、胎、养十二个位置，代表万物从出生、到成长、到鼎盛、到衰落、到消亡，然后又再重生的整个反复循环过程。其生老衰灭周而复始的原理，与周易的循环变幻不息如出一辙，以此比喻宇宙世间万物的不同发展阶段与趋势。长生十二宫在纳甲六爻中的应用，其推演是以金木水火土各属性之爻，各自十二神在十二地支的不同位置显示吉凶，但土是没有独立的十二宫位置的，必须寄生在其他属性的十二宫位置中，所以土爻的十二宫位置自古从没有定论，至清初，野鹤经实践验证，在其《增删卜易》中明确提出，土爻应寄生于水爻，其十二宫位置与水爻雷同。于是形成当今实用性非常强的古筮六爻长生十二宫体系，具体如下：

问终身功名，世官丑土是用神，官鬼持世发动化出午火回头生旺，定是终有功名的吉兆，元神临旺之时即验，因此在火旺的巳年加纳、午火逢合的未年出仕。动不为真破散，所以丑土即使受日令冲相也无妨碍，何况世官丑土是在月令辰土扶旺的状态受日冲，更可定义为暗动，成为吉兆快应的信息，所以占后只相隔一年便有功名机遇。虽然长生十二宫中，土爻的墓在辰土，这卦世官于月令，论扶旺或论墓衰似有争议，但在吉凶判断中，土爻是没有墓的，即使算有，“生旺墓绝” 的理论也不会存在于月令之中，这只是一个站不住脚的所谓争议。

吉凶层面“生旺墓绝”除了不能应用于月令，可以存在于日令，以及动爻、变爻当中，例如长生于日令、墓于动爻、化绝于变爻等等。对于“生旺墓绝”的旺衰征兆，古论多以爻逢长生与帝旺为旺相标志，逢墓逢绝则认为是衰败标志。正如《黄金策》所注：“长生帝旺，争如金谷之园；死墓绝空，乃是泥犁之地”。其实并非尽皆如此，“生旺墓绝”中的墓就不一定是衰败的征兆，清代野鹤老人就以实证为依据提出异议，明确指出“旺者与动者不为真墓”，可见并非所有陷入墓中的爻都会趋向衰败。古人对“生旺墓绝”中墓的应用学说最为系统，这部分内容将在之后的独立章节予以论述，其余长生、帝旺与绝都有其明确的旺衰性质，吉凶判断中爻遇长生、帝旺都会变得旺相；碰到绝位或化绝，那就非常不妙了。

到此为止，或许有学者觉得这不难明白啊？思路还挺清晰的，为什么会容易混淆呢？关键在于，这些旺衰元素的定性，与我们所熟知的很多卦理与卦意吉凶定性，是相互重复或者相互冲突的，这些会都成为我们思维的“搅拌机”，令我们在判断决策祸福的时候，陷入自相矛盾的困惑之中。我们举些例子：

△金爻长生在巳，那么酉金在日令、或动爻变爻中碰到巳火，到底算长生之旺，还是克伤之衰？

△土爻绝在巳，用神辰土变出巳火，是回头生旺相呢？或是化绝变衰败？

△水爻绝在巳，亥水于日令巳火，是被冲动呢？还是日绝？亥水变出巳火，算反吟？算化绝？申变寅呢？

△土爻长生在申，若世爻或用神是土爻，它变出申金却不幸遭遇月破，是按化长生算旺？还是以化破论衰？

△土爻帝旺在子，如果问孩子，出现土爻化子水，是按用忌互化卦意论凶，还是按用神化帝旺论吉？

我知道，此人这两年的股票市值翻了四个跟斗，所以底气十足站着说话不腰疼。不知不觉到了次年4月份，他手头的股票很不走运，股价已经惨遭腰斩，只剩下50%的市值，所以卦中应爻辰财旬空，与用神戌财对冲，原来这是辰月财产斩半的信号。他当时还保留着些自信：“怕啥，咱输得起，我看股市快见底了，再掉我就追加资金补仓”。

他不了解股市，股市要狠跌起来可真算是个无底洞，想在地板价上抄底？但地板下面原来还有地狱！地狱下面居然还有十八层！！填多少死多少，此人的补仓“子弹”很快就打完了，但股市还是没有止跌的迹象，到了6月底他的股票市值再被腰斩，只剩下25%，事主已经彻底沦为苦主，原本自信满满的他，发现连赚了四倍利润都可以全部吐干净的时候，开始害怕了，但跌到这份上，主动权已经不在自己手上，此时斩仓出局更是心有不甘。

9月初我们又见面了，他先很伤心地告诉我，从高位算起他所全仓的股票，价格竟差不多第三次被腰斩，曾经高达60元的价格，在没有配送除权情况下，不到一年时间内居然跌到目前的8块钱。之后便破口大骂，大骂自己如何利欲智昏、如何瞎了狗眼投资遭受重创，以致给自己和家庭带来巨大压力等等，眼见对方有些失态，只好安慰他要保持信心与希望，股市不可能无休止地延续下探趋势云云。如果他当初能重视这卦“醒脑丸”及时出局，又何至于陷入这种不能自拔的境况呢。

吉凶层面“生旺墓绝”于动爻虽然功用不大，但“生旺墓绝” 理论应用到变爻，其模式却很常见，对卦象定性吉凶作用也非常明显。因为动必有因，动爻往往预兆着事情的祸福趋势，所以动爻是否有用无用就显得非常重要，有用动爻是预兆吉凶趋势的关键，无用动爻则昙花一现，动了就如同没动一样，对事情的趋势没有任何作用，爻动而化绝，却可以使动爻变得无用衰败，其原理如同化破或化散，因终局变衰败以致无法解救。应用在吉凶层面的动而化绝，无非是两种情况：其一是子水动变出巳火；其二是酉金动变出寅木。如午月癸卯日占后运功名，得“艮之观”卦
