__旬空理论在应期层面的应用：__在古筮应用的应期层面，卦中出现旬空的爻，如果有作用，则要待出空的时日才能产生作用，用作忌神者出而为祸，用作元神者出则见福。不少易学者，断卦一遇到旬空的爻就糊里糊涂，不明白《黄金策》既然说“带旺匪空”，为什么应期又说旬内毕竟是空？既然说旬内空亡，为什么在判断吉凶趋势的时候又说当它不空？这其实是把空亡理论在吉凶判断层面与应期细节层面的应用知识混杂在一起，层次不清晰造成的思维混乱。所谓真空假空，只是专门针对卦象的吉凶判断层面分析而言，而到了应期细节层面分析，旬空的定义就有所改变，不论吉凶层面上属真空或者假空，在应期细节层面一律看作是空亡，区别只在于属假空的爻，待出空之时日就能产生作用，而属真空的爻，则已经衰败无用，永远堕入空亡不再有应期了。旬空之爻出空的理论形式常见有三种：

一种是待到值空的时日到来出空，称为填空，如寅木爻旬空。待到逢寅之年月日就会出空。如寅月庚戌日母占女病，得“姤之无妄”卦

__第二十九章: 六冲理论的应用__

六冲，属于古筮六爻预测中的一门重要理论，使用范围广泛。六冲，有分爻之六冲与卦之六冲。爻之六冲即十二地支中的对位相冲：子午相冲、 丑未相冲、 寅申相冲、 卯酉相冲、辰戌相冲、 已亥相冲。卦之六冲则是指周易六十四卦里的六冲卦，何为六冲卦，就是上三爻与下三爻都出现对位相冲的卦，因卦中六个爻都存在对冲状态，所以称为六冲卦。属六冲的卦总共只有十个，较好记忆，分别是：乾为天，坎为水，艮为山，震为雷，巽为风，离为火，坤为地，兑为泽，天雷无妄，雷天大壮。除了无妄与大壮卦，其余八个都是八宫的首卦，也叫八纯卦。自古六爻论著，对无论爻之六冲或是卦之六冲，皆无一例外地视为如洪水猛兽般的凶煞信息，逢之见之多言凶象，其实这是个很大的误区，实际上不管卦六冲或爻六冲，均不是吉凶的代名词，而是一种反映事态急变的信息，宇宙间冲的本质特征，意味的是急快之变速却不能持久，就象急浪疾风扑面而来，瞬间而至又瞬间消失，此冲力可使行舟更破浪前行，也可使船被掀翻，是吉是凶就要看舟船坚固与否了，并非逢卦冲爻冲就一定是凶兆。而世间万物均有其保证常态运行的中轴、频率与轨道，一旦偏离了正常的中轴轨道，就形成了冲，如同一个在竞赛中最后冲刺的人，快而急速！然相冲之后总要向轴心理性回归，因此不管冲起或冲散，其明显特征都是短暂不持久，好象人在冲刺过后反而不能保持往常的恒速一样，所以冲的另一个性质就是不能持久，古论中经常提到的“六冲不持久”说的就是这层意思。故而卦爻中见有冲动或卦现六冲，或是近兆临变之象，或预期事态多变事不能久，特别是爻变月冲变日冲，称为化破化散，是爻不能坚久趋向破败的典型。

    爻之六冲，应用范围十分之广泛，其理论在日令月令、以及卦中动静变爻皆有存在，在不同的位置引申出不同的卦理，主要有如下几种情形，其中很多知识之前的章节已经系统论述过，就当是在温故知新吧：

别看这卦动爻多发，其实只要明了卦理，分析起来颇为容易。问事业官鬼爻是用神，两元神财动生旺世官亥水，此乃事业顺境的预兆。她问的是两处事业的取舍，卦中生用神官鬼成就事业的两个财爻酉金，代表的正是两处事业，那么哪个代表哪处呢？也很简单，月令寓意过去，化出月令戌土的四爻酉金，就是指目前工作的地方，而另一个酉金变出空亡，旬空代表未发生的现状，这指的是她还没去的那个单位。两爻都是生旺世官的元神，可见无论选择跳槽与否，事业发展都将延续向好的预期。若将两者再细加对比，还是可以理出个泾渭头绪，化出月令的那个酉金是爻变回头生，源动力根深蒂固，可知目前工作的单位前景将稳健深入发展，不好的地方是生中带害，总有不满意之处并非一帆风顺。而准备跳槽那个单位，优势在于收入将比目前单位更高，因为酉金变出的卯木正好临禄神，但缺点也很突出，就是酉金变卯木，是动变相冲，为反吟之象，反吟者反复反复，事业前景将非常之不稳定。于是我建议她还是暂时不跳槽为妥，条件许可的话最好考虑身兼两职，这样既可以有效避免外单位前景不稳定带来的负面冲击，又可以主动趋应卦象中，世官亥水动变反吟事业将反复多变的预兆。特别提醒她应以目前单位那份工为主业，因为主卦代表现状，主卦之“鼎”就是目前事业作为核心的象征。她听从了我的劝告，没有答应对方全身跳槽的邀请，只答应兼职。结果几年过去了，她一直工作的公司果然在一步步稳定扩张，最近还申请了上市，她在那儿的事业虽然磕磕碰碰，但总体上还是一年胜过一年；而她兼职的单位，虽然曾给她带来过丰厚的额外收入，但却如“一年不发薪，发薪顶一年”那种，业务境况相当跳跃波动，如果当初真的选择了跳槽，在薪水空档期到底能否熬得住，还真是个问题。这里的酉金变卯木与世官变兄弟，就是卦爻动变相冲带来的一系列预兆信息。

    其四，爻之六冲存在于动静爻之间。一爻若动可以冲静爻，旺者被冲起有力有用，衰者则被冲散踪影全无。实际应用中，动静爻之冲性质上有纯冲与冲克之分，纯冲是动爻对静爻冲而不克，如巳火动冲静爻亥水，是冲而不克；冲克是动爻对静爻冲中带克，如子水动而冲克静爻午火，是为冲中带克；两者的区别，其实就是动冲者带“刺”与否的问题。冲而不克者，靠的是纯粹的冲力，如辰土动冲静爻戌土，对承受者而言，只要身旺则可无伤，如急浪拍船，船坚则反被冲起行之更快速，船破旧不堪浪打来才会遭殃，因此动爻冲静爻而冲中不克者，只要被冲的静爻旺相，受动冲不凶反为吉利，只有不旺的用事爻受冲、或者冲之动爻受制无力冲起用事静爻，才算作凶究。如卯月辛丑日占具文辨恳开销（即文书形式申请报销费用），得“讼之履”卦

这卦是动卦，兄弟持世并不是吉凶层面的祸福判断依据，但破财确实难免，因为动必有因，卦中父母寅木明动，作用目标竟然双向，既可冲用神财爻申金，也可以生旺世爻午火，但在吉凶层面动爻的运动方向只能是一个，同时作用的，理论顺序应该是先用神而后世爻，所以在吉凶判断层面，这卦寅木的目标首选是去冲动静爻用神申金，虽然只是纯冲，但妻财申金于月令休囚，即使得丑日相生，最多也是个平相，作为用神只有旺相才有成功的机会，用神不旺，它就不堪动爻寅木的冲击，随即被冲散，用神被冲散衰败，自然是破财的预兆。虽然不准开销达不到目的，但这也不尽是坏事，因为在细节层面，动爻的运动方向就可以回复其多向性，父母寅木动而生世，对事主来说这是好事，说明上奏章申请即使达不到报销费用目的，该行为也将对自己有好处，估计是可以让上层知道自己的实际付出吧。

纯冲者旺则冲起衰则冲散，这是动静爻之间见冲的通用性卦理。但通用并不代表一切情况尽皆如此，因万事万物都不可能带有绝对性，有通理的存在必然有例外的个案特征。在一种特殊条件下，受动爻纯冲的用事静爻却无须观其旺与衰，一律视为冲起旺相，那就是被动爻冲的该静爻刚好旬空。冲力对象只要属于有形，其弱则难敌，却冲力最怕打在空若无物的不受力之处，千钧力道瞬间化于无形，空爻它就象一张空洞的罗网，不管罗网坚固与否，疾浪带着什么东西来冲击，不仅自身完好无损，还对冲来之物照单全收，原本没力量也因此变得有力度。因此古论将空爻被冲称之为“冲空即实”，意思是说，把自己原本的空给冲实在而变得有力。古今与现实这样的卦例实在数不胜数，这里就不再繁复举例了，有兴趣的读者可以翻阅以前的暗动章。

    但冲克却完全不是一回事，冲而带克者，靠的不光是纯粹的冲力，同时更带着克损的力量，如同尖矛乘着急浪来击船，船坚韧与否都将难免受到挫伤。如酉金爻动而冲克卯木爻、或子水爻动而冲克午火爻，对承受者而言那将是恐怖的。凡见有用动爻冲克安静之用事爻，被冲克的用事爻不管旺相与否、无论是否旬空，都将肯定遭受伤害，因用事爻受动克本身就已经是衰败的定性。而冲克把冲与克两种力量都汇集在一块，其损害力度得到速度上的助推变得更加锐利，所以安静用事爻被动爻所冲克，其祸凶快而急速，静爻自身旺相与否在吉凶层面已经没有任何意义，只能在细节层面作为推敲受损程度之依据了，毕竟爻若得旺，即便受损也将有个限度。如未月戊辰日占官事罪之轻重，得“巽之涣”卦

原注：世临卯木，既墓于未月，又遇酉金鬼动冲克世爻，加临白虎谓之白虎刀，主罪之凶极。彼问：“凶在何时？”余曰：西金化午火，目下无碍，秋后防之，果至秋典刑。

这卦的动爻酉金对世爻卯木的伤害，就是属于冲克类型，严格讲应该是克中带冲，祸患更快刺得更深。世爻卯木被鬼动冲克，是被官非伤害的寓意，鬼动克世往往是大凶兆，因为鬼是祸患、疾病、官非、灾祸的通用代名词，而且卦中之鬼还连带着白虎，“白虎鬼”之凶恶意味在鬼爻中无出其右，据此可推断此人难逃铡刀见血极刑。有学者奇怪这里的动爻酉金是动变午火回头克，按理应该属于无用动爻才对，怎么可以作恶呢？这就是我们经常提到的“特殊日月组合”的特殊力量，改变了动变趋势的既往轨迹，卦中酉金受辰日合旺的同时，也受未月扶旺，于是高低共指向，特殊的日月组合，构成其不惧动爻相克的金刚本质，虽被变爻克制依旧可以发力致世爻死地。应期层面午未月份毕竟午火旺，所以目前暂不见灾，但等到秋天金旺之期，酉金适时借势，世爻卯木便在劫难逃。

而动静爻之冲的理论中，藏有一种有趣现象，就是动爻在一定条件下，能将卦中非世用身份的静爻冲动，使静爻由静态转为动态，再去作用用事爻显示吉凶趋势。这种理论如何正确实际应用正确呢？应用条件很严格，不是卦中所有动爻都会去冲动静爻，我们知道，动爻不会无故而动，动爻的运动方向是有着明确目标的，就是专门指向用神与世爻，以便预兆吉凶趋势，而动爻在吉凶判断层面，只能选择一个作用方向运动， 不能同时生克冲合多个目标爻，所以，动爻只有在不能参与直接作用世用的前提条件下，才会选择去冲动其他静爻，通过冲动静爻的方式曲线作用他爻，以达到间接性影响世用旺衰之目的，借力打力充分体现太极之妙。当然，即使符合冲动静爻的条件，静爻能否冲得起来，如上述所论，一要看受冲的静爻日月总体旺相与否；二要看受冲的静爻是否旬空；三要看冲静爻的动爻是否真正有力；唯独不再有纯冲与冲克之理论区别。如巳月丙戌日占通乡避贼乱，得“乾之大有”卦
