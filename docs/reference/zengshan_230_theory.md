## 一、特殊日月组合理论

**来源卦例：例3、例4、例18、例20、例22**

### 1.1 废爻型（凶）— 月破 + 日克

**定义**：某爻同时遭受月破（月令冲）和日克（日令克），构成"高低层次共指向衰败"的组合。

**效果**：
- 该爻如树木无根，永远成了朽木
- 即使有动爻来生旺，**生之不起**，动兆被废
- 动能轨迹终点法在此失效，日月组合取得主导权

**来源例证**：
- **例3**（大过之鼎）：世爻亥水遭巳月破、未日克，动爻酉金虽连动生世，"如树之无根，寒谷不回春"，卒于癸卯日
- **例22**（地天泰）：世爻辰土遭戌月冲（月破）、卯日克，即使子孙暗动制官鬼，世爻已废，"被杖"

**判定口诀**：月破日克废爻成，动生枉然改不了命

---

### 1.2 金刚型（抗克）— 月建 + 日生（或月生 + 日建）

**定义**：某爻同时得月建（月令临）和日令生旺（或月令生旺与日令临），构成"高低层次共指向旺相"的组合。

**效果**：
- 该爻如罩了金刚之身
- **逢动变回头克而不受克**，依然为有用动爻
- 受外克而不伤，照样发挥作用

**来源例证**：
- **例18**（屯之震）：父母申金值月建（申月）、得辰日合（日令合生），构成旺相式特殊日月组合，动化回头克午火之克，金刚不动，官父连动生旺用神子水，病痊愈

**判定口诀**：月建日生金刚身，化克依然有用动

---

### 1.3 判断层级排序

```
月破日克（废爻型）> 月克日克（双克型）> 月克日平 > 一般日月组合
月建日生（金刚型）> 月建日平 > 月生日建 > 一般日月组合
```

**核心原则**：
1. **废爻型**下，动爻能量被彻底覆盖，不再操控吉凶
2. **金刚型**下，回头克/化退/化绝等无用动爻定义全部失效，依然有用
3. 一般日月组合下，动变组合才是吉凶核心（动重月日，爻动势强）

---

### 1.4 特殊日月组合作用范围扩展

**例22的启示**：特殊日月组合不仅影响动爻，还可以**直接影响安静的世用爻**。

- 无需任何动爻生克，仅凭世爻被废爻型日月组合笼罩，即可判定自身最终受伤
- 因此，判卦顺序应优先检查世爻/用神是否落入废爻型组合

## 二、时效卦理论

**来源卦例：例4（铺垫）、例9（核心）**

### 2.1 月令时效卦（"月内时效卦"）

**定义**：所问之事的时效范围，**全程笼罩在当月之内**的占卦，月令的作用被大幅强化。

**特征**：
- 临月建之爻，如同手持皇帝圣旨，"如朕亲临，逢克不见凶"
- 该爻动变出回头克，在吉凶层面也不受伤
- 月令"说：这事归我管"，全局由月令主宰

**判定条件**：
1. 问占之事**必须在当月内完成/见分晓**（问当月考试、当月谒贵、当月手术等）
2. 用神或世爻恰好**临月建**

**来源例证**：
- **例9**（蛊之蒙）：酉月丙寅日占谒贵，世（用）官鬼酉金临月建，动化午火回头克，本应凶，但"事件全过程都在月内完成"，月建主宰，回头克不伤，断得见面。果于丙子日见贵

**判定口诀**：问事月内了，临建不怕克；月令统时效，金刚在月中

---

### 2.2 与废爻型的区别

| 比较维度 | 废爻型（凶） | 月令时效卦（吉） |
|--------|-----------|-------------|
| 日月组合 | 月破 + 日克 | 临月建（用神/世爻） |
| 时效前提 | 无特殊时效 | 问事必须在当月内完成 |
| 动变结果 | 动生不起 | 动克不伤 |
| 判断主导 | 日月组合主导 | 月令主导 |

## 三、真绊/假绊判定

**来源卦例：例1、例14、例23**

### 3.1 绊的基础

三绊类型：日绊、动绊、化绊（详见 `zhishidianzongjie.md` 三绊章节）

**总原则**：绊有真假之分；吉凶层面假绊不影响爻的正常作用，应期层面真假绊均须计入

### 3.2 真绊两类（卦象决定）

#### 类型一：卦象结构决定
- **外卦或内卦三爻全部动化绊**
- 或**六爻全部动化绊**

条件成立时，世爻或用神被真绊，事情悬而不决或事与愿违。

#### 类型二：问事性质决定
- **有明确时段目标的行人/出行之占**（如"今日能否出行"、"某日能否归来"）
- 此类占卦中出现三绊，即为真绊

**来源例证**：
- **例23**（明夷之小过）：申月丙子日占出行，世爻丑土动化午火，世与子孙卯木动化辰土，构成化绊，问的是当天能否出行，属明确时段行人占，三绊即真绊，断今日出行不成

### 3.3 假绊（最常见情况）

除上述两类真绊外，其余绊均属**假绊**：
- **吉凶层面**：假绊不影响爻的正常生克作用，当没绊看
- **应期层面**：假绊仍须计入，延迟到冲绊之日应期

**来源例证**：
- **例1**（乾之小畜）：午火化未土为化绊，但非真绊两类，吉凶层面忽略，午火正常生土；应期层面：到丑日冲绊，父母辰土病愈，"果于丑日起床"
- **例14**（履之中孚）：午火化未土化绊，假绊，吉凶层面午火照常克申金用神（凶兆不变）；应期延至丑日冲绊，"果卒于丑日"

### 3.4 速判流程

```
出现绊 → 检查是否为真绊两类？
  ↓是     → 真绊：吉凶层面不影响，应期冲绊日应
  ↓否     → 假绊：吉凶层面照正常生克断，应期亦参考冲绊日
```

## 四、六冲六合互变的就事论事吉凶直判法

**来源卦例：例21、例24（以例21为主要示范）**

### 4.1 基本原理

六冲变六合、六合变六冲，在特定条件下可**直接跳过五行分析，凭卦象变化定性吉凶**。

### 4.2 条件："就事论事，对应现状"

#### 六冲变六合 → 断吉（散而复聚）
**条件**：主卦六冲对应的现状，恰好是"此事之前已经失败/分离/受挫"。

**推断**：变卦六合 = 此事将从失败转为成功，从分离走向聚合。

**例证**（例21 坎之困）：
- 主卦六冲，现状"前试考过四等"（前次考试失败 = 六冲）
- 变卦六合，推断此次将转为成功
- 果"考一等，即于子科高发"

#### 六合变六冲 → 断凶（合状将破）
**条件**：主卦六合对应的现状，恰好是"此事正处于合作/蜜月/成功进行中"。

**推断**：变卦六冲 = 此合状态将遭破坏，由合走向分离或失败。

### 4.3 不符合条件时的处理

若不满足"就事论事对应现状"：
- **降回细节层面**分析，六冲六合只提供缓急程度信息
- 不能据此进行吉凶定性
- 仍须细看卦内五行生克动变

**判定口诀**：就事论事对现状，冲合互化可直判；不符条件降细节，生克动变才是纲

## 五、内重外轻原则 — 动爻内外作用区分

**来源卦例：例11（核心）、例15、例18、例28、例34**

### 5.1 外力与内力的定义

| 类型 | 定义 | 能量路线 |
|------|------|---------|
| **外力** | 其他动爻作用于目标爻（用神/世爻） | 动爻 → 目标爻 → （若目标爻再动，传至变爻） |
| **内力** | 目标爻自身发动 | 目标爻 → 变爻 → 回头作用目标爻 |

### 5.2 内重于外的核心推论

1. **目标爻若自变**，所有外力的能量将被传入变爻，再从变爻反作用目标爻
2. **变爻才是能量终点**，决定最终吉凶
3. 自变之变爻的生克方向 = 最终答案；外力只影响过程

**来源例证**：
- **例11**（困之恒）：外力午火（月建入动克用神酉金）属外力，用神酉金自动化退神为内力；内重于外，化退才是终局，大凶应在申年
- **例15**（遁之姤）：世爻午火虽得申月生（外力），但世自变出亥水子孙回头克（内力），内力主导，亥月病亡

### 5.3 应用推论

- **用神自变决定最终吉凶**，外力只影响过程和速度
- **世爻自变衰（回头克/化退/化绝/化忌）为凶**，即使有外力生助也不改变
- **逆推思路**：见目标爻有自变，先看变爻对目标爻的回头作用，再看外力

### 5.4 内外力的例外

**金刚型特殊日月组合**下（见第一章），即使内力表现为回头克，因爻本身刚强，回头克不显，依然按有用动爻处理（例18申金）。

## 六、卦意法补充 — 子鬼/父子/财鬼互化

**来源卦例：例17（子鬼互化）、例23（真绊与卦意结合）**

### 6.1 用忌互化三式

| 互化类型 | 触发条件 | 问事场景 | 判断结果 |
|--------|---------|---------|--------|
| **子鬼互化** | 子孙动化官鬼，或官鬼动化子孙 | 问孕育/子女/生产 | 凶（夭折、流产、不育） |
| **父子互化** | 父母动化子孙，或子孙动化父母 | 问孩子/文书 | 子孙有麻烦；文书废弃 |
| **财鬼互化** | 妻财动化官鬼，或官鬼动化妻财 | 问财/求财 | 凶（求财遇骗、因财招祸） |

**来源例证**：
- **例17**（剥之观）：子孙子水持世动化官鬼巳火，问生产（孕育），子鬼互化，"落草而亡"
- 原注："问子孙生养，得用神子孙与官鬼互化，均属夭折的凶讯"

### 6.2 属性与优先级

- 属于**卦意分析法**（非卦理）
- 优先级低于卦理（五行生克冲合）
- 当卦理与卦意矛盾时，优先从卦理定性；卦意作为辅助/补充

### 6.3 注意事项

- 官鬼为用神的事占，本身不存在鬼用互化（此时官鬼即用神）
- 此类互化，野鹤有言"万无一活"属夸张，但确属凶的概率极高的卦意征兆
- 原书对子鬼互化有"落草而亡"等实例验证

## 七、取用神细化原则

**来源卦例：例1-例23全部覆盖**

### 7.1 有殊取殊原则（位置优先级）

```
世爻/应爻位置 > 动爻位置 > 静爻位置
同等条件下：取旺相者（有旺取旺）
```

**说明**：处在"世应动"位置的用神，成为真用神的几率高达90%

### 7.2 客观取用原则

通过**与事实现状对证**来确认用神：
- 现状有受克/被伤 → 取受克之爻
- 现状有暗动/不安 → 取有异常信息之爻
- 排除与现状不符合的候选爻

**来源例证**：
- **例5**（萃之同人）：两个父母未土均动，取外卦父母未土为用，因其动化刑、化暗动、被月克，更符合父亲"已陷牢狱被执判"的现状

### 7.3 动变位置取用规则

**主卦动爻 > 变爻动爻**（取主舍变）

**来源例证**：
- **例11**（困之恒）：三爻酉金在变爻位置，五爻酉金在主卦位置动，两者同为用神备选，取主卦动爻酉金为用，舍弃变爻酉金

### 7.4 代占用神规则

| 代占类型 | 用神取法 | 说明 |
|---------|---------|------|
| 失控性代占 | 取六亲爻为用 | 如代占父母事，取父母爻 |
| 主控性代占 | 视主导权归属 | 决策权在代占者→取世爻；在被占者→取六亲爻 |
| 问他人 | 应爻优先为他人 | "世为自己，应作他人"（例1辰土应父）|

### 7.5 假应期卦识别

字面问应期但实质是问吉凶，须识别以下特征：
- **玄武持世** = 另有隐情私念，警惕假应期卦（例16）
- 事实上答案不确定却问"何时成"，属假应期卦，应按吉凶卦分析

## 八、应期细节补充

**来源卦例：例7、例12、例20、例29**

### 8.1 吉凶层面 vs 应期细节层面的区分

| 分析层面 | 月令作用 | 日令作用 | 动变作用 |
|---------|---------|---------|---------|
| **吉凶判断层面** | 只有旺相休囚 | 只有旺相/衰弱 | 生克冲合全用 |
| **应期细节层面** | 可见生旺墓绝 | 可见生旺墓绝 | 生克冲合全用 |

**核心原则（例20）**：月令对爻，**吉凶层面只有旺相休囚，应期细节层面方见生旺墓绝**。

### 8.2 日月代表事情现状

在细节层面，日月不只是旺衰工具，更是**事情现状的客观缩影**：
- **临日建之爻** = 当下状态（"卦中之爻代表事态目前的状况"）
- **月破之爻** = 已受损/尚未发生的现状
- **月令之爻** = 过去延续至今的长期现状

**来源例证**：
- **例4**（复之震）：日月克用神，作者判断"大罪难脱"，是将日月信息读作当下现状的体现

### 8.3 特殊动变应期规则

| 动变类型 | 应期规则 |
|---------|---------|
| 化进神 | 逢值（动爻临值）、逢合（变爻临合）、逢进（变爻临值）应 |
| 化退神 | 逢冲（动爻被冲）、逢退（变爻临值）应 |
| 化绊（动绊/化绊） | 冲绊之日应（冲开绊的那天） |
| 化空 | 填空（旬内逢值）、冲空（旬内逢冲）应 |

### 8.4 爻位排序法

- **初爻** = 一天/时段之始
- **上爻** = 时段之末
- 可用于推算时段内各阶段的变化进程（如例7推出当日酉时、次日初愈、戌日大愈、亥日痊愈）

## 九、特殊静卦经验

**来源卦例：例8（用神暗动）、例22（废爻月破）、补充自例27、例32、例33、例37**

### 9.1 近病用神旬空

**规则**：近病问卦，用神处于旬空状态，**无条件吉**（短期无碍），不受其他克伤条件限制。

**原理**：旬空代表"暂时不存在"，病未发作或暂时无碍，病根较浅，概率痊愈大。

**注意**：
- 只保证**近期短期**无碍，不代表最终趋势
- 久病用神旬空：多为"暂见好转"，非最终痊愈

### 9.2 近病六冲静卦

**规则**：静卦出现六冲，用神无动爻冲克，**断吉**（近病逢冲即愈）。

**前提**：
1. 占的是近病（非久病）
2. 用神不受动爻冲克（仅有六冲卦象，无动爻克用神）

**来源例证**：例1（乾为天六冲卦）原文："近病逢冲即愈，此卦乃六冲……"

### 9.3 代占静卦

**规则**：代占他人事的静卦，**不受"自占自事用忌持世"规则限制**。

- 自占自事 → 用忌持世规则全面适用
- 代占他人事 → 以六亲爻为用神，世爻只反映代占人自身状态，不适用用忌持世判断

### 9.4 问药效的特殊用神

**规则**：问药物/治疗效果，**取子孙爻为用神**（非世爻）。

- 子孙旺相 → 药效有效
- **父母持世** = 药效无灵（父母克子孙，克制了医药）

**来源例证**：例3（大过之鼎）中世持父母，子孙爻被克伤，用神亥水受废爻型日月组合，综合凶兆，与问病"父持世药无效"的卦意吻合。

## 十、综合应用框架

### 10.1 断卦优先级顺序

```
1. 检查世爻/用神是否落入废爻型特殊日月组合 → 若是，立定凶性
2. 检查世爻/用神是否临月建（时效卦前提） → 若是，月令主导
3. 检查是否有三合局 → 合局生克方向优先于单爻
4. 检查动爻内外力（目标爻是否自变）→ 内力（变爻）优先
5. 检查是否金刚型特殊日月组合（影响无用动爻判定）
6. 一般日月组合 + 动变组合综合判断
```

### 10.2 三大分析层面不串层

| 层面 | 适用规则 | 禁止混入 |
|------|---------|---------|
| 吉凶判断 | 旺相休囚、动变生克、用神与世爻 | 月墓（月令生旺墓绝）、六神独断 |
| 应期细节 | 月墓月绝、六神寓意、旬空进退 | 不可用细节推翻吉凶定性 |
| 现状对证 | 日月爻代表过去/现在信息 | 不可用现状覆盖未来趋势 |
