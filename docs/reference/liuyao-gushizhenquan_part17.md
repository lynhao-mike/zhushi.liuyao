__第三十二章: 三合局理论的应用__

三合局理论，对于古筮六爻理论体系来说，无论在吉凶判断层面，或是在应期细节分析层面，都具有相当重要的地位。三合局，指的就是十二地支中三个支聚合在一块所构成特殊性质的合局，为什么有三合局的存在呢？因为古人把天干地支，比喻为涵盖天地万物的信息代号，天地万物在不断循环的运动过程中，彼此不断产生冲与合，相持与对抗中，性质相近者自然形成一种群体聚合力，物以类聚组合成更强大的力量整体，以便能更好地保护自己，如宇宙在运动中产生各自不同的大小星系一样，各自星球聚合在一块形成力量体系，这是地支三合局形成的基本原理。三合局分别是：申子辰三合出子水局、巳酉丑三合出酉金局、寅午戌三合出午火局、亥卯未三合出卯木局。古论中所提到的三合局，只笼统说明合出什么五行性质的局，并没有明确合局的核心到底是哪个元素，如只提到申子辰三合水局，并没有提到这水局轴心在哪个爻上。我们知道宇宙万物都离不开轴心，如地球、太阳系、银河系等等，各自的联合体的循环转动都离不开轴心，轴心是运动不息的基础。没有轴心就不会产生运动，古语有言：“蛇无头不行”，因此三合局中必有作为合局轴心的爻，而能作为轴心者，必定是质量或力量的最强者，才有足够的定力让别人围着自己转，所以三合局中与合出五行属性相同的爻，三合的力量都聚集到此爻身上了，当属轴心不疑。如巳酉丑三合金局，形成组合的三个爻中，酉金也属金为合局轴心，因此完整的三合局应是巳酉丑三合出酉金局，各地支的力量都聚合到酉金身上了。由于三合局是根据自然界物以类聚的特性聚合而形成的，因此造就了三合局有别于其他个体的特殊性质，其特性就是其影响力与持久性均属非常  ─── 坚强而持久。问事而卦出三合，喜庆则久远而坚牢，忧疑亦锢结于心难以消释，祸事也会缠绵长久难灭。

不少现代易学者，他们否认三合局在六爻体系中的应用功能，认定三合局理论是古人编造出来忽悠后学者的，根本不存在。我则对这种观点持反对态度，无论从古例的剖析或实际操作中，无数的实例证明，六爻分析体系中不但存在三合局，而且在各个分析层面的应用都意义非凡。我举一个古例，寅月丙申日占升迁，得“艮之颐”卦
