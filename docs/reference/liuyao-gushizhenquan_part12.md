11、冲合预兆法

之前说过，世间万事万物都有其特性，均有其所好有其所恶，所好者会互相趋合照应，所恶者则相互背离冲突，卦爻是事物的缩影，因而每个爻都有其相冲者，也都有其相合者，从寓意上理解，相冲者代表着意愿的背离者，而相合者则代表着意愿的迎合者。当这种天然的属性与寓意原理融合到卦象的分析时，就产生一种别树一格的卦意分析方法 ———冲合预兆法。一种根据与用神相冲者或与用神相合者的变化趋势判断吉凶的卦意分析方法。冲合预兆法在现实的卦断应用中并不常见，即使搜索经典古籍也只浮出寥寥数卦，而且全部是另外的一种解释与说法。皆因此种卦意分析法的使用条件实在苛刻，除了与卦理存在冲突者不能使用自不必说，还必须具备用神定要藏伏；与用神相冲相合者在卦中独发动变；满足了这两大条件才可以正确使用该法。但既然有这样一种卦意分析方法客观存在，我们就必须及时加以归纳总结，好让我们的子孙后代有深入考究的空间。

__第二十五章: 卦意分析法之取用神__

这一章节阐述的其实也属于卦意分析法的分篇内容，只是用途的侧重点有所不同，上章主要论述根据卦意分析法用于吉凶判断，而这一章的研究重点则在于根据卦意分析法以明确用神。古筮技能，正确抓取用神是十分关键的一步，也是事关预测成败的关键所在，皆因用神是事主所关心事情的代表元素，卦中所有的动静信息点，其目标都是围绕着用神与世爻这两个核心来显示方向，一旦用神取错，其预测结果则多会不着边际。所以在用神章的论述中，对用神的选择与定位都有严格的程序约制。然而这还只是主观定位用神的取用方法，实际上能做到根据卦面信息客观取用，即并不根据问卦人口述的问题预先设定用神，而是在卦象形成之后，根据卦面信息的显示特征，从中洞悉问卦人的真实意向再确定真正的分析核心（用神），懂得客观取用，是预测准确率与稳定性出现飞跃性质变的重要一环。而卦意分析法中的取用神方法，正是介乎于主观取用与客观取用之间的一种取用神模式。

这里须要提点的是，卦象的分析途径有卦理与卦意两种。卦意分析法须于不存在卦理分析前提下方可以正确使用，因此在分析判断覆盖面上，需要运用到卦理判断分析的情况，几率总大于要使用卦意分析的情况，而在吉凶判断的应用上，即使卦理分析应用面较卦意分析广泛，但卦意分析法还是会大范围地存在，犹如一个人的左右手，虽然多数人因右手更为灵活而使用频率更多些，但大部分事情都需要左右手协调使用的，不懂得卦意分析法，就好象一个人缺了一只左手一样。但卦意分析法在取用神的应用上，其使用面则变得非常窄小，远没有其在吉凶判断时的作用广泛。因为卦象的常规显示模式，无非就是事态与心态两种，故取用的模式，也循着事态或心态的定性来定位，而卦意的用神，它性质上还是属于事态，没有发生变化，只是显示的模式更为直观与简单，在主观取用程序上无法解决但又不属于心态卦范畴的情况下，才可以考虑到这种取用神的思维方式，所以实际应用上出现这样的几率是不多见的，并不会大范围地出现在各类卦象中。之所以费那么大口舌说这番话，无非是想表达一个观点，大家在研读了这篇章的内容后，在实际应用中无需过分刻意地去重视它，了解有这种取用神的思维方式存在就可以了，避免以为取用神都须这样定位思考，因此形成错误的分析导向，反而会混淆自己已成系统的取用思维，影响正确的分析判断，应了买椟还珠这句成语。卦意分析法的取用神方法，大致归纳有三种，系统论述如下：

__1、用神替身法__

这种方法与卦意分析法中所使用的“代入确认法”、“藏爻确认法”十分相似，思维模式几乎是一样的，不同的是前者目的为了正确分析吉凶趋势，后者目的是为了正确定位用神。不少易者预测，总难避免遇见问事情，但所定位的用神与动爻、世爻却怎么样都拉不上钩，如用神藏伏在某爻里，或用神是由某动爻化出的变爻，就不知道应如何判断吉凶了，其实往往玄机就藏于此，伏飞与动变关系其实是一致的，都是根与支的关系，藏伏在某卦面爻里头的爻，古人称为“伏爻”，它上面的卦面爻就称为“飞爻”，有卦面才有藏伏，所以飞爻是伏爻的根源；同样道理，有动爻才有变爻的形成，所以动爻是变爻的根源。只要开拓一下思路，多想想为何用神偏要寄生于飞爻与动爻根源之内？若将用神所寄生的飞爻或动爻比作为用神，再与世爻动爻论彼此的生克关系，有时候你会发觉卦的吉凶趋势会顿时豁然开朗。这种“以根替支”的取用神方法就叫用神替身法。如亥月甲子日占仆人何日回，得“革之夬”卦

古论对于行人往归的总结之中有一条经验，它的实际操作应验率是相当高的，就是：“占问行人用空世空，必见归期”。我于寺庙所获的清代古筮占断散论古籍中，也列有此条，据之实践则灵验非常，后来在研读清初《增删卜易》过程中，也发现数例有此论述，始知道原来明末清初就有这个断法，估计总结得此经验者或年代更早。用神空，是指所占问的行人之爻正值旬空；世空，是指代表摇卦者的世爻正临空亡。古人认为“见空必填”，所以问行人用神空亡，必会填实而应有归期，这很容易理解。但为什么问行人世爻旬空，行人也必归呢？后来我仔细考究，发现这原来也有依据理由，占者问行人，多是自己关心的人，若所问之人回来了，他们必然会彼此相见，所以代表摇卦者的世爻临空，则寓意着摇卦者必将有与行人相见的一天，行人不回来的话彼此又怎可以相见呢？故问行人世爻空亡，一样也是行人将归的预兆。从此古例的原注得晓，野鹤断此人归来用的也是这个断法，他看见世爻兄弟亥水正值旬空，故有言：“世空者速至，此人来矣”，甚至应期分析也据此而发。若按正统的思路，问事首先要找用神，古代仆人也算是自己的财产，所以问仆人应取妻财爻为用神，卦面上没有妻财爻，正路应套取伏在三爻兄弟亥水之下的妻财午火为用神，用神伏空或伏破，都是用神衰弱的预兆，若真以午火为用，此仆人恐怕短期难到，但我们注意到飞爻亥水为何也空亡呢？问行人，用空世空的信息是行人必归的特有卦兆，由此可推论此卦显的真实意图，其实是为了说明真正的用神不是妻财午火，而是藏伏午火的飞爻兄弟亥水，原来藏爻妻财午火找了它的壳飞爻兄弟亥水做用神替身了，若把三爻兄弟亥水看作用神，分析思路将贯通无阻，用神空世爻空，都是人将速会的预兆，而且彼此都是亥水空，寓意仆人归来当天主仆将立即相聚。若不顺应思维，仍把藏伏午火作用神，用神伏破预兆仆人短期不归，世爻空亡却预兆仆人速归，判断依据矛盾则不知道应如何决策了。这是原用神藏伏，应定位飞神代替为用神的古例。

__2、拓扑用神法__

一卦里面所显出的信息繁多，有卦辞、卦宫、五行、六亲、世应、六神、星煞、十二宫、纳音、藏伏等等，凡卦断首先必须准确取用，这样才能更有效地抓住吉凶趋势的要害核心，这是毫无疑问的。具有千年历史的古筮金钱卦文化作为占卜术的大宗之法，古代流传下来的取用神公式，无论于事卦或者心态卦，基本上都是相对固定在六亲取用范围，这应该是取用神的最正统与最主流思维，坚持这种思维方式等于把握住了正确的研易路向。但卦象信息千变万化，其实有时候取用神不一定非要局限在六亲为取用条件，在某些特殊条件下其他如五行、六神、世应、星煞、甚至爻位所代表的爻，都可以取为用神，如问火灾水患，可直接取卦中的火爻或水爻为用神，这些是根据五行取用的模式；如占问出国游埠，可直接取临马星的爻作为用神，问小女孩若用神不见但却有桃花动的，可直接套取临桃花那个动爻为用神，这都是根据星煞取用；如守侯信笺或消息，可直接取朱雀所代表的爻为用神，或占测地震，可直接取临青龙或螣蛇有动的爻为用神，这都是根据六神取用的模式；如自问住院，可直接以世爻之墓代表医院，取为用神，这是根据十二宫取用的模式。如此类推。这类的取用神法类似于将用神的性质直接代入，显得更为直观与直接，故此法也称为拓扑用神法，俗称“类象法”。这些卦意取用法创于何时无从考究，而周易类象法的大成者则是“梅花术数”的创始人宋朝人士邵康节，当然此君据以时间类象为起卦方式，与摇钱古筮的成卦体系是格格不入，各自的分析重心也有天渊之别。自此后几乎每朝每代都有易者在此两法之中兜转，探索相互融通之路，或许拓扑用神法正是因此而被别具心思的易者总结发掘，并经过漫长历史经验的不断沉积而形成的。

    在实际的古筮应用过程中，时不时碰见拓扑用神法这类“原汁原味”的淳朴卦象在期间闪耀。例如卦象会非常直白地以所占事物的五行属性显示用神，将金属财经显示为金爻，水产交通显示为水爻，家私家具显示为木爻、火患用电显示为火爻、地域建筑显示为土爻等等，据以事物的五行特征定位用神，是拓扑用神法的典型显示模式，当然这种取用之法只是众多取用神模式中的一个点缀，并非主流的取用定律，只在按正统取用之法对不上现实状况、不符合推理逻辑之时，才可以发挥其作用，如戌月辛巳日占火患，得“坤之师”卦

    大家必须清晰的是，据六亲取用神，是卦理分析法的基本内容，也是最正统与最常用的取用神主流模式，而拓扑用神法源于卦意分析法的思维扩展，平常能应用操作的机会并不多，所以只能作为正统取用之外的有益的补充，其功能可作为六亲取用缺陷的互补与开拓思维的点缀，在某些特殊结构的问占模式上发挥正面作用，但要紧记千万不可矫枉过正，思维受其影响过大，导致颠覆正统以六亲为主流的取用神模式，把思路都集中到这些另类的取用方式里头，反成影响系统思维的累赘。其实，若一卦当中能以六亲取用就可以直接清晰地反映事态吉凶趋势，拓扑用神法则不能随意使用的，只有在正统六亲取用时卦象模糊、或论点矛盾、或无从入手之时，而卦意用神又突出在明显位置的条件下，适时采用拓扑用神法才可以做到准确断事。

随着对应爻功能应用的不断认识与深入，更让我渐渐发现了一个很有趣的实用卦种概念：双核卦象。双核卦象是由于应爻信息因某种指定被强化，而派生出来的卦象，这种卦象赋予了应爻更特殊的身份，将其作用上升到与用神不相上下的核心地位，当卦象动兆与之形成某种对比格局的时候，必须把应爻与用神信息组合应用，才能正确把握趋势定性。我把双核卦象细化为两类问占，一种叫特指事占，一种叫嫁接事占。特指事占指的是：某意念目标的成败，寄托在某指定的事物方位身上。例如问在上海的事业发展将如何？上海就是特指的地方，事业就是意念目标。类似的还有：指定某单位的应聘能否成功、指定某项目的运作是否盈利、指定某器材是否适合我专业的发挥、指定的某医院能否对我帮助等等，这些问念都属于特指事占的范围。那什么叫嫁接事占呢？原理上它和特指事占是一致的，不同的是指定的寄居对象，不是事物方位而是人，嫁接事占的定义是：某意念目标的成败，寄托在某指定的人物身上。例如问求助张某人是否能借到钱？张某人就是特指的人物，借钱就是意念目标。所谓嫁接，指的就是我之某事必须经过特定某人来衔接，类似的还有：与指定某人合作是否有财运、通过指定某人帮忙能否成事、问指定的某人能否完成我的需求、与指定某人的姻缘如何等等，这些问念都属于嫁接事占的范围。当了解特指事占与嫁接事占的定义后，我们自然发现这两种问念具有相同的特征，能影响事件的关键元素不单单是代表目标的用神，还有目标所寄身的、代表特定某人某物某方位的应爻，两个核心元素谁都不能被轻易忽略，所以我又把这两类卦称之为双核卦象。在双核笼罩下的卦象，特指的人事物使到应爻立即被上升到了吉凶定性层面。虽然双核问念下，代表意念目标的用神依然为重中之重，此乃双核中的核心，是影响事情最终成败的决定元素，所以很多易者本身并没有清晰的双核卦象概念，仅根据用神与世爻的互动关系，也能正确定性某些双核类事情的趋势。但双核卦象中，代表特指对象的另一个核心元素应爻，如果忽视它的存在，后果将很不堪，它就象一颗定时炸弹，会时不时窜出来搅局干扰正确定性，特别是一些额外提示的卦，应爻对定性的左右尤为显眼。例如问指定的某事物是否有碍财运，卦象显示另一事物有碍财运；问与指定的某人可否结婚，卦象显示与另外一人更有婚缘；问求助某人能否达至某目的，卦象显示求助别人才能达到目的；问到指定某单位考公务员能成否，卦象显示另一单位才有考公务员成功的希望；问到某地工作是否会不顺，卦象显示留在本地工作更不顺；问与某人合作是否有利事业，卦象显示与别人合作事业会更加理想等等。这些卦象由于代表意念目标的用神，本身已显示出直接的方向，因此对只注重用神旺衰以及用神与世爻互动的易者，产生分析定性的误导。问入职官生世预示能入职，但是不是所指定的单位能入职呢？问婚姻妻财旺预示能结婚，但是不是与指定的对象结婚呢？问跟随某人学习某项技能，父母克世预示不利，但不利是否就一定是此特指之人带来的呢？都不一定的，这时候就必须将用神与应爻综合分析，才能得出正确的预判。很多易者对于同一事项的不同决策，习惯一项接一项地去起卦预测，当掌握了双核卦象的分析技能，知道特指与目标的关系，懂得特指与同类的对比知识之后，选择其实不用逐一起卦那么麻烦，尽管心里有多套方案，只挑其中最上心的一个选择发问就可以了，心有所念卦有所显，心中既然有多套选择的想法，卦象自然也跟着显示各种选择的利弊，前提是你了解并掌握了特指及同类对比信息的技术。例如我易学弟子提供的一个实例，某人准备考公务员，他最心仪A单位及B单位，于是起卦自测能否考上A单位的公务员，得“鼎之小畜”卦

__第二十六章：月令的功能作用__

__月令的定义：__月令，即是我们在卦算中，日月组合里头所见的月支。月令作为日月组合的构成之一，其无论在吉凶判断层面，或在应期细节层面，影响力与信息功能都是非常重要的。古人对此则更为重视，明朝易卜经典《黄金策》有曰：“月将乃万卜之提纲。掌一月之权，司三旬之令”。月将就是指月建，又称为月令，意思是说，月令在一月上中下旬三十天内，当权得令，在此期间它的地位是至高无上的。我们知道一年有十二月令，按中国古代传统，将一年中分为十二节令与十二气令，统称为二十四节气，而我们卦算中所使用的月令，是按照其中的十二节令划分的，规划如下：正月为寅月，以立春节之日交令；立春节也是年与年之间的交令，是新一年的开始，我们每岁过年，过春节的习俗也源自于此。二月为卯月，以惊蛰节之日交令；三月为辰月，以清明节之日交令；四月为巳月，以立夏节之日交令；五月为午月，以芒种节之日交令；六月为未月，以小暑节之日交令；七月为申月，以立秋节之日交令；八月为酉月，以白露节之日交令；九月为戌月，以寒露节之日交令；十月为亥月，以立冬节之日交令；十一月为子月，以大雪节之日交令；十二月为丑月，以小寒节之日交令。这里大家有必要清晰的是，其实十二节令在一年中的日子是相对固定的，每年的误差只在一到两天之内，例如立春，每年的立春，不是在公历的2月4日，就是2月5日。由此可推算到，用于卦算的12月令时间范围，也如同公历一样，每年都是相对固定在某个时段范围内的，各月令公历时段范围大致如下表：

__月令范围的认识错误：__有好些初学者，误以为子丑寅卯月就是指农历的一、二、三、四月，于是一旦到了农历闰月，立刻昏头转向，搞不清楚应按哪个月令推算为准，闹出了诸如“逢农历闰六月，怎么跑出两个未月来的呢？”的笑话，其实子丑寅卯月等用的是干支历法，与农历的各月划分并不是一回事，彼此不能等同视之，这属于基础性的错误了。可能是受农历不是相对固定的观念影响，有一些初学者居然并不知晓十二月令时间范围的相对稳定性质，算卦总随身带着本万年历，子丑寅卯月相当于公历的哪些时日，都要到书上仔细翻查，其实没必要这样自己找自己麻烦，内行熟手的易学者，只需要记住子丑寅卯月各在公历的哪个时间范围段，跟别人分析时，便直接以公历告之，毕竟社会上绝大部分人，并不知晓子丑寅卯月各自代表公历的哪个时间段，内行人若不点明，求测人自然是你糊涂我跟着更加糊涂。

__月令对卦象吉凶趋势的影响力__：因一年有四季之分，各季节的具体划分为：寅卯辰月为木之春季，已午未月为火之夏季，申酉戌月为金之秋季，亥子丑月为水之冬季，一卦成后，卦中爻在月令里，适时得令者便得旺相，背道时令的便遭衰败，这就是月令能对卦中各爻产生旺衰影响的根本原理所在。通过本书总论篇对日月组合的单独介绍，我们已了解到月令能对卦中所有爻的旺衰产生影响，它是日月动变四大影响因素之一，月令作为日月组合的其中一个组成部分，虽然这种影响作用，在吉凶判断层面大多数情况下，并不能够与动变之爻的重要性相比拟，往往动兆才是事情吉凶趋势的最关键信息因素，如申月戊辰日占上书，得“中孚之损”卦
