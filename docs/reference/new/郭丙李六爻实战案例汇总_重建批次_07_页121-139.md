 搭桥法应用  
某人家中养的小狗病了，于是摇卦占问小狗还能活吗？  
 
思路：无论是六爻还是八字，或是其它术数，经历朝历代易学研究者的不断努力，挖崛出了
不少特殊组合，这类特殊组合在六爻理论中叫卦意法，它不按常规的卦理来分析卦象，今天
讲的这个搭桥趋变法，就是其中之一。在一卦中，在没有卦理可循的前提下，世用动变出寅
木，卦中又另有寅木动变出午火，于是视同于世用通过寅木为桥梁，动变出午火，直接以世
用动变午火看待，分析其吉凶。这就是卦意分析法中的搭桥趋变法，应用于吉凶判断层面的
常用卦意之法。为什么主卦是家人，家人可寓意亲人家人，现代社会有些人是把宠物当家人
看。这一卦取代表宠物的子孙为用神 ，应爻子孙变出的妻财爻未土并没有回头生克本位动
爻，四爻的妻财未土也没有生克应爻子孙，这卦直接按子孙搭桥变出代表疾病的官鬼，定性
这宠物结局并不是很好。官鬼带日月，卦变六冲，说明这病已有一段时间了，很可能是呼吸
方面的疾病，并且马上会发生突变。  
反馈：起卦次日，小狗就死了。
