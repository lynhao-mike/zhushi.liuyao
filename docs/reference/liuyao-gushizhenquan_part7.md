既然连问卦意念都不可替代，岂不是不能开口提醒他人应问何事？也不是不可以，但应正确引导对方自己主动产生问事的意念。对此野鹤老人提出解决的方法是，询问对方可否自占一卦问某事吉凶，但同时也如实告之：“这仅是我的想法，若你自己本身也有强烈问此事意念则无妨，若没有这个念头，就先别急着问事，先把我的提醒忘记掉，等你忽然自己想起要问这事了，再行占卜，这样问卦的意念就是自己的意念而不是别人的意念了，那摇出来的卦象，才是真正属于自己意念的自占卦象”。这种主动性的意念，是准确预测的最前提条件。我曾把这套把握主动性意念的方法，有效利用到个人的投资运作当中去。自从2006年底我以易学专家身份，与谭镜辉教授、邵伟华先生等四人，共同研究出席了中山大学周易应用研究室2007年经济投资领域八项预测报告会之后，我便从当时的股市趋势卦象中，预感到来年的投资机遇，开始怂恿身边的亲朋好友进入股市“抢钱”，他们自然非常期望我这根“名牌盲公竹”，能一直在股海波涛中为他们指点吉凶，最好能随时准确预测出股市的变化苗头，但我也知道股市投资中短期剧烈起伏的性质，将为实际的预测应用带来更大难度。而卦象显示的其中有一项特征，就是“不谋专事”，意思是说如果带着主观的计划性意念，去长期勘测预算某一件事情的趋势，这违反了客观条件预示趋势的卦象显示原则，结果将不可避免地越算越失准。于是我针对主动性意念的问题，想出一个应对的主意，我们几个投资股票的亲友形成一个交流圈，大家经常聚集在一块讨论，因为大家都是投资者，于是各种各样的投资意念都会产生，谁产生自己的投资意念就谁起卦，然后大家一起参考这个卦的趋势与启示。事实证明这个方法非常奏效，5、6月间与十月国庆节后股市出现的大幅调整，我们中大部分人均有幸躲过，冬至之后我们的股票收益率都十分可观。因为早在2006年底发布的股市预测报告中，就发现立冬后股市走势将有所趋缓，我于是规劝进来“抢钱”的亲友暂时退出股票市场，因为据我的投资经验，趋势决定投资，只要行情的长期上升轨迹受到压抑或改变，就应该立即从投资市场抽身，不管面对的是调整市道或是来回震荡的市场，对于一个外行人来说投资风险都将是巨大的。现在亲友们回首我当时的规劝，都不禁十分惊叹，当时谁也没有预料到，2007年那轮波澜壮阔的大牛市之后，股市竟然又开始了持续数年的漫漫熊途，将股海中的投资者打得七零八落。

__代占的隐患所在：__而请人代占，别人则更难清晰自身的问卦心态，别人代摇，卦象显示的难保不是别人的主观想法，除非代占之人心无旁念，非常专注于代摇之事，这样卦象信息或可以到位，但也毕竟因为隔了一重，求占人心态上的细微变化代占人是无法知晓的，因此卦象的细节程度也难免被打折扣；若问卦人心有隐晦，心中所念不如实告知断卦人，即使是自占之卦也难免或受误导，更何况是单凭对方只言片语而代占。如未月癸亥日占流年，得“艮为山”卦

另外注意主动性代占之中，又可细分为“失控性代占”与“主控性代占”两类。失控性代占指的是代占人对所占他人之事，完全没有任何主导控制力，只是关心关注而问占，自己无法主导或控制其变数。所以失控性代占，往往直接以被代占之人为定性核心，解卦思路相对较为简单，代表自身的世爻并不是分析重点，失控性代占的案例，也占了主动性代占中的主流，之上的代占举例，都属于失控性代占范畴。而主控性代占，指的是代占人对所占他人之事，其实质是受自己主导主控的。这类代占的卦象不太常见，而其影像会变得更为复杂，有时候它会以围绕代占用神的动态出现，那按常规的代占六亲思维定位就可以，代占父母事，就将发动或被动爻针对的父母爻，视为分析的用神。但有时候它会以针对世爻的动态出现，这就要注意了，虽然此事名为关心他人，实际上摇卦者自己，在事件中属于主控者的角色，卦中若出现了针对世爻（自己）的利弊指向，其实也与事件本身的趋势直接相关，当动态信息直接指向世爻的时候，我们对卦中六亲的定位也需及时调整，不再以代占的思维定位卦中六亲，而是应该以自占的思维定位卦中六亲。例如大人代问孩子上哪家幼儿园更合适，如果卦中出现了针对子孙爻的动变，事件的利弊指向自然聚焦在孩子身上，分析代表孩子的子孙爻就可达到目的；如果此时卦中出现了妻财爻发动克世，我们千万别以为代占卦与世爻无关，毕竟这类主控性代占事情中，被代占者（孩子）实质是没有自主权的，决策主导权反而在占卦者（世爻）身上，此时妻财爻针对世爻动克，这妻财乃是指大人自己的财，提醒大人应该避免选择太贵价的学校，以免因为决策错误遭遇破财损失。

__第二十章: 意念与卦象信息的关系__

__意念与卦象信息的关系：__这里也说说意念与卦象信息彼此间的关系与联系。念由心生，卦因念成，人们产生问事的念头，于是摇卦问卜，意念通过钱币工具构成卦象，所成的卦象则是昭示事情过去、现状、未来的全息载体，懂易之人通过解读卦象里面的各类信息，从中洞悉事情的前因后果，这就叫预测。从这个卦卜流程推理，意念与卦象之间构成一种因果关系，意念是起因卦象是与之对应的结果，卦象应是专门就求占者所问的事情而解释的象。事实上，一卦显一事，有大约八成的卦象，都是遵循这个方向显示信息的，怎样的念头引导出卦象显示怎样的事，当然若你问念不纯，意念影射到卦象上，导致卦象信息自然也会跟着模糊不清。但有另外两成左右的卦象却不是这样，即使你很清晰、很专一地询问这件事的趋势，卦象显示的却是其他的事情，这就是我们经常提到的“占此应彼”，对这样的卦象称之为变象。出现这类情形，除了与上一章提及的代占意念窜卦之外，往往与卦象本身的显像特征相关。总结起来，卦象信息显示共有六个明显特征。

__卦象显示特征之一：只答心念不答口述__。卦之显像，目的是为了回答一个人的疑惑，但注意了\!卦象回答的只是这个人心中关注的疑惑，而非这个人口中陈述的疑惑，它只会如实反映一个人心中在乎事情的趋势，而一概忽略掉此人口中陈述的问题，这就是摇钱成卦的显像特征之一。基础入门的学者，只懂得根据问卦者口述的问题切入分析，故之时有误读在所难免；而深入的经验易者，却往往懂得如何从卦象中，直接看穿问事人的心中所盼，断卦之精确度自然可以大幅提高。彼此切入重点之不同，正是易学业余与专业要求的分水岭。例如子月乙酉日，自占现任官职的吉凶，得卦“水天需”

__卦象显示特征之二：多问简显简问多现__。凡是通过手摇钱币出来的卦象，都存在着一个客观的信息显示特征，就是“__多问简显、简问多现”。__意思是说占卦问事，集中意志来去只有一个主题意念，则摇出来的卦象，卦象在清晰显示这件事情趋势的同时，也把相关的其他趋势信息预兆也显露出来，这就叫简问多现。若相反，你在摇卦问事之时，心思复杂不定，或同时问了这样又问那样事情，则摇出来的卦象，显示的信息反而只会简单反映几个问题中，重要性最大的一个问题的趋势，而且因问的事情太多，细节信息显得杂乱无章，根本无法抓主主次脉络，从而影响实际的预测效果，这就叫多问简显。故若有多件事情要问占，则宁愿一卦占完后，就另一事专门再占一卦为妥当。

__卦象显示的第六个特征：所答不谋专事__。现代不少人废寝忘食于钻研易道，原是为了想以卦卜为工具，专为某行某业某事项长久预测之用，如或专为测公司营运之用、或专为测期汇股市走势之用、或专为测彩票球赛之用、也有专用之于预测跑马竞技等等。类似这些专门性质的事测，间中测之则无可妨碍，但切不可抱有专门长浸于此的计划性意念，因卦象信息显示具有“不谋专事”的特点，一旦求测者在预测中尝到甜头，抱有专门就某事项长久钻研，计划性预测以图厚利的念头，随着时间的推移，卦的显像就会越来越变得模糊不清，准确预测的难度就越变越大，所以有此实际经历的研易者总会发觉这么一个现象，原本打算以易卜为专项工具，专门从事某事项的长久预测及跟踪研究，但不管多么用心去钻研去总结，其最终的预测准确率却只有一种趋势：高开低走。一开始的预测结果总是令人兴奋或给人希望，因为你具备易学的根基；但越往后测准确率就越变得飘忽不定，无论勤加总结与否，都无法改变这个总体趋势，乃是由于卦象信息开始越来越变得模糊，直至不再显示准确的信息所导致。这种奇怪的卦显现象，与研易者本身的水准并没有多大联系，而是卦象固有的信息显示特征在起作用，其关键原因，是这类计划性意念违反了卦象正确显像的初衷。操纵卦象显示的背后那个“隐身者”，感觉上它就象是自己人生中的一位导师，你困惑时它给你指出明路、你困境时它教你拔出泥潭、你迷失时它引导你直径方向．．．．．．但从更深入的层次看，卦象屡屡这样做的目的，好象并不单单为了帮助你变得顺利，更是希望你在反复经历后明白与反思过去，自己当时为什么会陷入被动困境或错误决策，从而通过总结与反思纠错过往的歧见思维，最终提升自身正确鉴别的能力，仿佛这才是卦象正确显像之最终目的：让你在总结中不断提升自身的智慧。

求占者的问卜意念，既然与卦象的信息显示存有那么些不对称性，那怎样从卦象信息中洞悉这些玄机另兆呢？主要方法有两个：一是要熟悉掌握卦象中过去与现状信息的解读方法。卦显于事，每个卦象都是所问占事情的信息载体，其中都或多或少藏有与问占事情相关的过去与现状信息，而这些信息藏在一卦的日月、卦象、卦爻、世应等等位置里头，把它们发掘出来，与客观现状一一对照，便可知晓卦兆是否属于就事论事的信息反映，还是另有别的预兆。二是要熟悉掌握“占此应彼”的特殊信息显示规律。卦象信息虽变幻无常，但它毕竟是一个预测未来趋势的工具，各式各样的预兆变化，都会转化为可分类解读、让别人能看得懂的有规律信息符号，所以但凡出现占此应彼的特殊信息，卦象本身都有特殊的信息组合给予及时的符号提示，而这些特殊信息组合都具有规律性、有形迹可寻的，不然我们也不会把周易预测术称为一项技能了，西方的星占学家不敢把星占预测称为科学，而称之为一种行为解释的艺术文化，我不希望国人也把自己的国粹周易预测文化，也理解为“行为解释文化”，怎么解释怎么有。这里也顺便讲一讲目前易学界十分时尚的一卦多断，无可否认，一卦中包含了很多与问占事情相关甚至某些无关信息，但易卜之道理，乃是冥冥客观中昭示趋势，所以“客观”二字贯通于整个易占的所有流程，哪个环节都必须强调客观性，当然卦象信息的显示也包括在内，能否一卦多断不应由断卦者自己说了算，而应由卦象信息本身说了算，卦象信息显示有的就说，卦象信息没有显露就不能胡说，这就是我平时经常强调的客观性一卦多断与主观性一卦多断的区别，但很多求学者却不是这样认为，不管三七二十一拿起卦就推：你母亲有多高、你父亲有多重、你兄弟姐妹有多少、你这里有多大那里有多长．．．．．．准确率与稳定性如何，这可能只有占者自己心知肚明，我要说的是，难道所有的问卦都会显示自己亲人或自身的全方位信息吗？于我的观点认为，显然不可能这样。

之所以出现上述这些意念的混杂信息，与我们起卦时的意念不纯或多或少都有些关系的，所以为了卦象信息更为清晰，预测更加准确，我们有必要注意起卦意念的把握。人的思想意念是很复杂的，摇卦的程序，主要捕捉的是意念信息，问事意念越单一，精神意志就越集中，灌输进卦象的信息就更清晰明了，但你若心中同时存在很多的思绪，传导到卦象意念信息的混杂程度就可想而知了。因此，一旦问事的意念产生，就应该立即问卦，这是清晰捕捉问事意念的最佳方式。而千万不可产生意念但即时却不问卜，过了一会又产生另外一些问题，杂七杂八的意念聚集在脑子里再去问，于是就产生信息显示的侧重性问题，你思维混乱不清晰，只好卦象代劳分清晰，然而经过卦象过滤后的信息显示，是否还是你想知道的信息，那就很难说了。更要命的是，有些人不单脑子里存的问题多，而且还希望象储物柜那样，把要问的问题都列明一二三四点存起来，然后才打包起卦，自己想着第一卦问这个问题，然后第二卦问那个问题，然后呢，第三卦再问其他问题，完全都计划好了。但这正好犯了易理的大忌，易理遵循的是客观的转变规律，不是任何主观性思维可以控制的，意念也是一样，你客观突发的意念，它会清晰传导，但你若想计划性主观去安排意念的发生，意念信息的传导可不由得你想怎样就怎样，你想第一卦问这事，第二卦问那事，信息传导有可能完全跟你掉转过来，有时候这种信息的错位后果是很严重的，会导致两件事情的预测都以失败告终，其罪魁祸首就是脑子里装的问题太多太杂了。如《增删卜易》内就有这方面的描述：“一念之诚，可格天地；二簋之用，可享鬼神。若有心怀两三事而占者，念既不专，应何能一？念多心乱，即是不诚，故曰遇事即占，不拘早晚，慎勿二念，二念不灵。曾有人先占求财，后占妻父之病者，于辰月乙丑日占求财，得“随之否”卦

相信不少人看了以上相关的论述，或产生这样一个疑问：我们通过钱币，将自身的问事意念转化为周易卦象里的各式符号，符号信息里装载着事情的过去、现状与未来趋势的各种信息轨迹，通过专门的理论方法解读，就可以预知事情的未来发展趋势，但这个过程的形成原理到底是什么？为什么卦卜里头会有问有“答”呢？而且这个“答”，还不单单只是问这就答这，从卦象显示特征看，它所回答的问题还懂得选择性，知道作答问题的哪儿轻哪儿重，懂得区分正心邪念之卜，也知道顾及社会发展的公平性，看上去只是寥寥无几的易经符号，却俨然如同一个具有独立思维的高等智慧生物。这背后到底是谁在操控？简直有些不可思议，当然也就别怪人们往神佛显灵那方面去想象。诸如此类问题，实话说，到现在依然无法彻底探索明白。依然没有探索明白？这算什么答案呢？然而，我觉得这就是答案。“易”即转变，包括宇宙之间的任何自然或非自然转变，我了解周易卦象，它是宇宙天象及所有物质运动过程与规律的符号浓缩；知道卦象它是事情过去、现状、将来的轨迹信息载体；知道易卜具有很强的预测实用性，也懂得不少正确解读周易信息符号的理论方法；我也隐约觉得任何有形始发于无形，物质的最终源头来自于想法，并且任何同位元素在不同时空中始终一脉相承，因此意念与未来相关物象，它们理应属于不同时空中的同位元素。但我却无法彻底弄明白，它为什么能具备这么多奇异功能的原理奥秘，我只能说，这可能是人类文明目前还没能力掌握与解释的众多宇宙神秘现象与规律中的一种，就象古人早在占卜中发现一个人的自主意念，居然可以转嫁到另外一个人的身上，但却一直到现代都无法找到合理的解释理由一样，或许我们目前的文明发展程度，还不足以掌握这些神秘现象与规律的背后真相，但它们却是实实在在地客观存在于我们的社会，我们也要正视如易经这类具有奇异预测功能的事实。在古代神化当道的社会，人们对神灵的存在深信不疑，于是将类似无法解释的天象神秘力量一概推之为“神”迹，连野鹤老人也深信背后操纵易卜的确实是神灵，以至于在《增》书中屡次把卦象信息指引尊称为“神示”，还专门著有鬼神章，时至崇尚科学破除迷信的今天，于是周易卦卜一同被贴上了封建迷信的标签，理由是科学无法证明未来是可以预知的，而且自古以来人们多把卦占与神迹挂钩，古人自己都认了，那还不证据确凿吗？我并非无神论者，神迹的有无我不敢妄言，我也亲身见识过某些寺庙确实存在特异的气场，因为在那求签是特别的灵验。但对于易卜而言，我很反感人们把这些能归纳成系统性理论的预测方法，一概扫进宗教神灵的框框，我敢说，卦卜推算精确与否跟神佛是否显灵，是完全不拉弦的两码子事。记得有位朋友把我向别人介绍：“这位先生算卦很准的，我觉得他简直就是神灵附体、或是开了天眼了”，对方立即投来尊敬的眼光：“大师您好．．．．．．”，然而我分明在他眼里看出“神棍”的意味，再听朋友的话语刚好触到我观念上的忌讳，于是断然否认：“这是未知的科学，与神灵完全无关”。我自小便对宇宙探索的相关知识兴趣浓厚，着迷于浏览世界各地UFO的报道摘文，相对于神迹之玄，我更倾向用科学解释神奇、更坚信外星文明存在的真实性，在无际的宇宙中，地球只不过是沧海一粟，若只有地球能产生文明，岂不浪费了这偌大的宇宙空间？记得当时最令我感兴趣的是，文明外星生物所处的星球，估计离我们地球起码有数百光年之远，那他们是怎么来的呢？即使有达到光速的机器，生物也不可能有那么长的寿命啊？当然站在地球人的科学思维方式，是打破头也想不出原由，但无法解释，只说明我们还没达到了解与掌握这些神秘力量的知识水平，只说明我们人类还有许多宇宙规律与知识还在探索之中，并不能因此而主观否定神秘现象客观存在的事实，不然便是本末倒置了。既然诸如外国催眠治疗术、中国古筮预测术这些技能术背后发掘出来的神秘力量，它们对我们人类有正面的利用价值，我们就应该正确使用它，至于它们为什么能具有这些特殊力量，或许它们是来自另一层宇宙空间的智慧体、或许它们是我们自身潜意识或潜能智慧的一种表达，我们目前还无法探索明白的，可以随着人类文明的不断进化来慢慢解决，而不应该把它们统统贴上“封建迷信”的标签，然后一棍子打死，这些解读神秘力量的知识理论一旦失传或沉沦，受损失的终将是我们人类自己。那种科学水平解释不了就应该是弄虚作假或封建迷信、应予取缔的观点，我个人认为并不是真正客观的科学研究态度。同时外界对待占卜态度的也相当重要，毕竟目前大众观念上，对古筮预测仍存在严重的认识偏见，不信者痛斥之，信者对占卜的需求，也和对待求神拜佛没有两样，视易学者如巫师的也大有人在，他们居然相信我们是神仙，不是神仙起码也是半仙，人可以犯错但神仙是不会错的，神仙说的话不容质疑也无需实证给谁看，信则有不信就算，于是易学界乐于去装神弄鬼，信易的人们也乐于见到他们装神弄鬼。但随着对易学认识的不断深入，当玄学的谜团逐渐被揭开变成科学，古筮学从神台回归至人间技能的时候，人们对待古筮预测者，是否也应该更加理性呢？

__第二十一章: 动卦中的卦理性吉凶卦局通论__

前章通过对“日月组合”、“动变组合”、 世用相互作用等相关知识的领会，相信我们对卦的吉凶昭示原理已有个较清晰的大概层次观念，为了更利于实际的应用，将上述理论知识进行综合整理，把它归纳成各种动卦吉凶卦局，这些卦局都属于五行生克的卦理性质，所以称为卦理性吉凶卦局。为什么这样说，是因为卦的吉凶判断，由两大部分组成，除了通过卦理分析得出吉凶结论，还有一种方法是通过卦意分析得出吉凶结论，而无须遵循卦理来分析的，卦意分析具体的理论与应用原则，将在后章中专门论述。动卦中的卦理性吉凶卦局系统论述如下：
