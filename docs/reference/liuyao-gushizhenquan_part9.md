这里要注意的是，即使变爻定义为“化散”，也存在真化散与假化散之区别，原因正如我在前章介绍那样，其一是“动能轨迹终点”，若动爻所化出的变爻不再回头作用动爻，则动能的运动方向是由动爻趋向变爻，变爻是动能运动的终点，变爻若定义为日破，则代表整个动爻的最后结局将是趋向衰败无用；但若动爻所化出的变爻可以回头作用动爻，该动能的运动方向开始由动爻趋向变爻，但变爻却已不是能量运动的终点站，只是中转站，动能在变爻身上转了一圈，既而又回复作用到动爻本身，动爻才是动能轨迹的终点站，所以即使在这过程中变爻遭日散，也不能因此定性整个动爻的最后结局，因为该动能还没运行到终点。动能轨迹终点的原理运用到动变中，就是在动变产生回头作用的前提下，即使变爻是日散，在吉凶层面一样可以发生作用，不能一概按动化破散衰败无用论，如卯月壬寅日占寻地，得“革之既济”卦

__第二十三章: 静卦的判断原则__

__静卦的定义：__静卦，泛指的就是没有动爻的卦象，不动就没有变化，因此静卦只有主卦而不存在变卦。那么卦中若没有明动之爻，但却有暗动之爻，算静卦还是算动卦？从暗动章，我们已了解到暗动之爻的性质特征，暗动之爻严格来说并不算是完整的动爻，只能在适当条件下才能发生作用，因此它与明动之爻是有区别的，它是没有变爻的动爻，没有变爻也就不存在变卦，而我们所论述的静卦含概范围，只要不存在变卦的卦象，即没有明动之爻的卦象，就归纳入静卦范畴，从这点上看，只有暗动没有明动的卦象，自然属于静卦的范围内了。或许求学者会不解，为什么非要弄得那么清晰呢？这个知识点掌握与否似乎无关痛痒啊？因为接下去学习就知道，静卦的判断原则，与动卦的判断原则，彼此思路上其实大相径庭，在断卦前首先定性卦象属动或属静，对下一步正确套用卦算原则准确预测是至关重要的。

__古传统的静卦判断原则：__在古代传统的六爻理念上，静旺之爻与动爻，均属于能产生效用的爻。在动卦中，卦机在于动，动爻往往是一件事情吉凶的预兆核心，而因在静卦中并不存在动爻，于是退而求次，静卦中旺相的静爻，功能就如同动卦中的动爻，可以生克作用其余静爻，若多爻同旺，彼此还可以连环作用，就如动卦中的多爻同动而需连动作用一般，这就是静卦的传统分析思路，如子月辛亥日占远行求财，得“山天大畜”卦

__第二步：__自占自事摇得静卦，先观是否有用忌神持世为吉凶判断的首要原则，这一定要记住。当静卦不是用神持世或忌神持世的时候，其次的分析重点就直接落到用神与世爻是否存在生克关系上。因为世爻是最终目标爻不会去作用旁爻，而用神却不一定就是最终目标爻，可以世爻为作用目标，因此彼此的生克关系其实是单向性的，只观察用神是否对世爻存在直接生克的格局，若存在，在吉凶层面则不用再费劲分析卦中其他静爻的旺衰与生克关系，直接以彼此关系，按下列分析原则定性吉凶：
