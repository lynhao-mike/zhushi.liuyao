__第三十四章: 化进化退理论__

相信有古筮学习基础的求学者都很熟悉《黄金策》上的一句话：“动变比和，当明进退”。比和者，即五行之同类也，在六爻中就是指五行属性相同的爻，这话的意思就是说，爻动而变化出与其五行属性相同的爻，则有化进与化退的区别。动为始变为终，动而变进化退，直接关系到此爻的旺衰与否，或有用无用。何为化进何为化退？是根据十二地支的排列顺序而定的：化出同属性的爻位于己之后称为化进，化出同属性的爻位于己之前则为化退。如寅木发动化出卯木，大家的五行属性都是属木，寅木排在顺数的第三位，而卯木排在顺序的第四位，卯木位列寅木之后，这种情形全称为化进神；又如酉金发动变出申金，大家的五行属性都是属金，酉金处于顺序的第十位而申金排第九位，申金位于酉金之前，对于酉金而言，申金等于是倒退回去了，遇此情况全称为化退神。

__化进化退在吉凶层面的应用：__在吉凶层面，十二地支各自的化进化退共有八种模式：

化进神：寅木动化出卯木，申金动化出酉金，丑土动化出辰土，未土动化出戌土。

化退神：卯木动化出寅木，酉金动化出申金，辰土动化出丑土，戌土动化出未土。

看到这，有学者或者会非常疑惑，不对吧？化进化退怎么都只有四对？记得我把这个理论，教给一个有些根基易学者的时候，他最初的反应就是拍案而起：“你这是在误人子弟！即使是你备加推崇的《增》书，论进退神也列明有七对，更何况四土爻互为进退，实际应该不止七对才是。再说如亥子、丑戌之间的互化该论化进或论化退；还有辰戌、丑未之间的互化该论相冲或论进退；这都是牵涉到六爻进退理论的精华部分，我还曾经有幸拜读过专门论述这方面的文章，怎么又不见你对此有只言片语？你所谓的精简，怎么把精华都给删去了，又怎可以把最基本的结构框架给删除掉了呢？”，面对一串串火辣的连番质问，我连忙赔笑：“你先别上火，你说的那些问题从纯粹理论上看似乎都应该重视，这样吧，你先把亥子、丑戌、辰戌、丑未等同属性互化的卦例找出来，还有巳午、辰未同属性互化的卦例，只要你找得出哪怕一个例子，也不用分析了，我承认绝对是我的错，你看行不？”，他见我笑眯眯的样子反而有些楞住了。结果到了第二天，此人灰溜溜地回来，满脸颓丧。因为亥子、巳午、丑戌、辰未、辰戌、丑未这六对爻，事实上两爻之间根本不存在动而互变的可能，连变都变不出来又拿什么论进退，还更深一步去兼论其他？这些建立在现实根本不存在的、海市蜃楼般的幻影理论，除了加重学易者的思维负担之外一无是处，不删掉它还能删谁。当然，在吉凶层面我们可以这样认定，但这些幻影理论也不能尽废，到了细节层面，进与退的对比思维将变得更加立体化，它们还是大有其功用的。

言归正传，作为古筮六爻的一个常用理论，化进化退的入门理论知识是相对比较好掌握的。在吉凶判断层面，作为目标爻，自身化进神或化退神，则是趋向旺相或衰败的征兆，世爻或用神动而自化出进神，则是事兴如意的吉兆，相反若动而自变出退神，乃属于事败萧条的预示。故《增》书有言：“进神者，由此而前进也，如春木之荣；退神者，由此而渐退也，瞬息凋残”。如亥月丙戌日占道考，得“丰之革”卦
