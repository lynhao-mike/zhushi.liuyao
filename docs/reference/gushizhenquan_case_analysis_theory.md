# 《1-2古筮真诠》案例分析理论提取

本文档从工作区根目录的 `1-2古筮真诠_part1.md` 至 `1-2古筮真诠_part7.md` 提取可直接用于六爻案例分析的理论，并整理为断卦报告可复用的分析框架。

## 一、总分析原则

### 1. 分层判断：吉凶、应期、细节不可混用

案例分析应按三个层次展开：

1. **吉凶定性**：先判断事情成败、利弊、趋势方向。
2. **应期推断**：在吉凶定性已经成立的前提下，推断何时应事。
3. **细节取象**：解释人物、状态、过程、原因、环境、心理与外部条件。

同一个爻在不同层面含义可能不同。吉凶层面不成立的用法，不代表细节层面完全无意义；细节层面的象，也不能越级推翻吉凶定性。

### 2. 先定用神，再定世用关系

所有案例分析必须先明确：

- 本占问的核心对象是什么；
- 对应六亲是什么；
- 用神是主卦之爻、变爻、伏神，还是卦意取象；
- 世爻代表问卦人或占事主体；
- 用神与世爻之间是否有生、克、冲、合、墓、空、动变关系。

若用神取错，后续旺衰、动变、应期都会偏离。

### 3. 动必有因，动爻优先

动爻不是普通信息点，而是趋势发动点。分析案例时应优先检查：

- 动爻是否直接作用世爻或用神；
- 动爻是否构成元神、忌神、仇神；
- 动爻是否化回头生、回头克、化进、化退、化空、化破、化绝；
- 多动时是否构成三合局、连环相生、动绊、化绊、间爻阻隔。

但动爻也要区分有用、无用、假象与细节象，不能见动即断。

## 二、静卦案例分析框架

静卦没有动爻提供趋势，应采用“三步曲”。

### 1. 第一步：用神或忌神持世

先看持世之爻是否为用神或忌神：

- 用神持世，多代表主体与事情同气，若旺相多利；
- 忌神持世，多代表主体被不利因素占据，静卦中常可先定凶；
- 若问病、忧患、官灾等特殊占类，需结合具体用忌转换。

### 2. 第二步：用神生克世爻

若不是用忌持世，再看用神与世爻是否形成生克：

- **用神生世**：通常为事情有益于我；
- **世爻生用神**：我去成就事情，需看世爻是否有力；
- **用神克世**：不可机械判凶，须看占事类型；
- **世爻克用神**：多表示我能制其物，也可能表示事情受我压制，须分占类。

### 3. 用神克世的特殊分辨

用神克世要结合占事长短和占类：

| 占事类型 | 用神克世含义 |
|---|---|
| 目下求财 | 财克世可为得财，只要世爻能承受 |
| 终身财福 | 财克世反为受财所累、因财成苦 |
| 行人归期 | 用神克世可作行人速回 |
| 长期事业、长期关系 | 多偏压力、牵累、消耗 |

因此案例报告中不能只写“用神克世故凶”，必须说明占类与期限。

### 4. 第三步：用旺世兴

若静卦不成持世格局，也没有明确世用生克，则分别审查用神与世爻旺衰：

- 用神旺相，说明事情本体有力；
- 世爻得日月之一生扶，说明主体能承载此事；
- 用旺且世兴，多可定吉；
- 用旺而世衰，多为事情有而我不能得；
- 世旺而用衰，多为我有心而事体不足。

## 三、动卦案例分析框架

### 1. 卦理优先：直接作用先于取象

若动爻与世爻、用神发生明确生克冲合，应先按卦理定性。卦意取象可作辅助，但不得推翻明确的卦理作用。

例如：用神发动克世，在求财短占中若世爻旺相，可断得财；即使卦意上出现背向，也应先从卦理定性。

### 2. 间爻发动：阻隔与过程障碍

间爻位于世应之间，若发动但不生扶世用，常主中间阻隔：

- 买卖：中介、佣人、手续、第三方阻挠；
- 求职：竞争者、流程障碍、资料问题；
- 租赁/交易：竞价者、经办人、条件变化；
- 感情：中间人、现实压力、沟通断层。

若间爻自身化回头克、化破、化绝，则阻隔虽有但未必能成最终败因。

### 3. 连环相生与桥梁动爻

当用神不能直接生世，但通过其他动爻连续相生到世爻，可形成“桥梁”结构：

- 用神 → 动爻 → 世爻，为间接成事；
- 中间动爻若月破、旬空，在吉凶层面未必废，需看是否是假破假空；
- 若桥梁动爻最终化废，可能中途断链。

案例报告应把动力链逐段写清楚。

## 四、卦意分析法

### 1. 卦意分析的定位

卦意分析不是灵感式断法，而是与五行生克并列的辅助系统。它适用于：

- 卦理难以解释但卦象结构明显；
- 世、应、动、变呈现方向性、背向性、牵连性；
- 伏神、藏爻、合冲、变卦名义具有明确现实投射；
- 需要解释细节、原因、环境、人物状态。

### 2. 世用背向

若用神与世爻在动变方向上彼此背离，常主事情与我不相就、方向不一致。但若同时存在明确卦理作用，应先取卦理，卦意只作细节。

### 3. 藏爻确认法

当用神藏伏或目标不显时，可通过伏神与世、应、动变结构确认事情趋势：

- 藏伏用神与世爻相合、相生，可视为暗中有机会；
- 藏伏用神受飞神压制，需看飞神是否空破；
- 藏伏用神不宜随意越过飞神与其他爻直接作用。

### 4. 冲合预兆法

冲合可作为事态趋向信号：

- 冲：急变、突发、分离、短促、突破；
- 合：牵连、延缓、缠住、合作、羁绊；
- 冲合不直接等于吉凶，必须结合世用旺衰与占事目标。

## 五、六冲与六合

### 1. 六冲理论

六冲不是凶煞标签，其核心是“急、快、变、短、不久”。

案例应用：

| 场景 | 六冲倾向 |
|---|---|
| 短线交易 | 可能利于快进快出 |
| 长期合作 | 多主不稳、不久 |
| 急事求成 | 若世用有力，可主快速成事 |
| 婚姻、长期事业 | 常提示难以长久维系 |

主卦六冲偏当前状态急变；变卦六冲偏未来趋势不久。

### 2. 六合理论

六合核心是“合、缠、慢、久、牵制”。

案例应用：

| 场景 | 六合倾向 |
|---|---|
| 长期合作 | 若世用有利，可主稳定 |
| 短线操作 | 可能进出不畅 |
| 感情 | 可主牵连、眷恋，也可能拖延纠缠 |
| 事情办理 | 多有延缓、等待、难迅速解决 |

### 3. 动绊与化绊

两动相合为动绊，动爻与变爻相合为化绊。绊的本质是被牵制，短期难以自由发挥。

但“绊”也要分真假：

- 吉凶层面若核心动爻被真绊，作用力受限；
- 若是假绊，不可废其生克；
- 细节层面可解释为被人事、手续、情绪、环境拖住。

## 六、三合局

### 1. 三合局基本原则

三合局包括：

- 申子辰合水局；
- 巳酉丑合金局；
- 寅午戌合火局；
- 亥卯未合木局。

三合局形成后，应作为整体力量分析，不宜再把单爻拆散逐个论。

### 2. 三合局优先级

动爻合成三合局时：

- 优先看合局整体对世爻或用神的作用；
- 单爻的回头克、化破、化绝等，在合局中常不再按单爻废用处理；
- 世爻自身趋势例外，仍须特别审查。

### 3. 三合局的案例用途

三合局适合解释：

- 群体力量；
- 多方竞争；
- 团队合力；
- 环境趋势；
- 被某种局面困住；
- 资源聚合后集中作用于目标。

若三合局克用神或克世，多为强大外力压制；若生用神或生世，多为群力扶助。

## 七、日令与月令

### 1. 月令：过去、背景、时效范围

月令不仅判断旺衰，也反映事态背景与过去延续到当下的信息。

案例应用：

- 月建：当前阶段最强背景力量；
- 月破：过去或当下存在损伤、破败、受冲；
- 月合：背景中有牵连、合作或束缚；
- 月墓：背景中有隐藏、封闭、受困；
- 月令时效卦：若占事明确在本月内完成，月令可决定短期有效性。

### 2. 日令：现状、即时状态、当前力量

日令多代表当下现状与即时作用：

- 日建：当前强力扶助；
- 日破：当前受损、破败、冲散；
- 日合：当前被合住、牵连、得助或受绊；
- 日墓：当前处于隐藏、拘束、迷茫、收束状态。

### 3. 特殊日月组合

有些爻在日月共同作用下，会改变普通动爻主导趋势：

- 日建月扶、日建月生、日合月生等，可成特殊强旺；
- 月破加日克等，可能成废爻；
- 月冲日冲未必必衰，要结合是否得动爻生扶；
- 日月组合要优先于孤立旺衰判断。

## 八、旬空、月破、日破与真假

### 1. 旬空

旬空在不同层面含义不同：

- 吉凶层面：可能表示不实、暂空、难落实；
- 应期层面：逢填实、冲实、出空可应；
- 细节层面：表示空缺、虚象、暂未出现、心理落空。

动空、化空未必必废。若在吉凶层面属于假空，则仍可照常生克。

### 2. 月破、日破

月破、日破表示受冲破损，但也要分层：

- 静爻被破，常主衰败或不稳；
- 动爻被月破，在某些吉凶层面不必直接视作无用；
- 化破若只是中转站，可能是假化破；
- 伏神之飞神被破，短期可成冲飞露伏。

### 3. 假象原则

案例分析中要识别：

- 假空；
- 假破；
- 假墓；
- 假绊；
- 假化散。

判断是否为假象，要看该爻是趋势终点，还是动力链中的中转站；是吉凶核心，还是应期细节信号。

## 九、墓理论

### 1. 墓的基本含义

墓又称库，含义包括：

- 客观层面：隐藏、收藏、穴居、拘禁、约束、包裹、封闭；
- 主观层面：迷茫、隐瞒、关注、沉迷、倾心、疑惑。

金墓丑，木墓未，水墓辰，火墓戌。土爻在吉凶层面通常不另论入墓。

### 2. 墓在吉凶层面

用神入墓、世爻入墓，要看：

- 墓是否为日墓、动墓、化墓；
- 墓爻是否空、破、化衰；
- 是否处于三合局内；
- 该墓是困住用神，还是收藏忌神。

墓空、墓破、化衰常可视为假墓，短期不必按真困判断。

### 3. 墓在细节层面

即使吉凶层面是假墓，细节层面仍可解释现实状态：

- 病人昏沉；
- 失物藏于某处；
- 人被管束；
- 事情被压住；
- 心理迷茫；
- 对某人某事投入关注。

## 十、伏神理论

### 1. 伏神不可乱作用

伏神伏于飞神之下，其核心关系是伏神与本位飞神。案例分析不可随意让伏神越过飞神，与其他动静爻直接论生克冲合。

### 2. 用神藏伏

用神藏伏时，先看：

- 伏神自身旺衰；
- 飞神是否压制伏神；
- 飞神是否空、破、受伤；
- 世爻是否受伤；
- 是否构成“用藏世伤”。

若用神藏伏且世爻受伤，可直接偏向事败。

### 3. 冲飞露伏

飞神旬空、月破、日破，好比覆盖伏神的盖子被掀开，短期事占中常主伏神易出。

适用场景：

- 短期病患；
- 失物寻找；
- 文书往来；
- 短期财运；
- 隐藏之事短期显露。

不宜无限扩大到所有长期占事。

## 十一、应期理论

### 1. 应期必须服从吉凶定性

先有吉凶，再推何时应。不可为了凑时间而推翻定性。

### 2. 常用应期规则

| 信息 | 应期取法 |
|---|---|
| 旬空 | 填实、冲实、出空 |
| 入墓 | 冲墓、冲爻、出墓 |
| 六冲 | 快期、冲期、短期 |
| 六合 | 合期、冲开合期、较迟 |
| 伏吟 | 逢值、逢冲，或按目标爻状态取期 |
| 多现多发 | 取墓库收藏之期 |
| 三合局 | 取合局核心、成局、冲局、墓库之期 |
| 动爻化进 | 进神当旺或进成之期 |
| 动爻化退 | 退神显退或衰退之期 |

### 3. 主应期与细节应期

- 主应期：先按事件性质确定时间单位，如日、月、年；
- 细节应期：在主应期基础上，再按逢值、逢冲、同属性、对冲展开。

长期占不能随意按日断，短期占也不宜拉到多年。

### 1. 取用神

### 3. 吉凶定性

按顺序判断：

1. 静卦还是动卦；
2. 是否用忌持世；
3. 是否世用直接生克；
4. 是否动爻作用世用；
5. 是否间爻阻隔；
6. 是否三合局优先；
7. 是否伏神、墓、空、破构成特殊格局；
8. 最终定吉、凶、迟成、短吉长凶、先难后成等。

### 4. 卦意细节

解释：

- 谁是阻碍；
- 事情卡在哪里；
- 对方状态；
- 环境条件；
- 主体心理；
- 可操作建议。

### 5. 应期

必须先说明以什么为时间单位，再列应期依据。

### 6. 结论

结论要明确，不要骑墙：

- 成或不成；
- 吉或凶；
- 快或慢；
- 长久或短暂；
- 应该进取、等待、止损、转向还是防范。

## 十三、可直接落地的规则清单

1. 静卦按“用忌持世 → 世用生克 → 用旺世兴”三步分析。
2. 用神克世须按占类判断，不可一律断凶。
3. 动爻直接作用世用时，优先按卦理定性。
4. 间爻发动且不益世用，多作阻隔。
5. 间爻化废，阻隔可弱化为过程障碍。
6. 三合局形成后，优先看合局整体作用。
7. 六冲主急变、短促、不久，不直接等于凶。
8. 六合主牵连、延缓、稳定或羁绊，不直接等于吉。
9. 动绊、化绊要分真假，假绊不废生克。
10. 月令可看背景与过去，日令可看现状与当下。
11. 旬空、月破、日破要分吉凶层、应期层、细节层。
12. 墓在吉凶层面可论困，在细节层面可论藏、管束、迷茫、投入。
13. 伏神主要与飞神发生本位关系，不可乱与他爻直论。
14. 飞神空破在短期占中可作冲飞露伏。
15. 用神藏伏且世爻受伤，为“用藏世伤”，多主事败。
16. 应期必须服从吉凶定性。
17. 主应期先定时间单位，细节应期再细化。
18. 报告必须把“因为 → 所以”的卦理链写清楚。

## 十五、全文核验后的补充规则

本节依据根目录 `1-2古筮真诠_part1.md` 至 `1-2古筮真诠_part7.md` 的全文扫描，以及 `docs/reference/` 下既有书籍与理论文件的覆盖扫描补充。核验报告见 `docs/reference/theory_audit_report.md`。

### 1. 暗动与日冲：冲起、冲实、冲破要分占事长短

日令冲静爻不能一概断为日破，应先区分：

- **旺相静爻逢日冲**：短事中多可视为暗动，能即时参与吉凶；
- **旬空静爻逢日冲**：可成冲空即实，短期可用；
- **衰弱静爻逢日冲**：多偏日破、冲散；
- **长期占事中逢日冲**：即使短时有动象，也难维持长久，常转为破散信息。

案例分析时应写明“此处为何按暗动、冲实或日破处理”，不得只写“日冲故动/故破”。

### 2. 静卦应期：动而逢值逢合，静而逢值逢冲

静卦没有动爻，应期不宜照搬动爻规则。基本框架为：

- 目标爻旺相而静，常取逢值、逢冲为应；
- 若静爻旬空，取填实、冲实、出空；
- 若静爻被合住，取冲开合处；
- 若静爻入墓，取冲墓、冲爻、出墓；
- 若吉凶层面已判不成，应期只能作为“不成之期、变故之期、消息之期”，不可倒推成事。

### 3. 卦意分析法的十二类方向

现有文档已提取世用背向、藏爻确认、冲合预兆。全文核验后，案例分析中还应保留以下卦意取法入口：

- **用神替身法**：当常规用神与卦象主线脱节时，寻找具备现实指代特征的替身爻；
- **动动相连法**：两个或多个动爻之间形成连续现实事件链时，可解释过程、牵连、转移；
- **代入确认法**：将世、应、动、变代入现实角色，验证哪一爻最贴合占事核心；
- **鬼用互化法**：病、忧、风险类占中，鬼与用神互化常提示风险与对象纠缠；
- **背向法**：与用神性质相冲或方向相反者发动、成局、趋旺，常主事与愿违；
- **大环境法**：用神藏伏不在卦面，可能表示所问之事不在当前现实环境内，或问念并非真实焦点。

使用卦意法的边界是：没有明确卦理作用时才可主导吉凶；若卦理明确，卦意只能解释细节与原因。

### 4. 三合局补充：静卦、真假三合、短事延迟

三合局用于吉凶层面时必须满足动变成局，不得把静卦中的静爻直接拼成吉凶三合局。

- **静卦吉凶层面不存在三合局**：静卦应先按用忌持世、世用生克、用旺世兴判断。
- **真三合**：成局爻均有发动或动变基础，且无有效冲局动力。
- **假三合**：成局同时有动爻冲局，合局不能成立，应还原为单爻作用。
- **世爻之局例外**：自占自事中，世爻发动或世爻形成的合局，原则上不主动去作用其他爻，须重点看世爻自身变向。
- **短事见三合**：即使世用同局偏吉，三合也可能代表牵连、聚合、延迟，短期求立刻结果时反成“暂不能成”。

### 5. 进神、退神、化进、化退

进退神不只用于旺衰，也用于过程节奏与细节象：

- 化进：渐进、增强、扩大、多点展开、后续更明显；
- 化退：退缩、降级、减少、热度下降、后续不如当前；
- 临急近事：进退神的“渐变”特征未必能立刻兑现，应避免把长期趋势误用于当天短事；
- 若化进化退爻处于三合局整体中，应优先看合局整体，不宜拆开单论。

### 6. 化散、假化散与中转站原则

动爻化空、化破、化墓、化绝、化散，不得机械视为无用。判断关键是该爻在动力链中的位置：

- 若为最终用事爻、趋势终点，化废多可影响吉凶；
- 若只是桥梁、中转站、过程爻，且已经完成生克传导，常可作假化散、假化破、假化墓；
- 若化废信息与应期相关，可作为事情中途受阻、延期、转折的细节；
- 若化废与卦理主线冲突，先看卦理主线是否已成立。

### 7. 反吟与伏吟

反吟、伏吟在吉凶层面不能脱离世用、动爻、用忌直接断凶：

- 反吟多主反复、往返、折腾、去而复来、关系或过程不稳定；
- 伏吟多主停滞、压抑、原地打转、进展缓慢；
- 若反吟伏吟出现在间爻、阻隔爻、忌神位置，细节上可加重阻滞；
- 若世用卦理链明确为吉，反吟伏吟多解释过程反复，不可推翻吉凶；
- 应期可取逢值、逢冲、冲开停滞之期。

### 8. 真空、真破、空亡术语统一

“旬空/空亡”在报告中统一归为“空”的体系，但须注明真假：

- 真空：目标暂不落实、无力参与吉凶，需待填实；
- 假空：动空、冲空、空而有用，仍可参与生克；
- 真破：静弱被破、核心用神被破且无救，多主衰败；
- 假破：动爻月破、桥梁爻化破、飞神被破露伏等，不必直接废用。

### 9. 不同占类的取用与特殊定性

全文核验显示，同一结构在不同占类下结论不同，案例分析必须先写明占类：

| 占类 | 取用与定性重点 |
|---|---|
| 求财 | 财为用，短期财克世可为得财，长期财福财克世可为受财所累 |
| 行人 | 用神克世、用神趋世可为人回；伏神应重冲飞露伏 |
| 占病 | 近病逢冲可能愈，久病逢冲可能危；子孙为医药喜神，官鬼为病灾 |
| 婚姻 | 世应用、财官、合冲、六合拖延、六冲不久都要结合关系阶段 |
| 官职/考试 | 父母、官鬼、世爻、元神与忌神并审，忌神持世或用藏世伤多不利 |
| 失物 | 用神伏藏、飞神、墓、冲飞露伏、方位与藏处细节并用 |
| 求谋/项目 | 父母为项目文书，官鬼为职位权力，三合局可能表示团队、竞争或延期 |
| 选择卦 | 围绕世爻两端的动兆比较利弊，不可只按单一用神套断 |
| 心态卦 | 子孙、官鬼、世爻状态可直接反映忧喜，不应强行按事卦取用 |

### 10. 趋避与建议层

案例报告最后的建议不得只写吉凶，应把卦中可操作信息转化为趋避：

- 间爻为阻：处理中介、手续、沟通、第三方；
- 用神藏伏：等待显露、补充信息、找飞神所指环境；
- 合绊：冲开、等待解绊、避免被牵制；
- 六冲：短事宜快，长事防散；
- 六合：长事可维系，短事防拖延；
- 墓：寻找隐藏点、解除封闭、避免被困；
- 反吟伏吟：减少反复，先解决停滞源头；
- 三合局克世用：防群体压力、制度压力或多方合力压制。

以上理论可作为后续分析《增删卜易》卦例、用户自测卦例、失物占、求财占、求职占、病占、婚恋占等案例的统一分析底稿。
