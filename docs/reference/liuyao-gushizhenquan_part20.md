__第三十五章: 三墓理论的应用__

三墓，是古筮应用理论中的一项常用知识。墓，就是指长生十二宫中的墓宫。又称为藏库，其影射到现实的含义则非常广泛，有体现在客观形态上的，如隐藏、穴居、拘禁、约束等等；也有反映在主观心态上的，如迷茫、被隐瞒、关注、兴趣等等。理论上金木水火不同属性各有其墓藏之处，分别是金爻墓在丑土、木爻墓在未土、水爻墓在辰土、火爻墓在戌土。古论因发现土爻寄生于水，所以也将水爻之墓辰土作为土爻之墓，但土爻本身就是金木水火之墓，又怎能以他墓盖之？土爻的特殊属性导致本身并不存在墓，因此在吉凶判断层面，辰戌丑未四土爻是不会入墓的，好些学者总被辰土与其他土爻的关系是该论墓？论进？论退？或论冲弄得昏头转向，其实没这个必要。

__墓于吉凶层面的应用：__墓在六爻预测的各个层面，都有其不同的应用功效。在吉凶判断层面，虽然理论上除了土爻，金木水火各有其墓，但前章已有说明，应用于吉凶判断的“生旺墓绝”有个独特性质，就是“从于卦理卦意”，凡遇到一切与卦理卦意相冲突者皆要让路。墓也带有这个特性，因而其在吉凶层面的实际应用中，所谓的四墓，实际只是两墓。其中金爻虽墓在丑土，但卦理上丑土同时也生金爻，若金爻为世用，日上见丑，或卦中丑动，或动而自变丑土，只能论丑土生金爻而不能说丑土墓金爻。如巳月庚戌日占孩子择校，得“睽之大有”卦

__第三十六章: 三绊理论的应用__

记得在之前章节的论述中，已大概接触到一些“绊”的相关概念。古筮纳甲法中有一个理论，名为“相绊”，卦中两爻相合，假若一方动另一方静，不管投靠或牵引的力量，都只是单向性的，谓之动合，或者叫趋合；但如果相合的两爻都同时发动，趋合或牵引对方，你拉我扯之下，两股方向相反的力度碰撞到一块，结果彼此不是被粘在一起、就是相互牵扯住，反而大家都动不了，这就是相绊理论产生的原理。应用在六爻吉凶层面的相绊理论，共有三种形式，因此也被称为“三绊”。第一种形式是日建与卦中动爻或变爻相合，例如寅日占卦，卦中亥水发动，这称为“日绊”，日建将动爻绊住使其不能发挥作用，所以古论有注：“爻静而逢日合谓之合起；爻动而逢日合谓之合绊”。第二种形式是卦中一动爻与另一动爻相合，称为“动绊”，例如卦中午火动、与其相合的未土也动，两爻都主动趋合，反而把各自给绊住不能发挥作用。第三种形式是卦中爻动，自化出的变爻与动爻构成相合，例如戌土发动，变出与之相合的卯木，这称为“化绊”，如同作茧自缚，自己变出的爻却把自己给绊住不能发挥作用了。在吉凶应用层面，绊的应用范围非常清晰细致，绊有三种模式，却不存在所谓的“月绊”，就是说，卦中不管动爻静爻，与月建相合，在吉凶层面均视为合起兴旺，并没有如日建那样，静合为旺动合为绊的说法。这是初学者很容易忽略的基础知识点。

__三绊的吉凶层面应用：__在吉凶层面的三绊理论，重点要了解真绊与假绊的区别。所谓假绊，指的是被绊住的爻，它的总体趋势并没有因此改变，就象竞跑中的人忽然被绊了一下，打了个跄踉却没有被摔倒，速度虽然一时受影响，但并不妨碍他继续向前奔跑。卦象中出现了绊，若定性为假绊，则把它当作没被绊住的正常动爻看待就可以，只是在应期层面的分析上要注意，出现被绊的爻即使趋势不变，被绊了一下，将比原先的进度稍显迟缓。而在吉凶判断上，基本上除了两种特殊条件的卦象，其余绝大部分的绊都应属假绊范畴，并不影响动兆的最终趋势。如未月甲午日占子痘症，得“履之中孚”卦

问卦者是我朋友。卦中妻财巳火，代表他怀孕的妻子，其变出的子孙寅木，就是将要降生的孩子了。巳火回头生得旺，寅木得月令照顾也旺，母子平安的征兆。但卦出六合变六合，乃缠绵不断之象，申金更明动于卦中，绊住妻财巳火化子孙之途，既然预产期已经定下生产的明确期限，他问预产期内何日产子，那这个动绊信息，就应该属于真绊了，启示预期之内事情将没进展。

我问朋友：“你妻子预产期是几号”？回答：“24号，就是说还有六天”。

我道：“绊住就动不了，我怕这六天内生不下来”。对方追问：“哪一天生的机会较大”？

“按卦象，我估计28号或31号可能性较大吧，不过你放心，大人孩子都应该没事的”。28号那天正好是亥日，财爻巳火刚好出空，而子孙又得日合。

果然到了预产期，孕妇还没有要生的迹象，朋友早有预见，倒没有太惊慌失措，27日做产检，医生说已经全入盆了，随时可以生。但28日生产的时候还是把他吓出一身冷汗，不时从产房传来妻子的痛呼，似乎隐约听到妻子在叫“好痛，我要死了”，另一个声音却在说：“你要生了”。正在彷徨不定，一转身猛然发现护士拿着纸笔正候着他：“先生，孩子出不来，必要时可能要钳产，你要签责任书．．．．．．”颤抖着手签了名，心里头更害怕，好在这种煎熬只过了几分钟，护士终于抱着一个女婴出来，报告母女平安。
