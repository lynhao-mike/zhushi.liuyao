# 杯碎外应复盘案例：父母白虎伤灾与财伏旬空破财

## 案例背景

- 占问事项：小儿子把杯子打碎了，预示着要发生什么事？
- 起卦时间：丙午年 甲午月 己未日 乙亥时
- 旬空：子丑
- 本卦：风山渐，艮宫归魂
- 变卦：风地观，乾宫
- 反馈：起卦第二天，卦主母亲去按摩，被按骨折；一周内卦主期货亏钱。

## 初断误区

初断误以“小儿子打碎杯子”为主语，直接取三爻子孙申金持世发动，断为孩子受惊、小伤、小病、家中口舌。此断偏窄，原因是：

1. 占问不是“孩子如何”，而是“杯碎预示什么”。
2. 杯子打碎属于外应，主破损、惊动、伤灾、破财，不应只看导致事件的人。
3. 卦中父母爻与财爻信号更强，尤其二爻父母午火临白虎、五爻伏妻财子水旬空。
4. 子孙持世发动化官鬼，更多说明卦主心神被惊、事件由小孩触发，并不必然应在小孩本人。

## 正确主线

### 一、母亲骨折：父母爻临白虎旺相

二爻父母丙午火临白虎，午月当令，日为己未，火土旺。父母爻代表母亲、长辈、身体依靠；白虎主伤灾、骨损、血光、强硬外力。父母午火既临月旺，又临白虎，是全卦最直接的伤灾信息。

按摩被按骨折，象义对应为：

- 父母爻：母亲、长辈。
- 白虎：骨折、伤灾、外力损伤。
- 午火旺：事件来得急、疼痛明显、炎症火气重。
- 二爻：身体下部、近身处、受人操作处，也可取为实际接触层面。

三爻子孙申金发动化官鬼卯木，是“动而化鬼”，说明小事触发忧患，动后成灾。此动爻不应执着为孩子本身，而应作为整卦的灾患引线看。

### 二、期货亏损：妻财子水伏藏旬空受月冲克

五爻父母辛巳火下伏妻财丙子水，且旬空为子丑，妻财子水正空。求财、投资、期货，关键看妻财。此卦财爻不现，伏于父母巳火之下，且逢旬空，又处午月火旺之时，财受制明显。

期货亏钱对应为：

- 妻财子水伏藏：财不露面，资金被压住，收益不明不实。
- 子水旬空：财空，账面不稳，钱财落空。
- 午月冲子：月令冲财，市场环境直接冲击钱财。
- 飞神父母巳火旺：文书、规则、系统、行情结构压住财爻。
- 朱雀临五爻：消息、行情、交易判断、平台信息引发财务波动。

这条线比“子孙申金是否生财”更关键。子孙申金虽理论上生财，但午月火旺克金，且申金动化官鬼，生财之源转为忧患，不能护财。

## 复盘结论

此卦应断为：杯碎为破损外应，主一周内家中有长辈伤灾与钱财破耗。重点不在孩子本人，而在“破器引动破身、破财”。

- 第一应：父母午火临白虎旺相，应母亲骨折。
- 第二应：妻财子水伏藏旬空受午月冲，应期货亏损。
- 辅象：子孙申金持世动化官鬼，卦主因孩子触发之事而忧患缠身，后续闹心。
- 卦象：风山渐为归魂，主家内、心神不安、事情渐显；变风地观，主观察、复盘、看清风险。

## 修正规则

遇到“某人造成某外应，占预示什么”时，不可机械取造成外应的人为用神。应按以下顺序判断：

1. 先定外应性质：破、响、惊、血、火、水、遗失、动物、来人等。
2. 再看卦中与外应性质最贴合的强信号：白虎、官鬼、父母、妻财、旬空、伏神、月破、日冲。
3. 造成外应的人只作触发点，不必然是承受者。
4. 若父母临白虎旺相，优先防长辈、身体、骨伤、房屋器物损坏。
5. 若财爻伏藏、旬空、被月日冲克，优先防破财、投资亏损、资金落空。
6. 若世爻动化鬼，表示卦主被忧患牵缠，需结合全卦判断忧患落点。

## 经验训诫

此类卦最忌被表面主语牵走。小儿子打碎杯子，只说明外应从孩子手中发出；真正应事，要看卦中哪个信号最强、最贴合“破损外应”。本例中，父母白虎与财伏旬空远强于子孙化鬼，因此应断母亲伤灾、钱财破耗。
