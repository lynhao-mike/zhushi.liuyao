（14）假设子水为用神，丑月亥日占卜，丑月对子水，两者既相合同时又相克，可论月合显旺也可论月克显衰。若子水论月合，爻得月合显旺，同时爻得日扶也旺，子水于日月同旺总体为旺相；若子水论月克，则爻被月克显衰，同时爻得日扶显旺，两者都属一般层次的作用力，两股性质相反的力量同时落在子水身上，爻于日月组合一旺一衰彼此抵消，则子水总体为平相。那子水于丑月该论合或是论克？正确方法一如上例：对照事情客观现状为依据作判断。事实上若存有受克伤之实情，则按克论，若存有得趋合合作的事实，则按合论即可。类似情况还有戌土遇卯月、申金遇巳月等等，彼此也是既相合又相克。《增》书中野鹤对戌土遇卯月，该论合或论克也有专门论述，但可惜他提出的“爻旺论合爻衰论克”分辨论点，却有点不切实际的嫌疑，不可取之为常用法则。如之上所举《增》书丑月庚辰日占子病，得“山地剥”之例，野鹤原想举此例，说明因用神子孙子水受动克、日克衰败，所以虽月建作合仍言克不言合，以证其论。但此卦致凶的关键并非日月组合该怎样论，而是动变组合的原因，卦中的忌神戌土，在月扶旺的状态下受日令冲成暗动，动而克伤用神子水，这才是事凶的核心信息，月令对用神该论相合或论相克，根本无法左右大局，无法证明论据。再看一例十几年前的旧卦，卯月壬寅日占父急症吉凶，摇得“天地否”

__第十五章: 动变组合重要性与旺衰显示方式__

__动变组合对吉凶趋势的影响力：__对爻的旺衰而言，卦中动爻与变爻共同构成另一个整体影响因素，就是“动变组合”。卦之动变属于卦之内因，而卦机在于动，卦爻之动变往往预兆着所预测事情的整个吉凶趋势，“动变组合”相对于“日月组合”而言，动变组合通常是吉凶判断所依赖的关键数据，所以古论有“动重静轻、重内轻外”的卦断法则。在一般的多数情况下，参照日月分析卦中的动变组合，洞悉其趋势与作用方向，即可以准确判断出卦象的吉凶预兆。

__动变组合的特征：__而动必有因，“动变组合”有个明显区别于“日月组合”的特征，就是“日月组合”面对卦中所有的爻产生影响，就如太阳月亮照射大地一般无处遗漏。但“动变组合”却有着明确的专向目标，尤其在吉凶判断层面，其作用方向只有一个爻，生了这个就不能克那个。而且它只会直接或间接性地，专门作用于用神或者世爻，不会针对无相关的其他爻发动。所以要准确分析动变组合的作用方向，首先必须定好用神，不确定用神你就不知晓动变组合运动的真实指向，一旦取错用神，则等于是理解错了动变组合的作用所指，所以说准确取用，是一卦是否能准确预测的前提条件。

__第十六章: 有用动爻的分析辨别__

__卦断吉凶的简便实用之法：__大家可能注意到，在前章论述他爻动，作用于世用使其显出旺衰的时候，提到了“有用动爻”与“三合局”的概念，为何要这样说？这就是这章要论述的内容：在吉凶判断层面，必须要了解动爻的无用之动与复合之动，掌握这两点在卦的趋势定性中非常重要。由于受古代传统六爻理论固定程式的影响，古今绝大部分易者，都不自觉地困在日月动变对用事爻综合影响的硬性对比之中，分析世爻、用神、动爻、变爻都要逐一对比日月动变对各爻的综合力量的大小，总被弄得头昏脑涨。其实大可不必如此，如上所述，只要理清分析层面，把影响一爻旺衰的力量按动静原则分成“日月组合”与“动变组合”两大整体，了解在什么时候应以“日月组合”为吉凶判断主导，什么时候应以“动变组合”为吉凶判断主导；一般情况下两种影响力量何为重何为轻；然后在吉凶层面分析卦中动变的时候，注意区分何为有用动爻何为无用动爻，何为复合之动，就足以快速简便地准确判断出吉凶的方向。我将这种独创的精简高效判断方法介绍给了不少研易者，然而他们一开始都很抵触，认为断易怎可以如此简化，却不知道“大道至简”，往往最简单直接的招数反而最高效实用，此招实乃六爻神断之“截拳道”。自掌握此卦卜内核方法而屡得神验之后，我便切实走上了对所有古筮理论，全面理清应用层次与精简实用化之路。
