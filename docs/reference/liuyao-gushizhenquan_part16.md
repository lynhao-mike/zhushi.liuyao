__六冲卦在吉凶层面的认识：__何谓卦之六冲？即是指出现上卦三爻与下卦三爻两两对冲的卦象，我们习惯称之为六冲卦。64卦中只有十个六冲卦，分别是乾、坤、离、巽、震、坎、艮、兑八纯卦，加上无妄、大壮两卦，记忆起来不难。古人多将六冲卦视为凶兆，但其实六冲卦只代表事情缓急程度与维持时效长短的一种细节性信息，并非吉凶信息的预兆。古论一直以来的观点，认为逢冲必散，散即凶兆，所以卦现六冲，谋事多有不遂。清代野鹤老人的看法虽然有进步，没再偏执于六冲与凶究等同的观点，而须兼顾卦中用神定性祸福，他写道：“冲者散也。凡占凶事，宜于冲散；占吉事，则不宜。亦必兼用神而言，用神若旺，虽冲不碍”，但依然认同“冲即凶”的定位。我则认为，从大的理论框架来讲，无论主卦见六冲，或是卦变六冲，不管占问何事，六冲卦本身都不能作为推测未来吉凶的关键信息，而只能作为吉凶判断的辅助性参考信息之用，卦内的五行动变生克变幻，才是真正吉凶判断的核心依据（当然某些细化条件下，六冲可以作为吉凶的代名词，例如长久之占等等）。或许有人会对此不以为然，古论中不是有句很经典的经验教条：“近病逢六冲即愈久病逢六冲即凶”吗？这就是单凭六冲卦象就可作为吉凶依据的例子啊？我说我也不赞成这种说法，多次验证后我发现，即使是问近病逢六冲卦或卦变六冲，也不能单凭六冲就武断为吉兆，若卦中用神受动变克伤或自变凶衰，照样是凶兆难避，只有在用神不受动克或自变凶衰的前提下，才可以依据六冲断病之吉凶与否。如午月己卯日占妻病，得“震之丰”卦

__主卦六冲的寓意：__主卦中的六冲信息，应用在细节分析方面居多，当然在某些事占上，如占病况、疑虑之占等方面，也可以在作为吉凶判断层面信息应用。问事摇得动卦，主卦往往代表事之现状，变卦才代表事之未来趋势，因此主卦见六冲，只是反映事件客观现实中，存在着近段突发之现状、或因事多变的行为。如问事业突起波澜、占寻人或出行正在张罗寻觅、求仕途而忽见机缘、求财运刚遇起伏、占健康而忽然发病等等事在临近突发的征兆。它反映的是一种现状性质的信息，而不是未来趋势的信息，因此它对事态未来变化的最终趋势，影响力较为有限。如之上未月辛卯日占某笔生意财运如何，摇出六冲艮卦变蛊卦，六冲反映的只是事态的某些现状，对趋势的方向性判断影响不大。又如卯月甲申日占会试，得“艮之益”卦

__第三十章: 六合理论的应用__

六合，也属于古筮预测法中的一门重要理论，其应用范围之广泛一如六冲理论。六合也分爻之六合与卦之六合，爻之六合就是指十二地支中的两两对应相合，我们祖先发明的天干地支符号，将天地万物之自然属性，高度浓缩在十数符号之间，世间的万事万物各自均有其天生的好与恶，相恶者如逢天敌，见之相互冲克，故天干地支均有对冲之爻；相好者如见知己，遇之则彼此合拍，因此天干有五合、地支有六合。天地万物在不断的生克冲合间擦出火花产生源源不断的动能，从而形成宇宙间循环不息的运转变幻。爻之六合分别为：子丑相合、亥寅相合、卯戌相合、辰酉相合、巳申相合、午未相合；古论中还将两爻间的相合细论为合化，如卯戌合化成火、亥寅合化成木等等，但事实上，传统古筮吉凶层面分析上，并不需要用到合化的理论，照样可以做到预测的精确无误，野鹤老人的《增删卜易》可谓是纳甲六爻经验论的大成之作，从头至尾并不见有丝毫合化言论见于纸端，或许有人不认同，祖宗既然有此理论流传下来，必有其精华所在，随便舍弃岂非可惜？我说，我从来没有轻易否定合化功用的存在，但我认为合化的理论并不适用于古筮某些分析层面，其功能不如其他六爻常用理论般有实效，要知道每一种理论学说，都只是预测分析的使用工具而已，就象上山砍柴，如果有多把斧头供你选择，你其实只要挑选其中一两把较为锐利趁手的斧头带上就足够了，难道能说舍弃的斧头都不能砍柴吗？当然不是，只不过斧头只是砍柴的工具，挑好使唤的带上就可以，没必要全部都扛到身上自己找自己麻烦。相似道理的还有，古人用摇钱六爻法断卦，只取日柱与月支就完全足够应付预测，而到了现代则花样变得越多，不单把月干都用上，还嫌日柱月柱反映的信息量不足够，硬是把年柱与时柱也添上去，竟以四柱时空断卦，以为工具多了判断就可以更详尽细致，我并没有验证过这类方法的实用性，当然不好说三道四，但我是绝对不去跟从这种断卦方式的，因为仅以日月为时空信息，就已经完全可以达到清晰预见与准确判断的功效，又何须为了相若的实用性，却去冒重整已成型的系统理论这个风险呢？要知道各种成熟实效的六爻理论都自成体系，就如同是一块钟表的各个零部件，你要加减一个零件，各个零件都需要因此重新整合，以便使新内容完全融入整体发挥应有的效能，从而形成全新的运作系统，整合得好效能或更高，整合不好反而会影响原有的系统性能，记得金庸的武侠小说《碧血剑》里，曾描述石梁派温氏五老的“五行阵”本来就威力无穷，后来这五老居然昏了头，又在五行阵背后加了十六人的所谓八卦阵，但人多不协调又有什么用，结果反而碍手碍脚。所以我自从体会到古筮学的实用功效后，知道自己已经摸索到了一套正确的研究系统，之后对吸纳每一项新理论我都非常的谨慎，首先考虑的不是该理论是否有效用，而是思索新理论是否与已形成的系统理论产生排斥，是否可以融入到固有的系统理论里充实与提高技能水平，生怕因新理论反而扰乱固有的系统理论思维，最终因小失大。别轻视我这番话，我说的可谓是肺腑之言，不少六爻学者总是瞧见哪儿花香就往哪凑，全然没有自己的主导核心理论，五花八门的理论一个劲儿往自己脑袋里塞，生怕会漏掉宝贝似的，全然没想到众多的各式理论，其实是存在于各自不同甚至相互排斥的理论框架里，不经过实用性筛选胡乱捣到一块，却不懂得如何有效利用与整合，又不舍得抛弃，如同满脑子的垃圾无异。据我所知有不少求学者不是垮在知识量贫乏的门槛上，却因为吸收了太多的不同理论导致知识的“消化不良”，到最后连自己都不知道该依据何种理论断卦才好，等于是自己把自己给废掉，殊为可惜！所以，理论的吸收与筛选，贵在精与实用，不在广博贪多，更贵在专一。

__六合卦在吉凶层面的应用：__卦之六合，即是上卦三爻与下卦三爻两两对合的卦象，也称为六合卦。64卦中共有八个六合卦，分别是天地否、地天泰、水泽节、泽水困、山火贲、火山旅、地雷复、雷地豫，这是必须熟记的。古论多将六冲视为凶煞，自然也错将六合的现象看作福祉了，古言之：“凡得诸合，诸占皆以为吉。占名名成，占利利就，占婚必成，占身发迹，占宅兴旺，占风水聚气藏风，占求谋遂心合意”。这种定位未免不着重心，所以很多初学者受此影响，问事一看见六合卦就欢天喜地，一碰到是六冲卦便立刻愁眉苦脸，以为六冲即是祸六合即是福，这其实是对六冲卦、六合卦性质了解不清晰所导致的误读。如同六冲的理论，不管卦出六合还是卦变六合，其本质并非吉凶的直接信息载体，它只不过是一种意象，表达延续不息与缠绵不止特征的意象，若将六冲比喻为短跑冲刺，六合则是长跑漫步，六冲之力是阳力，急速、快捷，但力过则衰不能持久，其本质在于短期与快节奏；六合之力却属阴力，拖延、缓慢，却可以延续不止影响恒远，愈久愈见功效，其主旨在于长久与慢节奏。问事出现六合卦或卦变六合，并非意味着所占趋势一定就吉利，而是一种说明事态缠绵不息的现状与趋势的信息显示，六合本身仅仅是一个持久性、慢节奏的代表符号而已。古论之所以认为占近病怕见六合，占久患却喜见六合，那是因为近病多是新发的疾病，六合缠绵的特征，将可能导致近病短期内无法根治，拖延变成顽疾；而面对长期不能痊愈的慢性病，最需要的是如何维持病况稳定不波动，占得寓意持久的六合卦，侧面证明病况进入稳定期，所以是好兆，根据的是六合本质寓意的延伸解读，六合信息本身与祸福定义是没有任何直接关联的。如戌月丁卯日占讼事，得“地天泰”卦

炒股等解套是何等的被动，此君听了非常不高兴，自然连睡觉都希望我断的不准。然而事情往往总是这样，越怕黑就越见鬼，周一股市开市即展开调整，虽然指数的跌幅并不大，但他所持有的股票却很邪门，一只跌4%，一只居然大跌8%，原本需要一周时间才能验证是非的卦象，结果只一天就有答案，而且是精确的答案。令他再也坐不住，立刻招呼我过去，自然首先掩盖着沉痛的心情，大大地恭维我一番，这样做的目的不外乎两个：一当然是要再问卦，二是拜托你老兄的嘴巴多吐些吉利话，不要再打击我了。再问所持有股票的趋势，得“临之泰”卦

__第三十一章: 六冲六合互化详述__

上两章分别阐述了六合与六冲的详细运用。了解到无论卦出六合卦六冲卦，或卦变六合卦六冲卦，它们大多不是卦象吉凶预兆的核心信息。但有一种情况却要格外留神，就是当六合卦与六冲卦相互间出现互变的时候，不单吉凶的判断原则有所改变，六冲六合的寓意也相应出现一些特殊的变化。所谓六合卦六冲卦互化，就是十个六冲卦与八个六合卦之间的相互转化，具体的转化形态有四种：六合变六合为其一；六冲变六冲为其二；六合变六冲为其三；六冲变六合为其四。上述四种互化模式，在吉凶层面大致分为两大分析类别。

一种是六合变六合与六冲变六冲。此两类互变，其现状含义与趋势昭示，与单独的六冲六合卦寓意没有太大区别，多应用在对证现状与细节层面拓展思维方面，其信息在吉凶层面，并不能成为左右祸福趋势的关键判断点，但由于合化合、冲变冲，在细节层面将较之一般的六合卦六冲卦程度更为突出。六合变六合，乃是拖延不决的现状加上缠绵不绝的趋势，意味着事情将在纠缠或延缓的基础上，又再缠绵不绝的细节反映，祸事遇之越沾越紧，吉兆逢之鱼水交融。古论把六合之象理解为“合好”，以为六合卦一定代表吉利，故而给现代的学易者带来六合化六合定是“好上加好”的误导，其实六合化六合与事态的祸福定性却是一点不拉弦。如丑月乙卯日占援例，得“豫之否”卦
