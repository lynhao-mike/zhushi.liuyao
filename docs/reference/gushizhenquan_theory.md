# 《古筮真诠》核心理论体系

> 本文档从《古筮真诠》全书23章及知识点总结中系统提取所有理论规则，
> 按九大板块组织，每条规则标注适用层面：[吉凶层面]、[应期层面]、[细节层面]。
> 旨在为六爻分析系统提供可实现的if-then规则基础。

---

## 一、总论与基础概念

### 1.1 两大分析层面

六爻断卦分为两大层面：

1. **吉凶判断层面**：确定事情的成败、好坏趋势
2. **应期细节层面**：确定事情发生的时间、过程细节

**核心原则**：
- 吉凶层面从严，规则固定，不可随意发挥
- 应期细节层面从活，允许飘逸思维，生克冲合信息皆有现实意义

### 1.2 七步完整分析流程（专业级）

```
步骤1：实证对轨 - 通过日月现状信息与客观事实对证，验证卦象真伪
步骤2：明确用神 - 确定真正的用神（真假辨别、客观取用）
步骤3：吉凶定性 - 根据卦理/卦意判断事态吉凶趋势
步骤4：应期推测 - 推算事情发生的时间节点
步骤5：细节铺开 - 展开过程细节、程度分析
步骤6：事外解读 - 分析与主问相关或无关的另外信息
步骤7：趋吉避凶 - 提供化解建议与指导
```

### 1.3 三步入门流程（初学级）

```
步骤1：取用神 - 确定分析目标爻
步骤2：断吉凶 - 判断成败趋势
步骤3：推应期 - 推算时间
```

### 1.6 核心原则

#### 动必有因
- 爻之发动必有原因，动兆是吉凶定性的主导信息
- 有动兆就有应期，无动兆则参照静卦规则

#### 重动轻静、重内轻外 [吉凶层面]
- 动因（卦内动变）重于静因（日月外部环境）
- 内因（自变回头作用）重于外因（化日月建等）
- 爻动而化月建同时又化绝：化绝为真（内因），化月建为假（外因）

#### 卦象显示特征
1. **只答心念不答口述**：卦象回答的是心中真实所想，非口头所问
2. **多问简显简问多现**：问得多卦象简单显示，问得少卦象信息丰富
3. **预兆舍远应近**：同时有远近之事，卦象优先显示近期事
4. **显像就重避轻**：有更重要的事，卦象会避开所问转显重要信息
5. **意念扶正伐邪**：卦象倾向指引正确方向
6. **所答不谋专事**：卦象未必只回答所问之事

## 二、日月组合理论

### 2.1 月令对爻的作用

#### 2.1.1 月令使爻旺相（四种方式）[吉凶层面]

| 序号 | 方式 | 说明 | 示例 |
|------|------|------|------|
| 1 | 爻临月令 | 爻的地支与月令相同 | 卯木爻在卯月 |
| 2 | 爻得月令合 | 爻与月令六合 | 卯木爻在戌月（卯戌合） |
| 3 | 爻得月令生 | 月令五行生爻 | 金爻在土月 |
| 4 | 爻得月令扶 | 月令五行与爻同类 | 金爻在金月 |

#### 2.1.2 月令使爻衰败（四种方式）[吉凶层面]

| 序号 | 方式 | 说明 | 示例 |
|------|------|------|------|
| 1 | 爻遭月令冲 | 月令冲爻（月破） | 卯木爻在酉月 |
| 2 | 爻被月令克 | 月令五行克爻 | 木爻在金月 |
| 3 | 爻克月令 | 爻五行克月令（耗气） | 金爻在木月 |
| 4 | 爻生月令 | 爻五行生月令（泄气） | 金爻在水月 |

#### 2.1.3 月令平相规则 [吉凶层面]

- **得月气**：爻在月令有余气，趋旺方向
- **失月气**：爻在月令无余气，趋衰方向
- 示例：巳火在未月，火爻气在未，虽生月属休囚，但有气可争取平相

#### 2.1.4 月令旺衰矛盾处理 [吉凶层面]

**规则**：爻与月令相合又相克（如卯与戌），或相冲又相扶（如丑与未）时：
- IF 客观现状是得到帮扶资助 → 论合（旺）
- IF 客观现状是遭伤受损 → 论克（衰）
- IF 无法对证现状 → 土爻逢月冲一概视为月破；爻被日月克中带合，一律视为相克

**重要区别**：月令之合与日令之合不同：
- 月令相合：不管动爻静爻变爻，吉凶层面都只论合旺，无月绊概念
- 日令相合：静爻论合旺，动爻论日绊

### 2.2 日令对爻的作用

#### 2.2.1 日令使爻旺相（五种方式）[吉凶层面]

| 序号 | 方式 | 说明 |
|------|------|------|
| 1 | 爻临日建 | 爻地支与日令相同 |
| 2 | 静爻得日令合 | 静爻与日令六合（动爻为日绊） |
| 3 | 爻得日令生 | 日令五行生爻 |
| 4 | 爻得日令扶 | 日令五行与爻同类 |
| 5 | 爻临日令长生与帝旺 | 如土爻逢申日（长生）、逢子日（帝旺） |

#### 2.2.2 日令平相（二种）[吉凶层面]

| 序号 | 方式 | 说明 |
|------|------|------|
| 1 | 爻克日 | 爻五行克日令 |
| 2 | 爻生日 | 爻五行生日令 |

#### 2.2.3 日令使爻衰败（二种）[吉凶层面]

| 序号 | 方式 | 说明 |
|------|------|------|
| 1 | 爻被日令克 | 日令五行克爻 |
| 2 | 爻遭日令绝 | 爻在日令处绝位，如子水逢巳日 |

#### 2.2.4 生旺墓绝在日令的应用 [吉凶层面]

**核心规则**：
- 长生与帝旺 → 代表旺相
- 绝 → 代表衰相
- 墓 → 中性，爻旺附旺，爻衰附衰

**条件判断**：
- IF 爻得动变力量生旺 OR 月令显出兴旺 → 日令之绝不论绝，为平相
- IF 爻在月令衰败 OR 被动爻相克 → 日令之绝论绝，显衰相

示例：酉金在申月寅日，酉得月扶故在日论平不论绝，总体旺相

### 2.3 日绊理论 [吉凶层面]

**定义**：动爻被日令相合，称为日绊

**规则**：
- 静爻得日合 → 论合旺
- 动爻被日合 → 论日绊（真绊/假绊）
- 变爻被日合 → 论日绊

**例外**：动爻被"旺相式特殊日月组合"笼罩，形式为月旺日合时，不论日绊，论高层次合旺

### 2.4 月破理论 [吉凶层面]

#### 月破现实对应

月破信息对应现实：
- 所谋已受重创
- 问病时已病重药穷
- 问工作时已遭解雇
- 事情尚未开展/尚未发生

#### 动破真假之分

| 序号 | 假破情形 | 说明 |
|------|----------|------|
| 1 | 动而逢破 | 动爻本身月破，不为真破 |
| 2 | 动而化破但变爻可回头作用 | 有回头生克关系，不为真破 |
| 3 | 动而化破但动化进退 | 特别是化进神又化破，进退为真破为假 |
| 4 | 自占自事世爻动而化破 | 世爻化破多通过变出六亲定性，不为真破 |

### 2.5 特殊日月组合 [吉凶层面]

#### 2.5.1 旺相式特殊组合

**高高共指向**：
- 月建+日建、月合+日建、日合+月建、日合+月合

**高低共指向**：
- 月建+日生、日建+月生、月合+日生、日合+月生
- 月建+日扶、日建+月扶、月合+日扶、日合+月扶

**特殊功效**：面对动变逢克不衰

#### 2.5.2 衰败式特殊组合

- 月破+日克
- 月破+日墓
- 月破+日绝

**特殊功效**：面对动变逢生不起

**注意**：日冲不能定义为衰败元素，冲本身无旺衰固定标签

#### 2.5.3 特组不能发挥功用的情况

| 序号 | 情况 | 说明 |
|------|------|------|
| 1 | 被特殊动爻生克 | 生中带合的动爻可生旺衰败特组；克中带冲、三合局克可克伤旺相特组 |
| 2 | 动而化进退绝破搭桥 | 动力方向为变爻，特组不再发挥作用 |
| 3 | 与太过卦混淆 | 旺相特组+动变生/化进=太过卦，按太过定性 |
| 4 | 时效性限制 | 时间越长特组功效越低 |

### 2.6 日月时段规则 [应期层面]

**核心口诀**：
- **短事**：日转月不转（日令主导，值日如司令）
- **常事**：月转日不转（月令主导，但日令恒长）
- **长事**：日月皆不转，只有太岁动

**解释**：
- 短事之测：用事爻临日令，当日如同司令受克不凶，过日失效
- 常事之测：月令功能总大于日，但月令常换日令始终不变
- 长事之测：日月组合只代表现状环境，不代表长期趋势

## 三、动变组合理论

### 3.1 趋向旺相的动变组合（五种）[吉凶层面]

| 序号 | 组合方式 | 条件 | 结论 |
|------|----------|------|------|
| 1 | 爻被有用动爻相生 | 动爻有用且五行生目标爻 | 目标爻趋旺 |
| 2 | 爻被动变三合局来生 | 三合局成立且生目标爻 | 目标爻趋旺 |
| 3 | 爻动变回头相生 | 变爻五行生动爻 | 动爻趋旺 |
| 4 | 爻动而化进神 | 寅→卯、申→酉、丑→辰、未→戌 | 力量源源不绝 |
| 5 | 爻动变出临日月之爻或合月之爻 | 变爻不产生回头生克墓等作用 | 动爻趋旺 |

### 3.2 趋向衰败的动变组合（六种）[吉凶层面]

| 序号 | 组合方式 | 条件 | 结论 |
|------|----------|------|------|
| 1 | 爻被有用动爻相克 | 动爻有用且五行克目标爻 | 目标爻趋衰 |
| 2 | 爻被动变三合局来克 | 三合局成立且克目标爻 | 目标爻趋衰 |
| 3 | 爻动变回头相克 | 变爻五行克动爻 | 动爻趋衰 |
| 4 | 爻动而化退神 | 卯→寅、酉→申、辰→丑、戌→未 | 后劲不继变废 |
| 5 | 爻动而化绝 | 且不存在回头生 | 动爻趋衰 |
| 6 | 爻动而化破 | 化月破或日破，且不产生回头生克墓 | 动爻趋衰 |

**特殊注意**：自占自事中世爻动而化破为假破，吉凶聚焦在世爻变出的六亲信息

### 3.3 旺衰间摇摆的动变组合（三种）[吉凶层面]

| 序号 | 组合方式 | 旺时效果 | 衰时效果 |
|------|----------|----------|----------|
| 1 | 静爻被有用动爻所冲 | 冲起不衰（有用） | 冲破而衰败 |
| 2 | 静爻被有用动爻所趋合 | 更显旺相 | 立足不稳被吸引显衰 |
| 3 | 静爻被有用动爻所墓 | 暂时被墓，出墓不衰 | 永远被墓显衰败 |

**冲的特殊规则**：
- 动爻冲克静爻且被冲者为用神 → 吉凶层面只论用神受动克
- 动爻冲克一般静爻 → 可论为冲起

**趋合的特殊规则**：
- 丑土克合子水、卯木克合戌土、巳火克合申金 → 吉凶层面论动克不论相合
- 应期细节层面 → 生克冲合都可自由发挥

### 3.4 无用动爻（四种）[吉凶层面]

| 序号 | 类型 | 特征 |
|------|------|------|
| 1 | 动变回头克之爻 | 变爻克动爻 |
| 2 | 动而化退之爻 | 卯→寅等退神 |
| 3 | 动而化破化散之爻 | 化月破或化日破，不同时存在回头作用 |
| 4 | 动而化绝之爻 | 如子水变巳火 |

**共同特征**：动而变衰，对世用不再产生作用

**无用变有用的五种情况** [吉凶层面]：
1. 特殊日月环境/特殊时效卦条件下
2. 化回头克/化破/化绝对应现实中放弃/衰竭/堵塞时，整体作为有用信息
3. 短事之占中：日破之爻、动而化退之爻、动而化绝之爻可提升到吉凶层面
4. 长事之占中：元神发动变废 → 根源衰竭为凶兆依据
5. 动化绝/化破同时又化用神：取决于事主主观是否放弃

### 3.5 矛盾趋势分析原则 [吉凶层面]

#### 原则一：动能轨迹终点原则
- 动爻的最终方向以变爻为终点
- 动始变终，变爻代表最终结局

#### 原则二：重动轻静、重内轻外
- 动因（卦内动变造成旺衰）> 静因（日月造成旺衰）
- 内因（化进退、化绝、回头生克）> 外因（化日月建、化月合）
- 示例：爻动化月建同时又化绝 → 化绝为真（内因），化月建为假（外因）

#### 原则三：自占自事世爻之变原则
- 世爻之动不存在有用无用概念，必须重视
- 世爻动变体现个性规则：
  - 世爻变出用神/忌神/官鬼子孙（不回头生克）时
  - 即使同时化日破月破/化日建月建
  - 吉凶按所化六亲定性，不按所化日月定性

**世爻特殊规则列表**：
- 世变忌神+化日建月建 → 变忌神不利为真，化旺为假
- 世变用神+化日破月破 → 变用神吉利为真，化破为假
- 世变子孙+化日破月破 → 变子孙吉为真，化破为假
- 世变鬼+化日建月建 → 变鬼不利为真，化旺为假

### 3.6 复合之动 [吉凶层面]

#### 表现形式

1. 数动爻构成三合局 → 整体以合局形式作用世爻或用神
2. 数动爻连动相生 → 整合形式指向作用世爻或用神
3. 数动爻连动相克对抗抵消 → 内部整合后指向作用世爻或用神

#### 复合动爻运转规则（四条）

| 序号 | 规则 | 说明 |
|------|------|------|
| 1 | 三合局优先于单爻连动 | 有三合局先论三合局 |
| 2 | 复合连动目标围绕世用 | 无目标则动爻失去连动意义 |
| 3 | 即使单爻可直接作用，仍须先连动 | 只要连动后方向仍围绕目标 |
| 4 | 世爻不参与复合连动 | 自占自事吉凶层面，世爻只被作用不作用他爻 |

### 3.7 世用关系影响下的吉凶判断 [吉凶层面]

#### 五条核心规则

| 序号 | 规则 | 解释 |
|------|------|------|
| 1 | 用神必须旺相 | 用神衰败则事不成 |
| 2 | 用旺世兴 | 用神旺+世爻得日月其一生扶=吉；用旺世衰=自身原因不成事 |
| 3 | 世爻不能受动克或自变衰败 | 世动化退/化绝/回头克/化忌神/化鬼=世爻受制（化破除外） |
| 4 | 动爻作用次序先用神后世爻 | 动爻先作用用神定事成败，再作用世爻定自身利弊 |
| 5 | 外动为过程内动为结局 | 外卦动爻反映过程，内卦动爻反映结局 |

#### 动爻无目标时的回归规则 [细节层面]
- 动爻不再作用世爻或用神时 → 冲合墓的自然属性功能扩大
- 可冲起或冲破静爻、趋合与动墓他爻
- 通过间接作用达到影响世用的目的

### 4.2 真假用神辨别 [吉凶层面]

#### 真用神五种特征

| 序号 | 特征 | 说明 |
|------|------|------|
| 1 | 有动爻作用的 | 被元神生或被忌神克 |
| 2 | 用神为动爻 | 包括暗动；或为动之变爻直接针对世爻显示吉凶 |
| 3 | 用神临世应 | 静卦忌神持世也算；临君爻同时为心念爻也算 |
| 4 | 用神与日月相关 | 值日、值月、合日、合月、日冲、月冲（月破） |
| 5 | 用神旬空或入墓 | 爻不会无故而空无故而墓，必与事态相关 |

**判定规则**：五项特征无一符合 → 基本确定为假用神

**注意**：伏神、飞神都不破不空时，藏伏的其实是假用神

### 4.3 客观取用法（三要点）[吉凶层面]

#### 要点一：真假用神辨别
- 可根据卦象重点显示形式定位用神
- 重点体现在：信息显示的重要程度 + 简单直接程度
- 以动爻直接作用、用神发动、用神持世、用神临应爻等方式直截了当表达

#### 要点二：根据客观现状灵活取用
- 取与客观事实相对应的备选爻为用神
- 用神多发时，对证现状取用神
- 卦象是事情的信息倒影

#### 要点三：懂得心态卦用神特征
- 对事情关心过度 → 卦随念转 → 事态卦变心态卦
- 心态卦：以子孙（喜神）和官鬼（忧神）为分析核心
- 事态特征单薄+心态特征明显 → 按心态卦分析

### 4.4 变爻为用神 [吉凶层面]

#### 4.4.1 世爻变出用神

| 情形 | 结论 |
|------|------|
| 世变用神回头生 | 事吉且受益 |
| 世变用神回头克 | 事凶受损（问病子孙回头克世、问财财回头克世亦凶） |
| 世变用神无回头生克 | 卦意"世变用神"吉兆（即使变爻月破日破不按化破看） |

#### 4.4.2 动爻变出用神

**承载爻能直接作用世爻时**：
- 带生：承载爻不被回头克，动而生世 → 吉
- 带克：承载爻不被回头克，动而克世 → 凶（即使特殊事占用克世为吉，间接克世一律凶）
- 带废：载体属无用动爻 → 变爻用神也一块作废

**承载爻不直接作用世爻时**：
- 成为间爻阻隔 → 卦意凶兆
- 反生应爻或与世相冲之爻 → 事不成（用神与他人相接与己无关）
- 占彼此关系：承载爻可直接作为用神替身与世爻对比
- 独发有用不作用世爻不在间爻 → 可为最终成功吉兆（动始变终）

**注意**：若变爻用神无法通过载体清晰回答事情 → 可能是假用神

### 4.5 日月为用神 [吉凶层面]

| 月为用，克世/冲世/世生月/世克月 | 凶兆（用神直接伤世型） |

**时效规则**：
- 月令用神时效：当月有效（长事之占应在当月赶快办）
- 日令用神时效：短在当日，长在月外

**取舍原则**：有用神在日月必有伏神，若伏神信息更简单直接能得出结论 → 取伏神为用

## 五、吉凶判断完整流程

### 5.1 动卦吉利卦局 [吉凶层面]

#### 5.1.1 世用受生局
- **条件**：自占事，用神与世爻重叠于一点，同时受到动爻生旺
- **结论**：事成且自身受益
- **特点**：众多吉利卦局中吉利指向最明显的一种

#### 5.1.2 用神生世局
- **条件**：用神是有用动爻，世爻不自变衰败，用神动而生世爻
- **结论**：所占事情使我获益
- **三个子类**：
  - 直接受益：用神直接生世
  - 间接受益：用神通过连动间接生世
  - 合局受益：三合局生旺世爻

#### 5.1.3 用旺世兴局
- **条件**：用神旺相 + 世爻得日月其一生扶（即便总体衰败亦无妨） + 世爻没受动克或自变衰败
- **结论**：事有成功机会且自身能把握住
- **注意**：人与事瓜葛不十分明显的卦局，容易与卦意法、双核卦冲突

### 5.2 动卦凶兆卦局 [吉凶层面]

#### 5.2.1 用神衰败局
- **条件**：用神在日月动变综合下趋向衰败
- **结论**：事情不成

#### 5.2.2 世爻受伤局
- **条件**：自占事，世爻受到动爻克伤 OR 动而自变衰败
- **结论**：自身受伤害

#### 5.2.3 世用受克局
- **条件**：自占事，用神与世爻重叠于一点，同时受到动爻克伤
- **结论**：谋事不成，且自身也因事败而遭受伤害

#### 5.2.4 用旺世衰局
- **条件**：用神旺相，但世爻衰败/不兴
- **结论**：事成机会有，但因自身条件不足最终不利
- **注意**：世爻衰弱有时体现主观参与欲望不强

#### 5.2.5 用神克世局
- **条件**：用神克世爻
- **结论**：不管用神旺否都不算吉利
- **细分**：
  - 用神衰败 OR 世爻衰败不堪用神之克 → 只是事情不成功
  - 用神有用克旺相世爻 → 短期不排除成事，但长久终不利
  - 短事变长久或本为长事 → 即使短暂成事，终走凶途

### 5.3 卦局通论中的特例 [吉凶层面]

| 特例类型 | 条件 | 结论 | 对世爻要求 |
|----------|------|------|-----------|
| 自占求财 | 财爻有用动克世爻 + 世爻日月之一生扶总体平相 | 有财预兆 | 需平相以上 |
| 自占疾病医药 | 子孙爻作用神动克世爻 | 医药有效（克的是鬼非世） | 无要求 |
| 占问行人 | 用神动而冲克世爻 | 行人将很快回来 | 无要求 |
| 忧患心态卦 | 喜神子孙动而冲克世爻 | 扫去心中忧虑的吉兆 | 无要求，不怕世动回头克 |

**求财特殊注意**：
- 财克旺世为得财 → 细节上过程辛苦/压力大
- 财生世为得财 → 细节上过程舒畅且自身受益
- 占终身财福 → 财克世为受财所累之兆

### 5.4 静卦吉凶判断三步法 [吉凶层面]

#### 第一步：先看有无用忌持世

**用神持世规则**：
- 自问自事得静卦用神持世 → 不管世用旺衰，即属吉兆
- **例外凶**：用神持世"无根"→ 短期无虞但长久为凶
- **例外凶**：用神持世"逢破"→ 凶兆即时生效

**忌神持世规则**：
- 自问自事得静卦忌神持世 → 不管世爻旺衰，即属凶兆
- **例外吉（四种）**：
  1. 求财之占：兄弟持世逢月破/日破/旬空/化墓/日月作财冲世 → 得财吉兆
  2. 医药之占：同上规则
  3. 行人之占：同上规则
  4. 忧患之占：同上规则

**不适用条件**：非以六亲为用神的卦（如取世爻为用、星煞为用、六神为用），代占卦

#### 第二步：再看世用生克关系

**用神生世格局**：
- 条件：用神在日月有生扶保持总体平相
- 结论：即为吉兆，不需再分析世爻旺衰
- 如同动卦用神动而生世

**用神克世格局**：
- 条件：不论用神或世爻旺衰如何
- 结论：即不为吉
- 细分：
  - 用神衰败/世爻衰弱 → 事败，但塞翁失马免受伤
  - 用神得力+世爻旺相 → 短期有成功可能，但长期引狼入室

**静卦用神克世反吉**：
1. 求财：财爻克世反得财（用神需平相+世爻不能衰弱）
2. 占行人：用神克世人速回

#### 第三步：非用忌持世又非生克世，参照"用旺世兴"

- 各自分析用神与世爻在日月的旺衰
- 或分析它们受暗动之爻的旺衰作用
- 用神旺相+世爻得日月之一生扶 → 可断为吉兆

### 5.5 静卦暗动应用 [吉凶层面]

**短事性质**：所有暗动都可介入静卦吉凶定性，成为决定事件趋势的关键

**长事性质**：
- 用神/元神暗动 → 长事快应征兆（不视为日破）
- 忌神暗动 → 同上可入吉凶层面
- 其他爻暗动 → 吉凶层面一概忽略，只在应期细节层面分析

### 5.6 卦意分析法（十二种方法）[吉凶层面/细节层面]

#### 1. 世化用忌法（自占自事）
- 世爻变出用神 → 吉兆
- 世爻变出忌神 → 凶兆
- **例外**：以官鬼为用神时世化官鬼 → 看日月旺衰（旺为官，衰为鬼）

#### 2. 用忌互化法（代占他事）
- 用神变忌神 → 事态由好转坏
- 忌神变用神 → 事态由坏转好
- **例外**：自占自事用神持世自变仇神时例外

#### 3. 鬼用互化法
- 问事鬼用互化 → 祸患与事情纠缠不散
- 求财鬼用互化 → 求财反被骗或落入祸非
- 问病鬼用互化 → 病重忧患趋势
- 当官鬼为用神时 → 不存在鬼用互化

#### 4. 世动化鬼法
- 世动化鬼 → 被祸患纠缠
- **例外**：官鬼为用神且变出官鬼日月不衰 → 此官鬼为官星
- **例外**：心态卦中父母持世动化官鬼回头生 → 卦理吉兆非化鬼

#### 5. 世用背向法
**三吉**：
1. 世兴前提下，用神动变出与世爻相合之爻
2. 用旺前提下，世爻动变出与用神相合之爻
3. 用神旺相不动，应爻动变出与世爻相合之爻

**三凶**：
1. 用神动变出与世爻相冲之爻
2. 世爻动变出与用神相冲之爻
3. 用神不动，应爻动变出与世爻相冲之爻

**优先级**：卦理（回头作用）> 卦意（背向法）

#### 6. 搭桥趋变法
- 世用动变出A，卦中A又动变出B → 视同世用直接动变出B
- **四个限制**：
  1. 两动爻间不能对世用有卦理生克关系
  2. 各自变爻与动爻不能有针对世用的回头生克
  3. 两变爻不能合成三合局
  4. 最终变爻以六亲定吉凶，非回头作用定吉凶

#### 7. 牵连聚合法
- 动爻与变爻相连（如女占世变官鬼=结合）
- 各动爻聚合成三合局（如世与财同在合局=得利）
- 可扩展到卦象层面

#### 8. 代入确认法（替身法）
- 分析与世用同属性/相冲/相合的爻
- 相冲者=对手，相合者=朋友/配偶，同属性=同类
- 通过载体间接与世爻产生对比关系

#### 9. 藏爻确认法
- 多见于有指定目标的嫁接用神之测
- 用神藏伏不露为明显特征
- 用神藏在与世应冲/同属性爻下 → 所求不在此处在另处
- 用神藏在应爻之内 → 找对门路吉兆

#### 10. 间爻阻隔法
- IF 阻隔成立 → 即使用神旺相也不改凶兆
- 动爻成为间爻阻隔元素：有用则阻隔凶，化衰则阻隔不成

#### 11. 冲合预兆法
- 根据与用神相冲者或相合者的变化趋势判断吉凶
- 条件：用神必须藏伏 + 与用神相冲/相合者在卦中独发动变

#### 12. 动动相连法
- 动爻与动爻优先连接的分析思路
- 适用条件：竞争性质、明争暗斗之事占、短事之占

## 六、应期推算完整体系

### 6.1 二十一条应期公式 [应期层面]

#### 公式(1)：静爻逢值逢冲
- 主事爻静旺 → 多应在逢冲之期
- 主事爻静衰 → 多应在逢值之期
- 有时应在元神或忌神临值之时

#### 公式(2)：动爻逢合逢值
- 动爻临酉金发动 → 逢辰期（合）或酉期（值）应事

#### 公式(3)：月破逢填实补破
- 逢填实（爻值日）
- 补破（逢合）
- 出月破之期

#### 公式(4)：日冲逢填实补破
- 逢填实或补破之时应事
- 最快当日或次日应事

#### 公式(5)：旬空逢填实冲实出空
- 主事爻临午火旬空 → 逢子（冲）或午（值）之期
- 或过了子午之期（出空）

#### 公式(6)：遇合逢两冲/遇冲逢两合
- 遇合逢两冲之期应事（含三绊应期）
- 遇冲逢两合之时应事
- 例：午动变未为绊 → 逢子、丑期应事；或午、未两值之期

#### 公式(7)：动逢月合
- 一般数后三个月内应事

#### 公式(8)：同属性之期
- 主事爻临子水应在子期，但可能在亥期（同属性相连）

#### 公式(9)：变爻应期
- 非回头生克 → 逢值逢冲之期
- 存在回头生克 → 逢值逢合之期

#### 公式(10)：化进神应期
- 逢值、逢合或逢化进之期
- 例：申化酉 → 逢申（值）、巳（合）、酉（进）期

#### 公式(11)：化退神应期
- 逢两冲或逢化退之时
- 例：酉化申 → 逢卯、寅（两冲）或申（退）期

#### 公式(12)：三合局应期
- 局内凶：破局之期（冲去三爻其一）
- 局外忌：成局之期（补众破绽）
- 吉断：喜局宜成忌局宜破
- 三合缺一：补缺成局之时（如巳酉缺丑→逢丑期）

#### 公式(13)：三墓应期
- 逢冲墓、冲爻或出墓期
- 例：午火墓于戌 → 逢辰期（冲墓）、子期（冲爻）、或过辰子之期

#### 公式(14)：三刑应期
- 常见：巳申寅、酉午亥、丑未戌、丑未辰、辰戌丑、辰戌未
- 三刑缺一 → 逢补全之期应凶
- 三刑齐全 → 逢构成元素临值之期应凶

#### 公式(15)：反吟应期
- 逢反吟元素逢值之期应事
- 例：反吟组成寅与申 → 到寅申之期出现反复

#### 公式(16)：伏吟应期
- 最终目标爻伏吟 → 逢值逢冲之期
- 非最终目标爻 → 与一般动爻无异，逢值逢合

#### 公式(17)：藏伏应期
- 逢值或逢合之期
- 飞爻逢冲之期（冲飞露伏）
- 飞爻空亡期间（飞空伏现）

#### 公式(18)：定性吉而爻受克
- 待忌爻受冲克之时应事

#### 公式(19)：定性凶而爻受生
- 待生旺元素被冲克之时应凶

#### 公式(20)：事吉爻过旺/事凶爻过衰
- 事吉爻过旺 → 逢墓或逢克之期应吉（太过者损之斯成）
- 事凶爻过衰 → 逢长生或逢生旺之期应凶

#### 公式(21)：用神多现/元神忌神多发
- 宜墓库以收藏
- 逢重叠多发者之墓期应事
- 例：辰土用神，巳午齐发生旺 → 逢火墓之戌期

### 6.2 五条应期原则 [应期层面]

| 序号 | 原则 | 说明 |
|------|------|------|
| 1 | 吉断吉应凶断凶应 | 吉兆对应吉利应期，凶兆对应凶险应期 |
| 2 | 盯面不盯点 | 锁定时段范围，多现/就近/相邻原则筛选 |
| 3 | 应众不应寡 | 多角度分析重复出现的理论应期概率更大 |
| 4 | 应早不应迟 | 最早到达的理论应期最大优势 |
| 5 | 应邻不应单 | 如卯巳午酉中，巳午相邻概率更大 |

### 6.3 应期变速因素 [应期层面]

#### 加速因素
- 用事爻明动冲起与暗动
- 带日令而动
- 卦变六冲

#### 减速因素
- 三合局
- 三绊
- 卦变六合
- 变出渐卦/恒卦

### 6.4 近占远占应期区别（八方面）[应期层面]

#### (1) 明冲之爻
- 近占：适宜反映，快速急促
- 远占：力不恒久，反复波动

#### (2) 暗动与日破之爻
- 近占：用神/元神日破可作为吉凶层面短期不利信息
- 远占：元神暗动或日破只能用到细节层面

#### (3) 复合动变
- 短占：剔除无用留下有用定性
- 长占：元神发动变废=根源衰竭凶兆

#### (4) 六冲与六合
- 近占见六合弊端大于见六冲
- 远占见六冲弊端大于见六合

#### (5) 三合与三绊
- 近占：延迟负面放大
- 远占：延缓释放久远利好

#### (6) 动而化绝
- 远占：肯定按化绝定位
- 短事/心态卦：不能轻易归类无用
- 特殊：化绝同时化冲（反吟）→ 按反吟有用看

#### (7) 日破月破/日建月建
- 临建：短事发挥利好，长事带变数
- 临破：长事发挥利好，近事利差

#### (8) 用神克世
- 近占：世爻旺相 → 短期可论吉利
- 远占：不管世爻如何一律凶兆
- 分界：一月左右为近占界限

### 6.5 静卦应期特点 [应期层面]

1. **静而逢值逢冲应**：旺逢冲，衰逢值
2. **元神信息地位重要**：静卦应期有时聚在元神而非用神
3. **静卦非必无急兆**：暗动或用神带日月 → 急兆信息

## 七、特殊卦类专论

### 7.1 暗动理论 [吉凶层面/应期层面]

#### 7.1.1 冲起暗动有用的六种情况

| 序号 | 情况 | 条件 | 说明 |
|------|------|------|------|
| 1 | 旺相者冲之不散 | 静爻得月令趋旺而受日冲 | 旺相之爻冲不散 |
| 2 | 有气者冲之不散 | 静爻得月令之气而受日冲 | 但丑月亥水/辰月寅木/未月巳火受日冲基本应日破 |
| 3 | 冲空 | 静爻旬空状态下受日冲 | 即使月破衰极也被冲起 |
| 4 | 动爻趋旺 | 静爻月令休囚但受日冲同时得动爻来生旺 | 外力补旺 |
| 5 | 用趋世兴 | 用神旺相条件下世爻受日冲 | 无论旺衰皆算冲起 |
| 6 | 动不为散 | 明动之爻受日冲 | 不论旺衰皆属冲起有用 |

#### 7.1.2 暗动固有特征

1. **有动无变、瞬间峥嵘**：暗动之爻没有变爻，只有短期动力
2. **过时则无效**：近兆不管远卜

**时效规则**：
- 短事（急病、应聘、短期投资等）→ 暗动与明动作用无差别
- 长事 → 一概算日破无用
- **例外**：长事中用神/元神暗动 → 不视为日破，是长事快应征兆

#### 7.1.3 暗动现实寓意

- 正在发生或即将发生的人事
- 不行将废：不及时把握机遇则时过将废
- 带有指导卦性质

#### 7.1.4 变爻中冲起与冲散

| 条件 | 结论 |
|------|------|
| 变爻得月旺而受日冲 | 化暗动（短期有用） |
| 变爻月衰囚而受日冲 | 化散/化日破（无用） |
| 变爻旬空而受日冲 | 化暗动（冲空即实，不论月令） |
| 长事之占 | 一概以化破论 |

**假化散三种**：
1. 化回头作用之爻（动能中转）
2. 动而趋亢进之爻（重内轻外）
3. 自占自事而世爻化散（世爻趋势）

#### 7.1.5 暗动应期规则

1. 冲中逢值逢合而应（日冲应期原则）
2. 顺时而应（如申日占暗动→可能应在次日酉日）
3. 冲空即时生效（多代表已发生/正在发生）

### 7.2 心态卦 [吉凶层面]

#### 7.2.1 明心态卦（问题直接反映忧虑疑惑）

**六种模式**：

**模式1：针对世爻之动**
| 情形 | 结论 |
|------|------|
| 子孙发动生世/持世受生/动克世鬼 | 放心喜兆 |
| 官鬼发动生世/持世受生 | 有患无实忧之好兆 |
| 子孙持世受动克 | 忧患 |
| 官鬼发动克世 | 实患大凶 |

**模式2：世爻之变**
| 情形 | 结论 |
|------|------|
| 世变子孙 | 放心 |
| 世变鬼 | 忧患 |
| 世变官鬼回头克 | 实患大凶 |
| 世变子孙回头生 | 放心喜兆 |
| 世变官鬼回头生 | 有患无实忧好兆 |
| 子孙持世变回头生 | 放心喜兆 |
| 子孙持世变回头克 | 有忧 |
| 官鬼持世变回头生 | 有患无实忧好兆 |
| 官鬼持世变子孙回头克（短事） | 放心吉兆 |
| 官鬼持世变子孙回头克（长事） | 放心又反复，最终伤身 |

**模式3：独发之变**
| 情形 | 结论 |
|------|------|
| 官鬼独发有用 | 忧虑 |
| 官鬼独发变衰 | 放心喜兆 |
| 动爻独发变鬼 | 终为忧患 |
| 子孙独发有用 | 放心 |
| 子孙独发变衰 | 忧患凶兆 |
| 动爻独发变子孙 | 放心喜兆 |

**模式4：多发变动**
| 情形 | 结论 |
|------|------|
| 子孙克伤发动之官鬼 | 忧虑可去 |
| 多动连环生旺子孙 | 无忧 |
| 多动连环克子孙 | 不利 |
| 多动连环生旺官鬼 | 有忧 |
| 多动连环克官鬼 | 有利 |

**模式5：静卦**
| 情形 | 结论 |
|------|------|
| 子孙持世 | 放心 |
| 官鬼持世 | 忧虑 |
| 逢破反断 | 子孙持世破→忧，官鬼持世破→放心 |
| 无官鬼+应临子孙 | 无忧 |
| 无子孙+应临官鬼 | 有患 |
| 非官持世+子孙暗动 | 短期放心 |
| 非子持世+官鬼暗动 | 短期忧患 |

**模式6：旬空**
| 情形 | 结论 |
|------|------|
| 子孙旬空（短事） | 忧患短期内存在 |
| 子孙旬空（长事） | 实际无患但放心不下 |
| 官鬼旬空（短事） | 短期无碍 |
| 官鬼旬空（长事） | 虽短期无患但祸患终会来 |

#### 7.2.2 暗心态卦识别

**条件（同时满足）**：
1. 事卦信息淡薄（用神藏伏/假用神/位置蹊跷）
2. 心态特征明显（世应子官、世变子孙/鬼、子孙官鬼发动）

**注意**：事卦信息显著时，即使有心态特征也按事卦分析

#### 7.2.3 事态心态难区分时

- 用心念爻鉴别：心念爻带子孙官鬼=心态，其他=事态
- 官鬼为用神时：静卦按事卦；世变官鬼看日月旺衰（旺为官衰为鬼）

#### 7.2.4 心态卦特殊规则
- 心态卦中无进退概念
- 心态卦百无禁忌不怕世动回头克
- 六冲卦辅助辨别心态卦（冲去顾虑忧患之意）

### 7.3 应期卦 [应期层面]

#### 定义
吉凶无虞只有应期之虑的卦：何时结婚、何时事业、何时下雨、占寿元等

#### 核心规则
- 所有动兆不论生克，都只是应期信号
- 有动兆就有应期

#### 提防假应期卦
| 辨别信号 | 说明 |
|----------|------|
| 玄武持世 | 事主心中藏有隐晦私念 |
| 心念爻与重点信息敏感 | 需要卦感熟练 |
| 无异象 | 有些假应期卦无丝毫信号 |

#### 应期意外
- 好事期盼的应期卦：世用无生扶的静卦=应期意外（事太远还没到）
- 坏事预见的应期卦：世用旺相的静卦=应期意外（寿命还长）
- 静卦"缺根"（无日月生扶）= 应期意外特殊信号

### 7.4 连占卦 [吉凶层面]

#### 注意点
1. 保持客观性
2. 同一天内
3. 注意意念（针对前卦疑惑补充）

#### 粗放分析规则

**静卦**：
- 凭用忌神持世即可定性（无需分析破与否/有根与否）
- 凭用神生世克世即可定性（无需分析用神旺衰）
- 心态卦/问病卦：用忌持世/临应爻即可定性

**动爻**：
- 凭动爻作用方向即可定性（无需分析是否变无用等）

**世用自变**：
- 凭变爻属性/回头作用方向即可定性

### 7.5 特殊时效卦 [吉凶层面]

#### 成立条件（两点同时满足）
1. 所占事情有清晰具体时间范围
2. 特殊元素在卦中清晰显现（用神与动变带日令/月令/日破/月破/持世）

#### 三种类型

**日令时效卦**：
- 事情当天内完结
- 特殊元素：带日令、被日破的用神与动爻
- 规则：月衰受日冲一律视为日破（有气也论破）；冲空仍算暗动
- 特殊：带日令时效的动爻化破/化退/化绝都不会变废

**月令时效卦**：
- 事情当月内完结
- 特殊元素：带月令、被月破的用神与动爻
- 规则：月为司令，月令入爻此爻不怕动克

**终身时效卦**：
- 事情带有终身性质（终身功名/财福/婚姻）
- 特殊元素：世爻（持世六亲为最终际遇）
- 规则：
  - 用神持世 → 终吉利
  - 忌神持世/被用神克者持世/鬼持世 → 终不吉
  - 动变只代表短期过程际遇

#### 注意事项
- 日不管月，月不管日
- 特殊性只限动爻与用神，不包括世爻（世爻破为现状信息）
- 世用合一受破时等于用神受破

### 7.6 太过卦 [吉凶层面]

#### 太旺卦公式
1. 旺相式特组 + 动变相生/化进
2. 日月同时令目标爻旺相 + 至少两处动变相生

#### 太衰卦公式
1. 衰败式特组 + 动变相克/化退
2. 日月同时令目标爻衰败 + 至少两处动变相克

#### 四点注意

| 序号 | 注意事项 | 说明 |
|------|----------|------|
| 1 | 只针对世爻或用神 | 非目标爻不论太过 |
| 2 | 现状信息往往不对轨 | 可作为识别太过卦手段 |
| 3 | 长短期寓意有区别 | 见下表 |
| 4 | 作用力须绝对倾向 | 不绝对性的作用力不归类太过 |

**太过卦寓意（按事情性质）**：
| 事情性质 | 太旺寓意 | 太衰寓意 |
|----------|----------|----------|
| 重病患者问康复 | 年轻=需较长时间；高龄=器满将倾 | - |
| 长期事占 | 短期停顿，长期缓慢 | 短期恶化加速 |
| 短期非成则败 | 趋势反逆转 | 趋势反逆转 |

**不能归类太过的情况**：
- 丑月对子水（可论合可论克，无绝对性）
- 发动趋合（旺者旺衰者衰，无固定倾向）

### 7.7 双核卦 [吉凶层面]

#### 两种类型
- **特指事占**：目标寄托在指定事物方位（某单位应聘、某项目盈利）
- **嫁接事占**：目标寄托在指定人物（与某人合作、某人帮忙）

#### 三个常见模式

| 模式 | 条件 | 是否干预定性 |
|------|------|-------------|
| 应爻与动兆用神无关型 | 应爻与动变用神无对比，不发动 | 不干预 |
| 应爻与动兆用神对比型 | 应爻与动变用神产生对比关系 | 干预 |
| 应爻与动兆用神关联型 | 应爻自身发动/动兆只针对应爻/应与用构成整体动兆 | 干预 |

#### 三点注意
1. 只有特指/嫁接事占才可套入特指分析思维
2. 应爻是通用性特指元素定位，非必然（有些以动爻方式出现）
3. 可旁通到选择类卦象分析

### 7.8 现状卦 [细节层面]

#### 适用问题
- 问某人/事/物目前状态
- 问地形地貌利弊
- 问阳宅阴宅风水
- 寻找某人/物方位
- 自问自身各方面现状

#### 四种改变

| 序号 | 改变 | 说明 |
|------|------|------|
| 1 | 所有信息为现状反映 | 世应动兆都不是趋势预示 |
| 2 | 空破冲合墓皆为现状 | 空=缺陷挖空，破=破损，冲=对立，合=粘连，墓=埋藏受困 |
| 3 | 利弊依据日月旺衰与动变作用 | 定格目前的显像反映 |
| 4 | 显像重心在分布非得失 | 如问物件丢哪=看用神位置，无需考究世爻旺衰 |

### 7.9 指导卦 [吉凶层面/细节层面]

#### 定义
卦象信息不显示趋势而是指导应如何运作

#### 四种类型

| 类型 | 说明 |
|------|------|
| 针对世用的化墓之动 | 趋势朦胧可变，读懂指导可达目的 |
| 带暗动的动变 | 暗动短期性现状性，源自当下举措 |
| 带日令的动变 | 日令代表当前，指导当前行动 |
| 带月令的动变 | 月令代表环境条件 |

**延伸**：很多变卦/静卦的卦辞本身也带指导含义（如卦变师=继续追究，变需=静观其变）

### 7.10 灾卦 [吉凶层面]

#### 四种公式

| 序号 | 公式 | 说明 |
|------|------|------|
| 1 | 日月皆克世爻+日月入动爻相克 | 内外夹击 |
| 2 | 众鬼发动克世 | 多重祸患 |
| 3 | 世爻发动变出鬼爻回头克 | 自招祸患 |
| 4 | 日月皆克世爻+世化回头克 | 内外皆伤（实例相对最少） |

#### 三个特征

| 特征 | 说明 |
|------|------|
| 预示范围 | 往往离不开问事范围，有相关信息提示 |
| 时效范围 | 最长一般不超半年，更多数日或数月内 |
| 留有蛛丝马迹 | 通过现状分析点、五行六亲、星煞六神可定位实际所指 |

### 7.11 六冲六合互化 [吉凶层面/细节层面]

#### 六合变六合
- 拖延不决+缠绵不绝
- 短期急件：不和谐，变拖沓，近期不了了之

#### 六冲变六冲
- 突发又起波澜，刚冲起又被冲散
- 短期动荡多变，败事概率大于成事
- 对长事：即使世用兼旺好事也不长

#### 六冲变六合
- 六冲代表已遭挫破败的现实
- 六合预兆破败将得到愈合
- 如：离异必再有婚，财破必又再兴

#### 六合变六冲
- 六合代表正处相合状态
- 六冲昭示相合将遭破坏
- 如：蜜月后见纠纷，合作将分道

#### 就事论事直判法
- **前提**：就事论事对应现状
- 六冲变六合：此事曾失败→将转成功
- 六合变六冲：此事正合作→将遭破坏
- **不符合条件**：降回细节层面看五行生克

### 7.12 三合局专论 [吉凶层面]

#### 吉凶层面三合局成立条件

| 条件 | 说明 |
|------|------|
| 三爻皆动（含暗动）在主卦 | 不包括变爻 |
| 三爻皆动（不含暗动）在初三爻位 | 包括动爻与变爻 |
| 三爻皆动（不含暗动）在四六爻位 | 包括动爻与变爻 |

**三合局形成后**：单爻趋势特征（回头克、化绝化破等无用动爻定义）一概不再适用（世爻除外）

#### 吉三合局（五种）
1. 用神与世爻同在合局内+非忌神局+世无回头克
2. 含用神的合局动而生旺世爻
3. 用神动生世+世爻构成合局且无回头克
4. 合局生旺用神+世爻得力
5. 合局生旺世爻+用神旺相

#### 凶三合局（五种）
1. 世爻在合局内但动变回头克
2. 用神构成合局动克世爻
3. 动爻构成合局动克用神
4. 用神构成合局但世爻在局外（卦意代入法+相连法）
5. 世爻未与用神合局反与用神相冲之爻合局

#### 假三合局
- 卦面有三合局同时有冲局动力存在
- 冲局者必须是动爻，受冲者也必须是动爻

### 7.13 旬空理论 [吉凶层面/细节层面]

#### 假空（吉凶层面不论空）
- 旺相之静爻：旺不为空
- 动变之爻：动不为空
- 化空：不为真空

#### 真空条件
- **用神真空**：静而平相得用神亦为真空（不符合用旺世兴）
- **世爻真空**：用旺条件下世爻于日月均无生扶，或受动爻冲克的静空

#### 旬空吉凶特殊应用
| 事占类型 | 旬空表现 | 结论 |
|----------|----------|------|
| 占病用神旬空 | 短期痊愈/好转 | 但不代表最终趋势 |
| 求财兄弟持世空亡 | 短期得财 | 不利长久，先盈后亏 |
| 问行人世爻/用神空 | 行人即将归来 | 出空必填相见 |
| 占仕途子孙持世空 | 短期有功名 | 功名难持久 |
| 忧患心态子孙逢空 | 忧郁短期无法了结 | 但终将无恙 |
| 忧患心态官鬼逢空 | 短期无虞 | 东窗事未发 |

#### 出空三种方式
1. 填空：待值空时日到来
2. 冲空：待冲空时日到来
3. 出空期：过了冲空或值空的时段

| 7 | 停止计划法 | 忌神带谋星动 | 停止计划之行 |
| 8 | 不出行法 | 忌神或世爻带马星 | 不要出行 |
| 9 | 不参与法 | 趋势不好 | 不参与是最常用的化解方法 |

### 9.4 卦种完整分类与识别

#### 按信息性质分类
| 大类 | 卦种 | 核心识别特征 |
|------|------|-------------|
| 趋势类 | 事卦、吉凶卦 | 事态用神特征显著 |
| 心理类 | 心态卦 | 子孙官鬼世应特征显著 |
| 时间类 | 应期卦、时效卦 | 吉凶无虞/有明确时限 |
| 指导类 | 指导卦 | 化墓/暗动/带日月动变 |
| 现状类 | 现状卦、影像卦 | 问当前状态/已发生事 |
| 警示类 | 灾卦、另兆卦 | 四种灾卦公式/避轻就重 |
| 复合类 | 双核卦、太过卦 | 特指嫁接/日月动变一边倒 |

#### 按占问方式分类
| 大类 | 卦种 | 识别要点 |
|------|------|----------|
| 主体 | 自占卦、代占卦 | 谁在问/主控还是失控 |
| 时间 | 近占卦、远占卦 | 以日/年为主应期 |
| 频次 | 连占卦 | 同日再占，粗放规则 |
| 形态 | 静卦、独发卦、独静卦、全动卦 | 动爻数量决定分析方法 |

### 9.5 独发/独静/全动分析法 [吉凶层面]

#### 独发卦规则
| 身份 | 有用 | 变衰无用 |
|------|------|----------|
| 元神独发 | 利（生旺世用） | 根源衰竭→反断为凶 |
| 忌神独发 | 凶（克伤世用） | 反断为吉 |
| 间爻独发 | 阻隔凶兆 | 阻隔不成→利好 |

**三点注意**：
1. 太过形式须迎合太过理论
2. 心态卦/短事占影响独发思维（无进退概念、日破可入吉凶）
3. 带放弃意念的问占改变独发正统思维

#### 独静卦规则
- 内外竞发事必翻腾
- 应期：静爻临值或相冲时

#### 全动卦规则
- 只需关注世爻或用神本身的动变趋势
- 吉凶定性大体了然

### 9.6 现状对证方法体系 [细节层面]

| 序号 | 信息来源 | 现状含义 |
|------|----------|----------|
| 1 | 日令月令生扶世用 | 过去/目前获益受辅助 |
| 2 | 日令月令克世用 | 过去/目前受损 |
| 3 | 日令月令合世用 | 纠缠/合作/时而发生 |
| 4 | 月令冲世用 | 已受破损/未发生 |
| 5 | 日月为长生/帝旺/墓/绝 | 长久存在/鼎盛/受困/绝迹 |
| 6 | 带日月的动变爻 | 过去现状信息 |
| 7 | 日月往来（动变同时带日月） | 日常重复行为 |
| 8 | 暗动之爻 | 正在/即将发生 |
| 9 | 日合月冲/日冲月合 | 目前存在的现状 |
| 10 | 主卦卦象 | 现在（变卦=将来） |
| 11 | 动卦持世之爻 | 现状或最终结局 |
| 12 | 旬空之爻 | 尚未发生/不存在 |

### 9.7 代占卦分析框架 [吉凶层面]

#### 代占三种类型

| 类型 | 定义 | 取用方式 |
|------|------|----------|
| 失控性代占 | 完全没有主导控制力，只是关心 | 取代表被代占者的六亲为用神 |
| 主控性代占 | 所占事实质受自己主导控制 | 可能取六亲为用，可能取世爻 |
| 被动性代占 | 被他人提议代占 | 视同提议者自占自摇 |

#### 主控性代占关键规则
- 动态指向代占用神 → 以代占用神为分析核心
- 动态指向世爻 → 以自占思维定位卦中六亲
- 被代占者无自主权时 → 决策权在世爻

### 9.8 反吟伏吟理论 [吉凶层面/细节层面]

#### 反吟
**卦之反吟**：主变卦对冲（如震变兑）= 六冲变六冲 + 更剧烈折腾

**卦宫反吟**：主变卦宫对冲
- 内宫反吟：内因反复
- 外宫反吟：外因折腾

**爻之反吟**：动吟/化吟
- 吉反吟：动变冲不克 → 不为凶只见反复，动爻如常作用
- 凶反吟：动变冲又克 → 反复事败，沦为无用动爻

**应期**：四个地支元素逢值之期出现反复

#### 伏吟
**吉凶层面**：
- 最终目标爻伏吟 → 趋势不吉利
- 作用他爻的动爻伏吟 → 仍为有用动爻

**细节寓意**：
- 内卦伏吟：内因呻吟/内心不愿
- 外卦伏吟：外因不宁/他意不安
- 间爻伏吟：过程压力停滞

**应期**：
- 作用他爻之动：逢值逢合
- 最终目标爻伏吟：逢值逢冲

### 9.9 伏藏理论 [吉凶层面/应期层面]

#### 概念区分
| 概念 | 定义 | 应用层面 |
|------|------|----------|
| 伏神 | 取为用神的伏爻 | 吉凶+应期 |
| 伏爻 | 藏于主卦各爻下补全缺失的爻 | 吉凶+应期 |
| 藏爻 | 所在宫位纯卦所有爻的总称 | 细节层面 |

#### 伏神吉凶三方面

**一、卦理见凶**：
1. 伏空：伏居空地事与心违（占病例外=短期无碍）
2. 伏破：深埋再遭破败，神仙难扭
3. 用藏世伤：用神藏伏+世爻受伤
4. 用神藏伏可能显示其他事情

**二、卦意预兆**：
- 藏爻确认法
- 冲合预兆法
- 环境不具备谋事条件

**三、心态反映**：
- 事态用神藏伏+心态用神清晰浮现 → 暗心态卦

#### 飞神空破特殊应用
- 短期事占+用神藏伏+飞神空/破 → 吉（冲飞露伏/空下伏神易引拔）

#### 伏神应期
1. 冲去飞神之时（冲飞）
2. 伏神逢值逢合旺相之时（露伏）

### 9.10 三墓理论 [吉凶层面]

#### 假墓（六种）
| 序号 | 情形 | 说明 |
|------|------|------|
| 1 | 作用他爻的动爻化墓 | 动爻继续有用 |
| 2 | 动爻被动墓 | 动爻不受制 |
| 3 | 动爻被日墓 | 动爻不受制 |
| 4 | 墓空墓破化衰 | 墓爻空/破/变衰则墓不成立 |
| 5 | 世用旺相被墓 | 用旺世兴即可 |
| 6 | 三合局内世墓 | 局内世爻不论真墓 |

**除以上六种外皆为真墓**

#### 墓的细节寓意
- 客观：昏迷、藏匿、穴居、拘禁、约束
- 主观：迷茫、隐瞒、蒙蔽、关注、倾心
- 指导：化墓指引正确方向（指导卦模式）

#### 墓的应期
1. 冲墓之时
2. 冲入墓之爻之时
3. 出墓期（过了冲墓/冲爻时段）

### 9.11 三绊理论 [吉凶层面/应期层面]

#### 真绊（两种条件）
1. 卦象构成：外卦或内卦三爻全部动而化绊，或六爻全部化绊
2. 问事性质：明确目标期限的短期事占（出行行人之占最多见）

**真绊结论**：世爻/用神被真绊=事悬不决拖延无期/事与愿违

#### 假绊（其余情况）
- 吉凶层面忽略，直接按生克分析
- 应期层面须计入

#### 两种反映现状的特殊组合
1. 爻在日绊同时月破 → 不为绊，为现状信息点
2. 爻动化绊同时日冲月冲 → 不为绊，为现状信息点

#### 三绊应期
- 冲绊：冲相绊两元素之时
- 解绊：合住动绊两元素为解绊之期
- 并不并因多字眼：相绊两元素多出一个同类发动 → 绊不成立

### 9.12 化进化退理论 [吉凶层面]

#### 进退定义
- **化进**：寅→卯、申→酉、丑→辰、未→戌
- **化退**：卯→寅、酉→申、辰→丑、戌→未

#### 吉凶应用
- 目标爻化进 → 趋旺，事兴如意
- 目标爻化退 → 趋衰，事败萧条
- 动力爻化进 → 有源之水力量不绝
- 动力爻化退 → 后劲不继终局变衰

#### 特殊应用
1. 心态卦中无进退
2. 短事之占（数天内）无进退 → 真动假进退
3. 往来占中无进退：化进=远离，化退=靠近

#### 化进化退应期
- 化进：逢值（动爻）、逢合（动变爻）、逢进（变爻）
- 化退：逢冲（动变爻）、逢退（变爻）

#### 细节十四类进退
进神：亥→子、寅→卯、巳→午、申→酉、丑→辰、辰→未、未→戌
退神：子→亥、卯→寅、午→巳、酉→申、辰→丑、未→辰、戌→未

### 9.13 长生十二宫吉凶应用 [吉凶层面]

#### 核心数据
1. 丑辰未戌四土爻长生在申
2. 辰未戌三土爻帝旺在子
3. 寅木卯木墓在未；巳火午火墓在戌
4. 子水绝在巳；酉金绝在寅

#### 应用原则
- 吉凶层面只存在于日令、动爻与变爻中
- 从于卦理卦意
- 化绝、化墓非常具有现实意义
- 化长生与帝旺虽属趋旺，但受"从于卦理卦意"限制，实际应用少

### 9.14 应爻特殊分析规则 [吉凶层面]

#### 应爻基本规则
1. 应爻是卦中重要信息点，代表事情也代表他人
2. 应爻不动 → 应期细节层面信息库
3. 应爻发动 → 吉凶层面重要信息点
4. 应爻独发 → 即使非用忌也是重要趋势预兆

#### 应爻为用神时的特殊规则
1. 跳出复合连动（三合局除外），直接作用世用
2. 存在反断原则：
   - 应爻忌神克世变回头克 → 反断为吉
   - 应爻元神变回头克 → 反断为凶
   - 应爻带用神趋合世 → 直接作用无须连动（留意世爻旺衰）

## 补充一：六冲理论详解 [吉凶层面/细节层面]

### 冲的四种存在形式

#### 形式1：爻之六冲存在于日令
- 静爻旺相受日冲 → 暗动（有用）
- 静爻衰弱受日冲 → 日散（日破，无用）
- 动爻受日冲 → 旺则冲起正得力；衰则破动（事情开始陷入困境，非破散）

#### 形式2：爻之六冲存在于月令
- 月破：破败+尚未发生的细节

#### 形式3：存在于两动爻之间或动变爻之间
- 反吟

#### 形式4：存在于动静爻之间
- 有用动爻冲克安静用事爻：不管旺相与否、是否旬空，都肯定遭受伤害
- 冲克损害力度得到速度助推变得更锐利，祸凶快而急速
- 静爻旺相只能在细节层面作为受损程度依据

**动爻冲静爻四条件**：
1. 受冲静爻日月总体旺相与否
2. 受冲静爻是否旬空
3. 冲静爻的动爻是否真正有力
4. 不再有纯冲与冲克之理论区别

**动爻冲静爻前提**：动爻只有在不能参与直接作用世用的前提下，才会去冲动其他静爻

### 六冲卦吉凶层面认识

**核心观点**：六冲卦只代表缓急程度与维持时效长短的细节信息

| 六冲类型 | 寓意 |
|----------|------|
| 主卦六冲 | 目前事态多变与突发性 |
| 卦变六冲 | 未来发展的短期多变或不能恒久 |

**主卦六冲细节应用**：
- 事业突起波澜
- 寻人出行正在张罗
- 求仕途忽见机缘
- 求财运刚遇起伏
- 占健康忽然发病

**静卦六冲与卦变六冲**：
- 反映短变急兆的未来趋势
- 持久性缺乏的代表
- 短事：快而急促，利短线操作
- 长事：有始无终、后劲不济

**占病特殊**：六冲卦 → 近病痊愈/久病恶化几率大

### 六冲卦辅助心态卦判断

- 郁结难除谓之合 → 六合代表忧患
- 无忧无虑谓之冲 → 六冲代表冲去忧虑
- 六冲六合信息可辅助正确的卦种判定

## 补充二：六合理论详解 [吉凶层面/细节层面]

### 合的四种存在形式

#### 形式1：存在于日令
- 静爻合旺
- 动爻或变爻被日合 → 日绊

#### 形式2：存在于月令
- 动变静爻皆为月合
- 细节上才可论绊（吉凶层面无月绊）

#### 形式3：存在于两动爻之间
- 动绊、化绊

#### 形式4：存在于动静爻之间
- 趋合：纯为趋合不带生克才是真正趋合
- 爻旺被趋合 → 得力更旺
- 爻衰被趋合 → 立足不稳被牵引
- 合中带生/合中带克 → 吉凶层面不能作合断

### 六合卦吉凶层面应用

**核心观点**：六合卦非吉利代名词，是缠绵不息的现状与趋势信息

| 六合类型 | 寓意 |
|----------|------|
| 主卦六合 | 所问之事现状已拖延良久/维持一段时日 |
| 卦变六合 | 趋势将变拖延、缠绵不绝 |
| 静卦六合 | 同卦变六合 |

**对不同时效影响**：
- 占长事：吉则吉兆久远，凶则长恨缠绵
- 占短事：影响更突出，可干扰利好定性
  - 若吉：主要体现应期迟缓
  - 若凶：郁结不解/不了了之/被迫中止

**特殊应用**：占寿元见六合 → 不用担忧短期变故

### 卦变六合六冲的指导意义

- 变卦卦辞可能是主观指导信号（"半个指导卦"）
- 动变核心信息定性不利时：
  - 卦变六合 → 事情短期应暂缓搁置的指导
  - 卦变六冲 → 应半途中止中断的善意提醒
- 卦内显出吉兆定性时 → 六冲六合为客观特征解读

## 补充三：占卦意念与代占详解 [吉凶层面]

### 占卦基本原则

- 一念之诚可格天地
- 遇事即占不拘早晚
- 慎勿二念，二念不灵
- 提防礼让导致的意念窜卦

### 代占类型详解

#### 失控性代占
- 对所占他人之事完全没有主导控制力
- 只是关心关注而问占
- 围绕代占用神（代表被代占者的六亲）分析

#### 主控性代占
- 所占他人之事实质受自己主导主控
- 两种信息指向：
  1. 动态指向代占用神 → 以代占六亲为分析核心
  2. 动态指向世爻 → 以自占思维定位六亲

**示例**：大人代问孩子上哪家幼儿园
- 卦中动变针对子孙爻 → 利弊聚焦孩子
- 卦中妻财动克世 → 提醒大人避免选太贵学校

#### 整体视觉代占
- 不取被代占人为用神
- 审视条件直接取目标六亲为用
- 示例：妻代占夫求财，若夫钱属家庭共享 → 直接取妻财为用

#### 被动性代占
- 被他人提议代占
- 视同提议者自占自摇

## 补充四：三墓三绊三合局应期综合 [应期层面]

### 三合局应期详细规则

#### 找破绽法
| 破绽类型 | 应期 |
|----------|------|
| 局内逢日破月破 | 待值令补破之时成局 |
| 局内逢被墓/动墓 | 待冲墓出墓之时成局 |
| 局内逢旬空 | 待填空冲空之时成局 |
| 局内被日月动爻绊住 | 冲绊之时成局 |
| 三元素缺一（静爻） | 冲起静爻或逢值之时 |
| 三元素缺一（不见） | 转到补全元素的时令 |
| 三元素缺一（在日令） | 短应当日，长应逢值时令 |
| 三元素缺一（在月令） | 一般应在当月 |

#### 外局喜成内局要破
- **外局**：三合局以外来者身份生克世用 → 喜成
  - 无破绽：合局整体临值临合之时成局
  - 示例：巳酉丑合酉金局 → 逢酉（值）或辰（合）成局
- **内局**：世爻在三合局内/世用均在内 → 要破
  - 冲掉三爻其一即可破局
  - 示例：巳酉丑局 → 逢亥/卯/未冲去其一

### 三合局细节层面应用

1. 占长久之事 → 长远持久
2. 非长久事 → 事态迟缓应验/短事被拖长
3. 祸福程度超乎常态（特别强烈）
4. 世用处于合局内 → 受困/难有作为（行人/行程/调动方面）

### 三墓应期详细

| 墓类型 | 吉凶层面 | 应期层面 |
|--------|----------|----------|
| 日墓 | 爻旺附旺衰附衰 | 可论 |

## 补充五：过去现状信息对证完整体系 [细节层面]

### 七类现状信息来源

#### (1) 日令与月令
- 日月生扶世用 → 过去/目前获益
- 日月克世用 → 过去/目前受损
- 日月合世用 → 纠缠/合作/时而发生
- 月令冲世用 → 已破损/未发生
- 日月为长生/帝旺/墓/绝 → 长存/鼎盛/受困/绝迹

**注意**：克世为吉利的事占，日月克世照样不吉（如求财日月财克世=曾经因财困损）

#### (2) 带日月的动变静爻
- 动变同时带日月 → 日月往来（日常重复行为）
- 主卦静爻藏伏/对位爻带日月 → 现状标志

#### (3) 暗动之爻
- 暗动代表正在发生
- 冲空模式暗动 → 更明显的现状信息（即时生效）
- 动爻月旺受日冲 → 代表正在发生/即将发生
- 动爻月衰受日冲 → 暂时不见效/未发生
- 静爻日破 → 现状信息

#### (4) 冲中逢合
- 日合逢月冲 / 日冲逢月合 → 目前存在的现状信息
- 可扩展到伏藏、对位爻

#### (5) 主卦卦象
- 主卦寓意现在，变卦代表将来
- 静卦：既代表现状又说明未来趋势

#### (6) 动卦持世之爻
- 反映现状信息 OR 事后最终结局

#### (7) 旬空之爻
- 动变之爻/静爻/世应用神旬空 → 尚未发生/不存在
- 可扩展到对位爻、藏伏之爻

## 补充十一：月令应期层面完整知识 [应期层面]

### 临月建之爻的应期应用

| 条件 | 应期结论 |
|------|----------|
| 用事爻临月建得旺 | 可能当月应吉 |
| 长久之测用事爻临月建被动兆克伤 | 虽最终凶但当月内可保无碍 |
| 临月建化回头克 | 有时不是当月无碍，反是当月被克伤提醒 |
| 爻临月建动而生旺世用 | 可能当月应吉 |
| 爻临月建动克世用（忌神） | 可能当月应凶 |

### 得月合之爻的应期应用

| 条件 | 应期结论 |
|------|----------|
| 卦象属凶而世用得月合 | 数下三个月内暂时无碍 |
| 卦象属吉而世用得月合 | 数下三个月内应吉 |
| 动爻得月合动而生旺世用 | 数下三个月内应吉 |
| 动爻得月合作忌神冲克世用 | 数下三个月内应凶 |

**月合局限性**：
- 当令之爻与月合之爻是敌人时 → 法则失效
- 示例：辰月占，酉金动变午火回头克 → 不再是三月内免遭殃，午月就会倒霉
- 时段范围太长或太短都不适用

### 逢月破之爻的应期应用

| 应期类型 | 说明 |
|----------|------|
| 实破 | 逢临值之时 |
| 补破 | 逢合之时 |
| 出破 | 时期出月破之时 |

**规则细分**：
- 动变之爻月破 → 多应在实破或补破时日
- 静爻月破 → 不排除应在时期出破的时日
- 也可能众多应期信息一一轮过才应事

**月破受克的特殊**：用事爻月破+受克显凶 → 破爻逢伤更重，祸事连连或始于当月

### 月令信息的全面覆盖

月令涵盖的不仅是过去信息，还包括：
- 过去已经存在而延伸到当下的信息
- 过去、现状、未来、定位、强化、另兆、指导、祸福程度

月破细节附加寓意：
- 目前"尚未发生"的寓意（类似旬空）

## 补充十二：日令应期与细节完整知识 [应期层面/细节层面]

### 日令应期层面知识

#### 短事之测（日转月不转）

| 情形 | 应期结论 |
|------|----------|
| 用事动变爻临日令 | 快当日应验（或顺延次日） |
| 吉凶上断定趋势即将衰败 | 当日可暂时不见凶兆 |
| 用事爻/动爻受日冲暗动 | 短期很快有结果（当日/次日/逢值逢合） |
| 元神用神日破或破动 | 即使吉兆不大可能急应当天或次日 |

#### 常事之测（月转日不转）

| 情形 | 应期结论 |
|------|----------|
| 用事爻临日令 | 需防范冲破日建或忌神临值逢合的月份 |
| 其余月份 | 可无往不利 |

**日月功能差异**：
- 吉凶层面：日月同功同权
- 应期细节层面（常事长事）：不同功同权
- 值月功能 > 日令功能
- 月令常换日令始终不变 → 月令主宰不持久，日令力量次位但恒长

### 日令细节层面知识

#### 1. 临日建动爻的作用程度
- 临日建动而生克世用 → 作用程度明显大于一般动爻
- 用事爻得临日建之爻生旺 → 吉运亨通深远
- 用事爻被临日建之爻克衰 → 雪上添霜凶急犹久

#### 2. 临日建之爻代表目前状况
- 可用之对轨
- 动爻/静爻临日破 → 目前是动之徒劳/尚未出现/尚未触及
- 动爻与变爻分别临日建月建 → "日常往来"常态现状

#### 3. 日合月冲/日冲月合
- 无论动静爻 → 一律视为目前存在的现状信息

### 日令时段规律详解

| 事情性质 | 日令作用时段 | 月令作用时段 |
|----------|-------------|-------------|
| 短事 | 管头尾（当日有效） | 不转（全月有效但不主导） |
| 常事 | 管头尾（日为恒长背景） | 管中间（月为主导但常换） |
| 长事 | 不转 | 不转，只有太岁动 |

## 补充十三：旬空理论深入展开 [吉凶层面/应期层面/细节层面]

### 旬空基本原理

- 旬空只是事物状态标志，非纯粹旺衰标志
- 空反映到现实：不存在、消失、踏空、没发生、未出现

### 假空与真空完整规则

#### 假空（吉凶层面不论空）
| 条件 | 说明 |
|------|------|
| 旺相之静爻 | 旺不为空 |
| 动变之爻 | 动不为空 |
| 化空 | 不为真空 |

#### 真空条件
| 对象 | 真空条件 |
|------|----------|
| 用神 | 静而平相得用神亦为真空（不符合用旺世兴） |
| 世爻 | 用旺条件下，世爻日月均无生扶，或受动爻冲克的静空 |

### 旬空吉凶特殊应用（五类）

#### 1. 占病用神旬空
- 近病者：病根浅痊愈概率大
- 久病者：暂见好转更恰当
- 旬空=短期吉凶关键依据，动变=长期最终答案

#### 2. 求财兄弟持世空亡
- 短期得财
- 不利长久之财，先盈后亏
- 有小利及时收手否则得不偿失

#### 3. 问行人世爻/用神旬空
- 行人即将归来
- 出空必填等行人来填实相见

#### 4. 占仕途子孙持世旬空
- 短期（数日内）或有功名及身
- 但功名难持久：得官难到任/得位半途被削/因病辞职

**世破特殊**：问官世破=官持世变鬼缠身

#### 5. 忧患心态之占
- 子孙逢空 → 忧郁短期无法了结（但终将无恙）
- 官鬼逢空 → 短期无虞（东窗事未发）
- 子孙空有时反映长期心理障碍（杞人忧天）

### 旬空出空三种方式

| 出空方式 | 说明 | 适用 |
|----------|------|------|
| 填空 | 值空时日到来 | 动变爻旬空多应在此 |
| 冲空 | 冲空时日到来 | 动变爻旬空多应在此 |
| 出空期 | 过了冲空/值空的时段 | 静爻旬空多应在此 |

### 旬空细节层面应用

#### 时空特性取象
- 不管动爻/静爻/藏爻/对位爻
- 可作为事态过去、现状、未来分析的解读点
- 示例：问病卦中巳火旬空 → 可能病人之前在巳/亥年月生过病

#### 现状信息
- 动空 → 寓意即将发生
- 静空 → 寓意总会发生

#### 事后信息（逢空必填）
- 静爻凸显的空亡信号 → 预告事后某种可能
- 学界首提静爻"逢空必填"概念

#### 世爻旬空心态取象
- 不安定、不踏实、心悬着
- 没把握、底气不足、犹豫不决
- 心虚、不上心、没诚意
- 粗心大意、无心装载

#### 实物取象
| 五行 | 空象 |
|------|------|
| 水空 | 水瓶、水井 |
| 火空 | 燃气瓶、塑胶管 |
| 土空 | 道路下陷、陷阱、门窗 |
| 金空 | 钟鼓、长笛（金空则鸣） |
| 木空 | 衣柜、书架 |

## 补充十四：反吟伏吟完整规则 [吉凶层面/细节层面]

### 反吟完整体系

#### 卦之反吟
- 主卦与变卦对冲（如震变兑、兑变震）
- 必然六冲变六冲
- 短期内反复多变且更为剧烈

#### 卦宫反吟
- 主卦宫与变卦宫对冲
- **内卦宫反吟**：因内因反复
- **外卦宫反吟**：外部环境令事情折腾

**注意**：静卦无反吟（静卦折腾用游魂/归魂判别）

#### 爻之反吟
- **动吟**：两动爻互冲
- **化吟**：动爻与变爻互冲

**吉凶区分**：
| 类型 | 条件 | 结论 |
|------|------|------|
| 吉反吟 | 动变相冲不相克 | 不为凶只见反复，动爻如常作用 |
| 凶反吟 | 动变相冲又相克 | 反复事败，沦为无用动爻 |

#### 反吟应期
- 两动加两变共四个地支元素
- 逢值之期 → 出现反复周折

#### 反吟细节
- 外卦反吟：外部原因造成事态反复，身不由己
- 内卦反吟：内部原因使事情周折（自身因素/决策/心态变幻）
- 不同动爻变爻彼此相冲 → 也可作"反吟"态势理解

### 伏吟完整体系

#### 基本寓意
呻吟、哀怨、叫苦、不宁之象：
- 占功名 → 宦途哀怨
- 谋财利 → 进退两难
- 问家宅 → 迁欲不就
- 占婚姻 → 不宁
- 求医药 → 徒劳
- 问行人 → 在外忧愁

#### 吉凶层面
- 最终目标爻伏吟 → 趋势不吉利
- 作为作用他爻的动爻伏吟 → 仍为有用动爻

#### 细节寓意
- 内卦伏吟：内因呻吟、内心不愿
- 外卦伏吟：外因不宁、他意不安
- 内外伏吟：内外忧郁呻吟
- 占彼此（排除世应化吟）：内卦我心不愿，外卦他意不安
- 间爻伏吟：过程压力令事停滞

#### 伏吟应期
- 作用他爻之动：逢值逢合（与一般动爻无异）
- 最终目标爻伏吟：逢值逢冲之期应事

## 补充十五：化进化退完整展开 [吉凶层面/细节层面]

### 基本进退

**吉凶层面进退**：
- 化进：寅→卯、申→酉、丑→辰、未→戌
- 化退：卯→寅、酉→申、辰→丑、戌→未

### 细节层面十四类进退

**进神（7种）**：
亥→子、寅→卯、巳→午、申→酉、丑→辰、辰→未、未→戌

**退神（7种）**：
子→亥、卯→寅、午→巳、酉→申、辰→丑、未→辰、戌→未

**不可能互变的六对**：亥子、巳午、丑戌、辰未、辰戌、丑未
- 用于横向对比方法

### 化进化退特殊应用

#### 1. 心态卦中无进退
- 心态卦出现化进化退 → 不按进退定位

#### 2. 短事之占无进退
- 周期数天内 → 真发动假进退
- 辅助确认信息：动爻临日月化日月、临暗动化暗动等短兆/现状特征

#### 3. 往来占中无进退
- 以摇卦人为轴心
- 化进 = 远离
- 化退 = 靠近（回归）
- 无明确往来目的时：化进→出差出游，化退→回国回乡回家

### 化进化退应期

| 进退类型 | 应期公式 |
|----------|----------|
| 化进 | 逢值（动爻）、逢合（动变爻）、逢进（变爻） |
| 化退 | 逢冲（动变爻）、逢退（变爻） |

**综合考虑**：旬空、日破、月破等因素

### 化进化退细节寓意

**化进神**：
- 求财得财过后又得财，财利不绝
- 问事业此地如意那方又受重用
- 卜生意将各处顺景渐发大
- 同属性多处多面开花

**化退神**：
- 占病越来越重人渐被消磨
- 问工作诸多不顺
- 求仕途江河日下官运渐离

**同属性互变寓意**：
- 子孙化子孙 → 多胎多儿女
- 土爻变土爻 → 到多地游览
- 兄弟变兄弟 → 同伴多人
- 官爻化进生世 → 事业多点开花

## 补充十六：世用关系深入补充 [吉凶层面]

### 用旺世兴的详细理解

**用旺世兴为吉**：
- 用神旺相前提
- 世爻得日月其一生扶（即使总体衰败也无妨）
- 寓意：事情有成功机会 + 自身能把握住

**用旺世衰为凶**：
- 事情并非没机会成功
- 但因自身主客观条件不足，最终不利
- 或自我对事情参与欲望不强，错过机会

### 世爻受制的各种形式

| 世爻受制形式 | 说明 |
|-------------|------|
| 世动而化退 | 世爻后劲不继 |
| 世动而化绝 | 世爻趋向绝境 |
| 世动而回头克 | 世爻自变伤自身 |
| 世动而化忌神 | 世爻变出不利因素 |
| 世动而化鬼 | 世爻被祸患纠缠 |

**注意**：世动而化破 → 除外（不算世爻受制）

### 动爻作用次序

**吉凶层面**：先用神后世爻
- 动爻生旺用神 = 事情可成
- 同一动爻又克伤世爻 → 吉凶层面论事成

**细节层面**：
- 动爻生旺用神 + 克伤世爻 = 工作可得但不利自身发展/有压力

### 动爻无目标时的回归

- 动爻完全不再作用世爻或用神时
- 天性"冲合墓"自然属性功能无限扩大
- 可冲起/冲破静爻，趋合与动墓他爻
- 目的：间接性地对用神与世爻产生作用

### 外动内动规则

- **外动**（外卦动爻）：反映过程
- **内动**（内卦动爻）：反映结局

## 补充十七：不适合摇卦的情况 [实战应用]

### 两类不适合摇卦的心理状态

#### 1. 主观参与欲望不强
- 对事情的主观介入意愿缺乏
- 导致世爻变衰/世爻不兴等差卦趋势
- 并非来自事情客观趋势，而是自我怠慢心态所造成

#### 2. 高度慌张失措
- 短暂的非理性心理状态下不适合立即摇卦
- 这样的卦一定会显出鬼爻旺
- 鬼爻旺并非寓意忧虑成真
- 而是事主高度紧张的即时信息投影
- 会造成心态卦的误判

## 补充十八：增删卜易230例补充规则 [吉凶层面]

### 废爻型（凶）特殊日月组合
- 月破（月令冲）+ 日克（日令克）→ 动爻生旺失效，凶性确立
- 应用：直接废掉用神/世爻，不看动爻，吉凶立定

### 金刚型（抗克）特殊日月组合
- 月建（临月令）+ 日令生 → 传统"无用动爻"定义失效
- 动爻金刚不坏，化克依然有用

### 判断层级
```
月破日克 > 月克日克 > 一般日月组合
动爻能量 > 日月（废爻型例外）
```

### 内重外轻要点
- 外力（他爻动克/生目标爻）→ 只影响过程
- 内力（目标爻自变）→ 变爻决定终局
- 目标爻自变时，外力能量一并传入变爻，以变爻回头作用为终局

**推论**：
1. 用神自变化退/化绝/化回头克 = 终局凶，外力无法改变
2. 世爻自变化凶 = 终局凶，外力生助枉然
3. 见目标爻自变 → 先看变爻 → 再看外力（加速/减缓）

### 综合分析优先级
```
① 废爻型特殊日月组合 → 最高优先级
② 时效卦（临月建）前提 → 次高优先级
③ 三合局方向（合局优于单爻）
④ 内力（自变变爻）> 外力（他爻动变）
⑤ 普通日月组合 + 动变生克
```

## 补充二十二：读心术与心念爻展开 [细节层面]

### 心念爻定位完整规则

**优先顺序**：
1. 世爻不动 + 对位爻与世不同 → 世之对位爻为心念爻
2. 世动或对位同世 + 世之藏爻与世不同 → 世下藏爻为心念爻
3. 以上不成立 + 君爻为静且与世不同 → 君爻为心念爻
4. 世位不成立 + 君动 + 君下藏爻与世不同 → 君之藏爻为心念爻

### 心念爻的作用与限制

**作用**（仅细节层面）：
- 完善断卦的某一方面信息
- 问病：心念爻出现官鬼=担心反映，子孙=求医想法，用神之墓=想知应期
- 不能用于吉凶定性

**限制**：
- 与吉凶定性完全割裂
- 不能根据心念爻定位用神（本末倒置）
- 只是事主一种想法，心念与问题不一定同步

### 心念爻辅助卦种判断

**适用场景**：事态与心态信息都很明显（特别是世化鬼/世化子/世应子官发动方向矛盾时）

**判断规则**：
- 心念爻带子孙或官鬼 → 多为心态卦
- 心念爻为其他六亲 → 多为事态卦

## 补充二十三：特殊情况处理汇总 [实战应用]

### 用神多现处理法

当卦中同一六亲出现多个时：
1. 用真假用神五特征判定
2. 对证现状选择与事实对应的爻
3. 动爻优先于静爻
4. 持世持应者优先
5. 与日月相关者优先

### 用神藏伏处理法

1. 确认是否需要取伏神（卦面无该六亲时）
2. 判断伏神吉凶（伏空/伏破/用藏世伤）
3. 考虑是否暗心态卦
4. 短期事占利用飞空伏现

### 世用合一处理法

- 世爻与用神重叠（如自占疾病世爻持鬼）
- 受生为世用受生局（最吉）
- 受克为世用受克局（最凶）
- 受破：等于用神受破

### 卦象无动兆处理法（静卦）

1. 先看用忌持世
2. 再看世用生克关系
3. 参照用旺世兴
4. 考虑暗动影响
5. 短事所有暗动可入吉凶层面
6. 长事只有用神/元神/忌神暗动入吉凶层面

### 多动爻矛盾处理法

1. 检查三合局（优先）
2. 连动规则（围绕世用方向）
3. 重动轻静重内轻外
4. 世爻不参与复合连动
5. 动爻无目标时回归自然属性
